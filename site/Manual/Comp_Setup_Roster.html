Returning to the decoder programmer selection pane (Tab Programmer Setup), you will select a decoder exactly as before. In both the basic and comprehensive programmers, though, if you have already entered at least one locomotive in the roster, you may also select that specific locomotive from the pull-down menu. See the illustration below for an example from my roster:
 
 

Here I select a locomotive from my existing roster, in this case, a Kato SD40-2 Snoot, road number 4204.

DecoderPro checks the data in the file, and learns that this locomotive has a Digitrax DN149K2 decoder in it. It not only will load the proper decoder programmer, it will also recover all the CV settings from the file as well. This makes restoring lost settings much easier.

Clicking on the "Open Programmer" button will now open the appropriate Comprehensive Programmer. All programmers, Basic and Comprehensive, open to the Roster Entry pane, which will be similar to that shown below.

The actual Roster pane is essentially unchanged from the Basic programmer, but suddenly there are a LOT more options in the Tabs at the top! If you are unsure how to handle the Roster information, go back to that section in the Basic Programmer. If you're ready to move on to bigger and better things, let's go to the next tab, the (expanded) Basic Tab.
