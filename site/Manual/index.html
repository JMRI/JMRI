DecoderPro v1.0.6

Updated 9/5/2002


A Java-based cross-platform application for the programming of Digital Command Control (DCC) decoders for model railroad use.

DecoderPro is one of a set of cross-platform applications for model railroaders. It can run on any computer system that will run Java 1.1.8, whether it is Macintosh, Windows, or Linux based. It does require that Java be installed on the computer. You can get Java as a free (but very large...) download at:

Java Link
 

Note that all screen shots in this manual are off of a Macintosh, using the Mac Graphic User Interface option. While your screens will probably differ in some layout details, they will be essentially the same.
