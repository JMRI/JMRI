JMRI LocoNet tools

The JMRI libraries contain a number of LocoNet tools. This page introduces them, describes how to use them, and points you to installation and other information.
  • LocoNet monitor - Display and log the traffic on the LocoNet, in either hex or human readable form.
  • Slot monitor - Watch the contents of the command station's slots as they're changed.
  • Packet source - A tool to let you type and send packets to the LocoNet
  • LocoIO programmer - A tool to let help you configure a LocoIO board.
The libraries also contain utilities for controlling your layout (operating turnouts and signals, responding to inputs and block occupancy, etc), for programming decoders, and others. These are not LocoNet specific, so are described in the main JMRI pages.

LocoNet monitor

The LocoNet Monitor tool displays LocoNet traffic in a human-readable form. The last 20 messages are available in a scrolling window. Optionally, you can include the time the message was received, and/or the raw packet bytes. The log information can also be stored in a text file.

Command station slot monitor

The Slot Monitor tool displays a table of the "command station slots'" contents. Slots are used to control individual locomotives and consists. The display includes the decoder's speed step format, current speed and function settings, consist information and status. You can choose to show either all slots or just the ones being actively used.

Sending packets

The Send Packet tool lets you type in a LocoNet packet and send it to the layout. You type the packet in hexadecimal as pairs of characters, in the same way its displayed in the LocoNet monitor.

Programming LocoIO boards

The LocoIO programmer provides a simple interface for configuring John Jabour's LocoIO board, a powerful component for building CTC boards and control panels. (More information on the board is available at http://groups.yahoo.com/group/loconet_hackers/files/LocoIO/) A more detailed description of the programmer is available on a separate page.

Getting these tools

A complete version of these tools is available for download. It will work on Macintosh, Windows, Linux and OS/2 computers. The JMRI libraries will work with NCE, EasyDCC, Atlas, Wangrow, Lenz and Digitrax command stations, but of course the LocoNet tools only work with a Digitrax LocoNet. Note that JMRI is free software, and there is no "pay" version!

For more information on how to install the demo, whether it will run on your computer, etc, please see the README file.

JMRI is an open-source project, so you can get the entire source code at any time via the Project page. For more information, please see the main JMRI page.

Running the tools

  • Download and install the JMRI demo program.
  • Start the program.
  • From the preferences window, select either "MS100" or "LocoBuffer", depending on how your layout is connected. Then select the serial port, and your command station type. If you're using a LocoBuffer, you should also select the baud rate corresponding to the LocoBuffer's jumper settings. Click "save". You'll be prompted to quit and restart the program.
  • After restarting the program, select a tool from the "LocoNet" menu.
  • Send us feedback on how to make this better.
