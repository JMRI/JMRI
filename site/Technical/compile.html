This page discusses how to compile a simple java class. For more complicated situations, including building the complete JMRI libraries, see our Technical page. For more information on JMRI, see the main JMRI page.

Preparation

For simple compilations, you only need the Java "Software Development Kit" (SDK). Macintosh computers already have it installed. For Linux and Windows computers, this is a free download from Sun:
Main Java download page
Download page for the popular Java 1.3.1 version.
Note that this is a large (30MB or so) download!

Once you've installed the Java SDK, you must run the JMRI installer again. This is necessary to reconnect the JMRI communications routines with new versions of the Java libaries you've just installed.

Compiling - Windows

In the project folder, you'll find a compilation script called "compile.bat". (Depending on your system options, you might need to "show details" to see the .bat extension). Double clicking on this file recompiles all the Java source code in the folder.

When you double click, a new DOS window opens and a "javac" command is issued with a bunch of options specific to your system, the java version, etc. If there are any compilation errors, the error messages will appear in that window.
