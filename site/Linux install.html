Linux installation instructions:

These steps outline how to install DecoderPro on Linux. These steps were tested on a system running Red Hat 7.2, but should work with all recent Linux distributions.

david zuhn has another set of instructions posted on his website.

These steps need to be performed with superuser priviledges.

  • Install Java Runtime Environment (JRE). To do this, download the Java 1.4 "Linux Red Hat RPM shell script" from http://java.sun.com/j2se/. This should give you a file named "j2re-1_4_0-fcs-linux-i386.rpm". Install that with: 
       rpm -i j2re-1_4_0-fcs-linux-i386.rpm
  • Download the JMRI 1.2.5 version from http://prdownloads.sourceforge.net/jmri/JMRI.1.2.5.tgz?download and unpack it. This will create a "JMRI" folder. To install, you just have to move the JMRI folder to where you want it on your computer.

    The 1.2.5 version is our most recent "production-quality" version. The project is continuously adding features, however, and new "test" versions appear every couple of weeks. You may find one of these has features that you really want. These are announced in the "jmriusers" Yahoo discussion group at http://groups.yahoo.com/group/jmriusers/. To download one of these, please see the test version download page.

  • Install portions of Java Communications API. Download the file "javacomm20-x86" from http://java.sun.com/products/javacomm/index.html Then install it with 
              
   cp comm.jar /usr/java/j2re1.4.0/lib/ext/
   echo Driver=gnu.io.RXTXCommDriver > /usr/java/j2re1.4.0/lib/javax.comm.properties
  • Install Rxtx. Download the file "rxtx-bins.1" from http://prdownloads.sourceforge.net/jmri/rxtx-bins.1.tar.gz?download. Install it with 
        cp rxtx/1.4/jcl.jar /usr/java/j2re1.4.0/lib/ext
    cp -d rxtx/1.4/i386-pc-linux/lib* /usr/java/j2re1.4.0/lib/i386/

    If that doesn't work with your Linux version, download and compile the latest Rxtx version from http://www.rxtx.com/download.

  • Give user-level application access to serial ports 
       chmod 666 /dev/ttyS0
   chmod 666 /dev/ttyS1
  • Give user access to lock files (required on Red Hat). Edit /etc/groups & /etc/gshadow to add user to 'uucp' and 'lock' groups.
The install is complete. Switch to the JMRI directory and run the programs using one of 
  ./DecoderPro.csh
  ./LocoTools.csh
  ./JmriDemo.csh

Connecting your layout to the computer

To connect your layout to the computer, follow one of these links:
  • Atlas Commander via Lenz LI100: Note that you can't program decoders with this configuration.
  • Digitrax
  • EasyDCC
  • Lenz XPressNet via LI100
  • NCE
  • SPROG
  • Zimo MX-1

You can also go directly to the pages describing how to use the DecoderPro program, use the LocoTools program, and the Jmri libraries and JmriDemo program.
