Digitrax-specific information

This page contains links to various Digitrax-specific information.

The JMRI LocoNet tools. Describes the various LocoNet-related tools that the LocoTools program provides. These are also present in the JMRI demo program, and to some extent in DecoderPro. And, of course, you can build them into a program of your own!

The JMRI LocoIO programmer. Describes a programmer for John Jabour's LocoIO board.

A note on connecting a BDL162 to a Uhlenbrock Intellibox DCC system. This was written by Michael Beck.

Measurements on the power circuits of DigiTrax components. This discusses why various Digitrax components can't share a power supply at a slightly technical level.

Using KeySpan USB adapters with a MS100 interface. This describes some tests on various Keyspan USB adapters. One worked, one didn't. It includes some pointers to information on pin assignments, cables, etc.
