JMRI THROTTLE LAYOUT

User Guide


The JMRI Throttle Layout is just one of the many digital railroading tools that are part of the JMRI Project. This user guide was intended to introduce those unfamiliar with the throttle tool. After reading through the sections identified below, and trying the throttle tool on your own, hopefully it will increase your enjoyment of the DCC model railroading hobby.



HOW TO USE THIS GUIDE

This user guide was set up almost as a training guide. Each section link below is organized like a book. If you are new to using the throttles, follow each section chapter below in sequence and the text will almost walk you through the process step-by-step of opening up a new throttle layout, customizing it, and then saving all your hard work. If you have already found the throttle layout tools, but are looking for more information about them, then hopefully there is a section or two below that can help.

Select one of the chapter sections below, and a new web page will be displayed. When you have finished with the chapter, you will find a link that will bring you back to this page where you can select a new chapter to view. Also within each chapter section, there will be links available where you can jump directly to the next or previous chapters instead of returning to this page first.

IMPORTANT NOTE
Multiple programmers are involved with the code development on this project. Some work on the "front end" of the code (which is the interface that you see on your computer screen), and others are working on the "back end" of the software for each particular DCC system needs. Although the "front end" should remain constant across all systems, there may be slight differences experienced on your particular DCC system and computer in comparison to what is described and shown in this user guide.

The examples and references shown in this guide are based on version 1.3.10 of the software, operating under Windows 2000 with a Digitrax system.
