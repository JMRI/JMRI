The JMRI/DecoderPro README file

This page contains the most recent information about the JMRI downloads, including the DecoderPro program.

The current production version is "1.1", issued October 20, 2002.

For instructions on how to download and install this software on your computer, follow one of these links:

  • Windows
  • Macintosh MacOS "Classic"
  • Macintosh MacOS X
  • Linux
  • OS/2

To see what you'll need to connect your computer to your layout, follow one of these links:

  • Atlas Commander via Lenz LI100: Note that you can't program decoders with this configuration.
  • Digitrax
  • EasyDCC
  • Lenz XPressNet via LI100
  • NCE
  • SPROG
  • Zimo MX-1

For more information on the general program, click here. Follow one of these links for information on a specific program:

  • The DecoderPro programming tool
  • The LocoTools suite of LocoNet tools
  • The JmriDemo demonstration program

As the program develops, we also produce test versions every two or three weeks. You may want to try these, as new features will show up in them first. They are announced in the JMRI users email group at http://groups.yahoo.com/group/jmriusers/

Bruce Petrarca of Litchfield Station sells a CD of these files if you'd prefer that to downloading them. Click here for more information.
