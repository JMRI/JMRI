OS2 information:

You need to have Java (at least 1.1.8), Swing and javacomm installed on your machine. These can be downloaded from the IBM site.

A screen size of 800x600 or larger is strongly recommended.

Download the 1.2.5 version from http://prdownloads.sourceforge.net/jmri/JMRI.1.2.5.zip?download and unpack it. This will create a "JMRI" folder.

To install, you just have to move the JMRI folder to where you want it on your computer. You can then start a program by running a .CMD script. For example, run the DecoderPro.CMD script from this directory with 

   DecoderPro

Connecting your layout to the computer

To connect your layout to the computer, follow one of these links:
  • Atlas Commander via Lenz LI100: Note that you can't program decoders with this configuration.
  • Digitrax
  • EasyDCC
  • Lenz XPressNet via LI100
  • NCE
  • SPROG
  • Zimo MX-1

You can also go directly to the pages describing how to use the DecoderPro program, use the LocoTools program, and the Jmri libraries and JmriDemo program.
