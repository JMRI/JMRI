Add/Edit Block Table Entries

Enter the system name for the Block you wish to add. For example, a valid Block system name for a LocoNet layout could be LB456. Another example, a valid internal sensor system name could be IB123. The optional user name can be anything you wish to describe the Block. See "Names" help page for more information.
