JMRI Install Guide: 64-bit Windows

This page is a start at documenting how to use JMRI on 64-bit Windows.

JMRI runs in 32-bit mode, even on 64-bit Windows systems.

Even after installing Java on a Windows Vista 64-bit computer, JMRI applications won't start because the "java" command can't be found.

John Sacrey provided instructions on how to update the Windows "path" variable so that Java is found:

This will get you to where the System Environment variables are defined:
  • Right Click on "My Computer" and select "Properties"
  • Click the "Advanced" tab
  • Click the "Environment Variables" button
  • Scroll the System variables list until you see "Path", select it
  • Click the "Edit" button and it will bring up an edit box
  • Locate "%SystemRoot%\system32;" in the string and insert "%systemroot%\SysWOW64;" after it, minus the quotation marks.

This works on BOTH XP x64 and Vista x64.
