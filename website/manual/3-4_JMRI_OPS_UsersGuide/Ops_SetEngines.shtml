The Set Locomotive Location window and allows you to override the program and set an locomotive at a location that is defined in your Operations data base. Dialog preceeds the Set Locomotive window

Menu is standard for DecoderPro®.

Locomotive displays the selected engine.

Type is displayed

Status
Out of Service
Takes the Locomotive out of service and will not be assigned by the build program

Location
Name and Track you can set the locomotive at any defined location on your layout.
Auto limits selections to valid locations

Optional--Normally Set by Program--

Destination optionally set the Name of location and the Track .

Train optionally set the train.

press to save the locomotive locations that you have set.

Close the window when completed.
