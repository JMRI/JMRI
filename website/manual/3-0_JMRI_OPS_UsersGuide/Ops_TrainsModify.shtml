Opened from Trains window Tools menu Use this tool to quickly modify the type of cars your train will service. Very useful when adding a new car type to your roster.

Select car tpe you wish to use. The program will then show which Trains can service that car type. Check the Copy box to make a car behave like another. The car type will show to the right of the Copy checkbox.
