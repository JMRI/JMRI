Submitting to JMRI

How do I go about getting these approved and into the next release?

If you're happy with them, you can contribute them to the next release by creating a "Patch" for the project.

To do this:

  1. If you have a SourceForge account (they're free!), log on.
    If not, don't worry about it. Go to the patch web site on SourceForge:

  2. Click on Add new on left side of the screen as shown
    .

  3. On the new page that comes up (shown above), enter something in the following fields:
    1. Select the Category: XML files from
    2. Summary: type in a short summary of file
    3. Description: (Just a little more info) If you don't have a SourceForge account, make sure to put your name and email address in the Detailed Description field so we know who to thank!
    4. Upload a file attachment File: locate the file on your computer and select it.
    5. Now form is completed, click on Your file is submitted for inclusion in a release.
