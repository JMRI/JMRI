  • Set Train Icons Coordinates
    This tool is used to set the default X and Y coordinates of your train icons for each location. The tool allows you to specify the icon coordinates based on the trains direction. You can also update all of your existing routes and locations with these new coordinates. As an alternative, you can use a route's Tools-->Set Train Icon Coordinates for this Route to create coordinates unique to a specific route.
