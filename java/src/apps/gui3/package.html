The gui3 package provides basic function for JMRI applications that use the new "Gui3" user interface first seen in the updated DecoderPro (formerly known as DecoderPro3).

Related Documentation

This package works in concert with the {@link jmri.util.swing} package, which contains the Swing-related implementation classes.

The overall capability is discussed on the Swing support page in the JMRI Developers pages.
