Provides implementations of various JMRI interfaces.

As a long term project, we're moving various Abstract* and Default* implementations out of the {@link jmri} package itself, and moved into here. In the meantime, you'll find many of them still in {@link jmri}.

Implementations of managers can be found in the {@link jmri.managers} package.

Implementations that use Swing should be put in the {@link jmri.implementation.swing} package. @see jmri.managers @since 2.1.5
