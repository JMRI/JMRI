    Dateizutritt

    Direkter Zutritt zu Dateien und Verzeichnisse:

    /prefs
    Die Verzeichnis mit Voreinstellungsdateien des aktuelles JMRI Profils.
    /dist
    Die JMRI Programmverzeichnis.
    /web
    Verzeichnis mit Programme und Dateien für JMRI Web Server.
    /images
    Bilder gehörend zu JMRI.
    /xml/signals
    Lokal gespeicherte Signaldefinitionen, in Klarschrift.

    Wenn eine Verzeichnis eine index.html Datei umfasst, wird den Inhalt dieser Datei gezeigt statt eine Liste der Verzeichnisdateien.

    Wenn eine Verzeichnis sowohl im Nutzerablageort als in der JMRI Programmverzeichnis besteht, wird das Nutzerablageort gezeigt außer wenn das URL Pfad mit /dist oder /xml beginnt.
