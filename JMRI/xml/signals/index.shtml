JMRI: Signal System Definitions

JMRI provides a number of definitions for signal systems via this directory.

For more information, please see the JMRI help page on Aspect-based signaling.

The available definitions include:

basic
A non-prototypical set of aspects and appearances that most model railroaders will understand easily.
AAR-1946
American Association of Railroads 1946 rule book
