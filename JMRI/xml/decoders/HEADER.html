The decoder files definition files at http://jmri.org/xml/decoders/ are regularly updated from the latest (unreleased) development build.

An (unreleased) JMRI development build with the latest definitions included can be downloaded from http://builds.jmri.org/jenkins/job/Development/job/Packages/.
