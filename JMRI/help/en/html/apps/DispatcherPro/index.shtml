JMRI: DispatcherPro Main Window

Dispatcher provides functions and organizes information relating to dispatching trains on a model railroad layout, allowing for automated running.

You access the DispatcherPro main window by starting PanelPro or SoundPro and selecting Dispatcher... from the Tools menu.
