JMRI Virtual Sound Decoder

This is the quick reference page for the JMRI Virtual Sound Decoder Manage VSD Locations tool.

Tabs

  • Reporters Tab: Set locations for JMRI Reporters
  • Blocks Tab: Set locations for JMRI Blocks
  • Operations Locations Tab: Set locations for JMRI Operations Locations
  • Listeners Tab: Set location and orientation of the Listener

Bugs, Notes, and other Oddities

  • If you add new Reporters, you must close and re-open the Manage VSD Locations window to have them appear.
