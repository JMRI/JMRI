Liste des Régulateurs

Cette liste est accessible depuis la barre d'outil du panneau Régulateur.
La Liste Régulateurs affiche tous les régulateurs actuellement ouvert dans JMRI, si une icône est disponible pour les listes dans la base de donnée , elle sera automatiquement utilisée.

Cliquer sur une ligne amènera jusqu'au régulateur correspondant.

Nouveau bouton

Ce bouton ouvre une nouvelle fenêtre régulateur. (donc différent du bouton Nouveau dans la fenêtre barre d'outils Régulateur )
