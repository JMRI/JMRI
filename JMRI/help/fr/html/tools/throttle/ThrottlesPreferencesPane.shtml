Préférences Régulateurs

Ce panneau des Préférences Régulateurs est accessible à partir des des sous-menus Régulateur dans le menu Outils . Pour une meilleure expérience, le choix recommandé est celui illustrée ici.
Il est recommandé de fermer toutes les fenêtres Régulateurs avant d'appliquer un nouvel ensemble de Préférences, puis rouvrez les une fois les nouveaux paramètres appliquée ou sauvés.
  • Le bouton Annuler ferme la fenêtre et laisse les paramètres actuels des Régulateurs comme ils sont.
  • Le bouton Appliquer appliquera l'ensemble actuel des préférences et la fenêtre des préférences restera ouverte.
  • Le bouton Enregistrer enregistrera vos préférences actuelles, et fermerera les fenêtres Préférences, les réglages seront chargés automatiquement au prochain démarrage du JMRI.

Utilisez le Régulateur étendu

Ce réglage activera/désactivera toutes les fonctionnalités étendues des régulateurs.

Réglez les Commandes transparentes

Mettra le fond de la fenêtre interne du régulateur en transparence ou semi- transparence.
Cette fonctionnalité est appelée à bien travailler sur OS X, mais Linux et Windows peuvent aussi fonctionner, cela varie complètement suivant l'interface actuelle Java (modifiable dans préférences principales JMRI )

Utiliser l'image de l'Inventaire comme fond

Quand un inventaire est sélectionnée, l'image définie dans son Panneau des médias sera chargée comme un fond pour la fenêtre du régulateur.
Lorsque cette option est activée et l'adresse sélectionnée est 0 ou 3, une image par défaut sera chargée.

Redimensionner la fenêtre à la taille de l'image

Quand une image de fond est chargée pour le régulateur actuel, cela va forcer la taille de la Fenêtre du Régulateur à celle de cette image.

Rechercher les infos liste quand l'adresse est indiquée

Quand une adresse est inscrite dans le volet Adresse, la base de données JMRI recherchera une entrée de d'inventaire correspondante (et sera donc des options de déclenchements décrites ci-dessus et ci-dessous si elles sont activés).

Chargement automatique de la fenêtre de préférences Régulateurs lorsqu'un lien est trouvé avec l'inventaire

Quand un Inventaire est sélectionné, si un fichier de définition régulateur correspondant existe, JMRI le chargera automatiquement.

Ces fichiers sont stockés dans <le Répertoire Preference JMRI >/Throttle/XYZ.xml où XYZ est dans l'ID de l'Inventaire.
Utiliser l'entrée Enregistrer du Menu Fichier dans une fenêtre régulateur va créer ou mettre à jour ces fichiers.

Ignorer position du cadre Régulateur

Pour les vieux fichiers Régulateurs qui contiennent des position de Fenêtre Régulateur en eux. Faites comme décrit.
Remarque: les positions des Fenêtre Régulateurs sont maintenant sauvegardées dans le Fichier de Sauvegar Configuration Régulateur (pas dans n'importe quels fichiers de sauvegardes Régulateurs )

Masquage indéfini de la Liste Boutons

Quand une liste est sélectionnée, les boutons des panneaux d'adresse sont initialisés comme définis dans le panneau de Fonctions dans DecoderPro, si cette option est activée, les boutons non définis (pas de texte) dans PanelPro seront automatiquement cachés dans le Panneau Fonction .
Remarque: vous pouvez réinitialiser la visibilité des Boutons de Fonctions dans une Fêtre régulateur en allant dans le Menu Afficher et cliquer sur "afficher tous les boutons de fonction" .

Activer la barre d'outils papillon

Cette option activera une Barre d'Outils qui apparaîtra dans toutes les Fenêtres Régulateurs .
