JMRI: Documentation Aiguillage

Que sont les aiguillages?

JMRI appelle peu près n'importe quelle sortie d'un réseau un " Aiguillage". A l'origine, JMRI contrôlait simplement les aiguillages via les décodeurs d'accessoires DCC. Comme le temps passait, davantage de capacité ont été ajoutées à JMRI, mais il a gardé ce nom. Maintenant, par exemple, les broches de sortie individuelle sur un système C/MRI sont également référencées via le nom "aiguillage".

Comment Configurer des Aiguillages

Propriétés Aiguillages

Les Aiguillages sont crééer et modifiés dans le Tableau Aiguillage .
Une rangée de colonnes montre les réglages pour chaque Aiguillage. Le long du bas se trouvent des cases à cocher pour afficher des colonnes supplémentaires

Commander un Aiguillage

Les deux états de sorties sont appelés "Direct" et "Dévié". "Droit" est l'état inactif, non commandé. "Dévié" est l'état actif, l'état commandé.

Dans la plupart des cas, vous pouvez simplement utiliser un aiguillage et JMRI comprendra ce que vous voulez dire. Par exemple, vous pouvez ouvrir la "Commande Aiguillage" du menu "Outils", entrez un numéro d'aiguillage (numéro de l'adresse DCC ou C/MRI), et cliquez sur Dévié ou Droit. JMRI agira à partir de là.

Sauvegarde de configuration d'Aiguillage sur le disque

Les définitions d'aiguillage sont enregistrées dans votre fichier de configuration de réseau, avec les capteurs, les têtes signal, etc. Pour stocker ces informations sur le disque, utilisez Sauvegarder Configuration ... dans le menu Fichier en haut du Tableau Lumière (ou d'autres tableaux à partir du menu Outils), ou sélectionnez Sauvegarder Panneau... dans le menu Panneau .

Ajout d'un nouvel Aiguillage

Pour configurer entièrement un nouvel Aiguillage, suivez cette procédure:

  1. Cliquez Tableau Aiguillage dans le menuOutils pour faire apparaître Le Tableau Aiguillage, puis cliquez sur le bouton Ajouter... au bas du tableau pour ouvrir la fenêtre Ajouter/modifier Aiguillage.

  2. Depuis la liste déroulante Système , sélectionnez le système auquel l'Adresse de l'aiguillage est connectée. Dans la zone Adresse matérielle entrez dedans l'Adresse accessoires DCC de l'aiguillage

  3. Entrez un nom utilisateur. Toute chaîne de caractères qui est différente du nom utilisateur d'appareils de voie sera acceptée, mais il est sage d'utiliser une chaîne qui décrit l'utilisation prévue de l'Aiguillage. Vous pouvez laisser ce champ vide si vous le souhaitez.

  4. Si vous souhaitez ajouter un certain nombre d'appareils de voie à la fois, cliquez sur la case à cocher Ajouter une Plage et puis dans le champs Nombre à ajouter entrez le nombre d'appareils de voie que vous souhaitez ajouter. Si vous avez entré un nom utilisateur , puis il sera ajouté avec un certain numéro pour s'assurer que chaque nom utilisateur est unique. Si une adresse matérielle qui existe est déjà dans le plage qui est en cours d'ajout , alors le système va sauter cette adresse et poursuivra.

  5. Cliquez sur le bouton OK situé au bas de la fenêtre. Si tout va bien, un message indiquant " Nouvel Aiguillage ajouté ..." sera affiché dans la zone des notes. S'il y a des problèmes avec quoi que ce soit, une erreur ou un message d'avertissement sera affiché dans la zone des notes; vous devrez alors corriger l'erreur et cliquez sur OK à nouveau.

  6. Retour au Tableau Aiguillage principal, vous pouvez ensuite sélectionner des options pour l'aiguillage et le contrôler en cliquant sur le bouton d'état.

Les définitions d'aiguillage sont enregistrées dans votre fichier de configuration de réseau, avec les capteurs, les têtes signal, etc. Pour stocker ces informations sur le disque, utilisez Sauvegarder Configuration ... dans le menu Fichier en haut du Tableau Lumière (ou d'autres tableaux à partir du menu Outils), ou sélectionnez Sauvegarder Panneau... dans le menu Panneau .
