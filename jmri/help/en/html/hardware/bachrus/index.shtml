JMRI Hardware Guide: Bachrus

Supported Hardware

JMRI supports the Bachrus MTS-DCC Model train Speedometer, which is an input only device, to capture speed information from a reader module on a Bachrus "Saddle" running stand. For more information on the Bachrus products see the Bachrus website.

When the Bachrus MTS-DCC is set as a layout connection preference, the speedo utility can be used to display scale running speed for any scale from Z to O and beyond.

When the Bachrus MTS-DCC is set as a layout connection alongside a DCC hardware system the speedo utility allows automated measurement of a DCC decoder equipped loco's speed profile.
