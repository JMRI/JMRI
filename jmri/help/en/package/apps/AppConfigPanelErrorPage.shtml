JMRI: Setting The JMRI Preferences

This help screen opens automatically when the JMRI preferences have not been set, or there was a preferences-related error while the program was starting up.

Once the preferences are set properly, this screen won't display. The Help pane for setting the Preferences is at main preferences help page.You can also open the pane by selecting "Window Help" in the Help menu of the preferences window.
