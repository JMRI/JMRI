JMRI: Détails Support Technique CBUS

Le support sur cette page est en évolution rapide, le code réel pourrait étre en avance ou en retard sur la documentation un jour donné.

Cette page décrit les détails techniques du support JMRI pour CBUS Voir aussi les pages sur le soutien général JMRI pour CAN .

CAN adaptateurs

L'approche générale JMRI est de supporter le plus grand nombre de combinaisons de matériel possibles. En soutenant le plus grand nombre possible de personnes et leurs matériels, nous pouvons créer une communauté grandissante de développeurs, donc plus de code.

Dans le cadre de la présente, le soutien JMRI CAN est conçu pour utiliser autant de types d'adaptateur CAN que possible..

Le MERG CAN-RS peut faire "l'énumération " pour CBUS, mais de nombreux autres adaptateurs ne peuvent pas étre utilisés en raison de leur latence intrinsèque. Leurs réponses ralentit efficacement et les rend invisibles.

Par conséquent, Mike Bolton, l'un des développeurs du CBUS a suggéré que JMRI utilise un ID CBUS de 126. Actuellement, c'est une valeur fixe.
