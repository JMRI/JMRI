Configurer les Nœuds Grapevine

Ceci est la page d'aide help/en/jmri/jmrix/grapevine/nodetable/NodeTableFrame

L'outil «Table Nœuds" vous permet de voir tous vos nœuds à la fois. Chaque ligne du tableau correspond à une adresse de nœud. Les colonnes sont les suivants:

Adresse
Le numéro de nœud
État
L'état actuel de ce nœud.
Configuré
Ce nœud a été configuré dans JMRI. vous pouvez dialoguer avec ses Aiguillages, Têtes de Signal, Capteurs, etc
Présent
Ce nœud a été détectée sur Grapevine, mais vous devez le configurer avant de pouvoir y accéder dans le programme.
(blanc)
Le nœud n'est pas configuré, et n'a pas été vu sur Grapevine.
Ajouter/Éditer
La troisième colonne fournit un bouton pour vous pourajouter un nœud manquant ou modifier la configuration d'un nœud existant. Si vous Ajouter ou modifier un nœud, n'oubliez pas de sauvegarder vos préférences à nouveau!
Init
Si le nœud est configuré, il ya un bouton dans la 4e colonne qui vous permet d'envoyer les messages d'initialisation pour le nœud. Cet est utile si par exemple le nœud a été Off et On, il doit donc être réinitialisé. Ceci est uniquement disponible pour les nœuds configurés.

Au bas de la fenêtre est un bouton "Vérifier" et un champ état. Cliquez sur "Vérifier" pour balayer le Grapevine pour vérifier son état et localiser les nœuds. Les champ d'état sont les suivant:

Démarrer
Un balayage commence, mais aucune réponse n'a été entendu de Grapevine encore.
Vérification
Balayage en cours, et le Grapevine semble fonctionner.
OK
OK Analyse terminée.
Grapevine échoué
Analyse terminée sans rien entendre de Grapevine. Peut-être le câble n'est pas complètement connecté, ou quelques éléments ne sont pas alimentés.

Il y a aussi un bouton « Changement d'Adresse Nœud Carte", ce qui peut être utilisé pour changer l'adresse d'un noeud de Grapevine. Il ouvre une nouvelle fenêtre .
