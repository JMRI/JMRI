JMRI: B&O-2005 (CSX) Signaling Definition

This directory contains signaling definitions corresponding to the 2009 CSX rules as applied to ex-B&O lines as described on the Signals of the US website

See the aspect page.

Not all possible lamp combinations have been defined as separate masts so if the mast you require is not defined just use a mast that includes the marker lamps that you require. Generally speaking only the full mast is required since on the prototype B&O system nearly all signal masts are the same but with unused lamps left off.

Signal Mast definitions:

  • Signal mast with full set of marker lamps
  • Basic full speed home signal mast with top centre marker lamp
  • Basic medium speed home signal mast with bottom centre marker lamp
  • Basic slow speed home signal mast with no marker lamps
  • Basic full speed block signal mast with top centre marker lamp and number plate
  • Full speed distant signal mast with full top row of marker lamps
  • Medium speed distant signal mast with full bottom row of marker lamps
  • Dual speed junction signal mast with top and bottom centre marker lamps to control a medium or limited speed simple divergance
  • Full speed distant signal mast for medium speed divergance with top left and centre marker lamps
  • Full speed distant signal mast for slow speed divergance with top right and centre marker lamps
  • Medium speed distant signal mast for medium speed divergance with bottom left and centre marker lamps
  • Medium speed distant signal mast for slow speed divergance with bottom right and centre marker lamps
