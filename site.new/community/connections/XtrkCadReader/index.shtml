Using JMRI and XTrkCad Together

XTrkCadReader is a tool that can take a layout file created with the XTrkCad layout design program and convert it to a panel file JMRI can use.

When you design a layout with XTrkCad, the information about track, turnouts, and their connections are stored in a "XTC" file. When you run XtrkCadRead with an XTC file as input, it creates an "XML" file for JMRI that contains the corresponding information. You can then read that XML file into JMRI to create a layout editor panel that represents your trackwork.

XTrkCadRead can be downloaded via this link.

A manual describing how to run XTrkCadReader, including the options and limitations, is available as a PDF file.

Technical Details

The code for XTrkCadReader is available in the JMRI CVS Repository. You can also look at it in your web browser.

If you'd like a complete copy from CVS, for example to modify or improve it, you can check it out from CVS using the commands: 

   cvs -d:pserver:anonymous@jmri.cvs.sourceforge.net:/cvsroot/jmri login
   cvs -d:pserver:anonymous@jmri.cvs.sourceforge.net:/cvsroot/jmri co XTrkCadReader
(If you're asked for a password after the first command, just hit return)
