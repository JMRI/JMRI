JMRI: General Tools

JMRI provides several tools that are optimized for specific purposes. General tools:

Route Table

A better tool for driving multiple turnouts when something happens on the layout. You can have a panel switch through all the turnouts in your yard throat, etc.

Speedometer

A tool for determining the scale speed of your locomotives. You can use this while setting them up, to catch speeders, or for whatever you'd like.

LocoNet-specific Tools

A set of Digitrax LocoNet-specific tools:
  • LocoNet monitor
  • Slot monitor
  • Packet source
  • PM4 programmer
  • BDL16 programmer
  • SE8C programmer
  • LocoIO programmer
