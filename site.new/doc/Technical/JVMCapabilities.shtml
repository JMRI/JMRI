JMRI: Java Version-Specific Features

JMRI currently builds to Java 1.1.8, with some minor extensions and compatibility hacks; we have a separate page describing those.

If we move JMRI's reference Java version forward to a more recent one, new features of Java will become available to JMRI developers. This page contains information to help make the decision about when and how to do that.
