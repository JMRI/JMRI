JMRI Documentation

JMRI has no single manual. Instead, we have a bunch of different types of documentation, each intended for a different need.

Since JMRI is a group of volunteers, working on the aspects that interest each of us most, the documentation can be quite uneven. It's always a great contribution when somebody writes something new! If you'd like to do that, you can

  • Put a new entry in the JMRI "instant documentation" Wiki. Pages from there that are found to be useful and correct are periodically moved to the main site.
  • Edit an existing web page, and either post it as a contributed "Patch" to the common source, or post it to the jmriusers discussion group.
