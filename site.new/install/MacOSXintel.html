JMRI: Install Guide for Intel-based MacOS X Computers

First, make sure you have the proper drivers for your USB-serial hardware. Most adapter manufacturers have produced new versions of their drivers for use with Intel-based Macintosh computers.

If you're using the Keyspan adapters, you can find drivers here:
http://www.keyspan.com/downloads/

If you're using a LocoBuffer-USB, you can download the driver from here:
http://www.rr-cirkits.com/locobuffer-usb/osx/mac-patch.html

Next, download and install the Java communications libraries to your computer: http://prdownloads.sourceforge.net/jmri/RXTX-for-Intel-Mac.pkg.zip?download

Finally, download the JMRI version you want. The download will provide a folder containing application icons for the programs, and all the support files they need to function. You can put this folder anywhere convenient.

At this point, you should be able to start any of the JMRI applications (or other Java programs) and communicate via serial ports. Please send feedback to let us know whether it worked or not!
