LibUSB

libusb-Java requires Java 1.5 or later. (It might be possible to make small modifications and compile it with Java 1.4, but we haven't done that)

Windows XP and later: Download and install the libusb-win32 installer.

Ubuntu Linux: Start by opening

System > Administration > Synaptic Package Manager

and search for "libusb". There are four packages that start with that name; install them all.

Packages for MacOS X, see here (you don't need the TWAIN or others, just LibUSB).
