JMRI Install Guide: Uhlenbrock Intellibox via Serial Port

DecoderPro can communicate with an Uhlenbrock Intellibox via the built-in serial port and the LocoNet protocol.

Note that this capability was added to JMRI in test version 1.5.2, and is not present in the previous 1.4 production release. If you are using a JMRI version before version 1.5.2, please go to the download page to download the latest version.

Configure the options of your Uhlenbrock Intellibox to be:

  • SO 2 value 4
  • SO 5 value 1
  • SO 6 value 255
  • Baudrate 19200
Then start the JMRI program and configure it to:
  • "LocoNet Intellibox Serial Port" protocol
  • 19200 baud
  • Select hardware handshaking
  • Select DCS100 command station
