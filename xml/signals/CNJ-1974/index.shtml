JMRI: CNJ-1974 Signaling Definition

This directory contains signaling definitions corresponding to Central Railroad of New Jersey rules as described in http://http://signals.jovet.net/rules/CNJ Signal Rules.pdf and http://raildata.railfan.net/cnj/SigRules74/CNJSignalRules74.html. These rules were also used by the New York and Long Branch.

See the aspect page.

Signal Mast definitions:

  • Single searchlight high absolute signal
  • Double searchlight absolute signal
  • Triple searchlight absolute signal
  • Single searchlight low absolute signal
  • Double searchlight permissive signal
  • Triple searchlight permissive signal
  • Single searchlight dwarf signal
