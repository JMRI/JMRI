JMRI: B&O-Semaphore Signaling Definition

This directory contains signaling definitions corresponding to the 1969 B&O rules.

See the aspect page.

This signal system just contains B&O semaphore masts but will work in conjunction with the position light masts of the B&O-2009 system if required. This is a speed signaling system therefore speeds will need to be set on blocks or turnouts (either will do) to ensure that the correct aspects are displayed. As per the prototype no account is taken of turnout position when assessing the aspect to be displayed.

Signal Mast definitions:

  • Buffer stop that always indicates stop for attaching to buffer stops
  • Single arm dwarf signalmast
  • Single arm block signal mast
  • Double arm absolute signal mast
  • Triple arm absolute signal mast
