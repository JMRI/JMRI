JMRI: NYCS-1937 Signaling Definition

This directory contains signaling definitions corresponding to the NYCS-1937 rules as described in the September 1937 version of the New York Central System publication "Rules for the Government of the Operating Department", chapter on Signal Aspects Signal Indications and Signal Rules.

See the aspect page.

Signal Mast definitions:

  • Triple searchlight high signal
  • Double searchlight high absolute signal
  • Double searchlight high permissive signal
  • Single searchlight manual block signal
  • Single searchlight trainorder signal
  • Double searchlight dwarf signal
  • Single searchlight dwarf signal
