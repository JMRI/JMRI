JMRI: CROR 2008 Signaling Definition

This directory contains signaling definitions corresponding to the CROR Signal Rules from the current 2008 rulebook TC O-0-93 This rulebook is with a few minor exceptions a superset of all the other CROR rulebooks I have seen so should make a good basis for a CROR signal system with universal appeal. Unwanted aspects can be disabled in the mast properties to create signals for the older rulebooks.

CROR 2008 rule book

See the aspect page to see how the aspects look.

Signal Mast definitions:

  • Triple head searchlight high signal
  • Triple head searchlight high signal with alternative Green over Red based aspects
  • Triple head searchlight high signal with a yellow triangular 'L' plate
  • Triple head searchlight high signal with a yellow rectangular 'DV' plate

  • Double head inline searchlight high signal
  • Double head inline searchlight high signal with a yellow triangular 'L' plate
  • Double head inline searchlight high signal with a yellow rectangular 'DV' plate

  • Double head staggered searchlight high signal
  • Double head staggered searchlight high signal with a yellow triangular 'L' plate
  • Double head staggered searchlight high signal with a yellow rectangular 'DV' plate

  • Single head searchlight high signal
  • Single head searchlight high signal with white circular 'A' plate
  • Single head searchlight high signal with a yellow rectangular 'R' plate

  • Double head searchlight dwarf signal
  • Double head searchlight dwarf signal with a yellow triangular 'L' plate
  • Double head searchlight dwarf signal with a yellow rectangular 'DV' plate

  • Single head searchlight dwarf signal
  • Single head searchlight dwarf signal with a circular 'A' plate
  • Single head searchlight dwarf signal with a rectangular 'DV' plate

    With the exception of single head masts plates are only shown on the masts where they modify the meaning of the aspect. If you are using a plated mast be sure to disable any aspects which should not be shown, I have left some in on the 'L' and 'DV' plated masts which I suspect should not be there. With the exception of single head masts plates are only shown on the masts where they modify the meaning of the aspect. If you are using a plated mast be sure to disable any aspects which should not be shown, I have left some in on the 'L' and 'DV' plated masts which I suspect should not be there. The 'L' plate elevates medium speed to limited speed for a given aspect, and 'DV' similarly elevates slow speed to diverging speed.

There is only one issue to be aware off in that the CROR-2008 refers to a speed named DIVERGING and there doesn't seem to be a way to configure it as an additional speed. It is between SLOW and MEDIUM so users will probably be able to use one or the other as a workaround. It only affects 'DV' plated signal masts which are not key to the system functioning otherwise since they are optional.
