JMRI: FS - Italian State Railways - Signaling Definition

This directory contains signaling definitions corresponding to the Italian rules as described in Il Treno Company book “Inserimento in linea dei segnali FS” according to the 1987 Signal Rules “Regolamento Segnali” of the Italian State Railways (“Ferrovie dello Stato”, FS today RFI).
