JMRI: SE-TSFS-2010

This directory contains signaling definitions corresponding to the TSFS 2010 - JTF rule book for Swedish Railroad described in the September 2009 version of the "3H Signaling System", Module 3H.
The wikipedia has also some english information in its Swedish railway signaling page.

See the aspect page.
