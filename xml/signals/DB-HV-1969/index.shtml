JMRI: DB HV System 1969

This directory contains signaling definitions corresponding to the 1969 DB "Einheitsbauform".
This is the Home signal/Distant signal (Hauptsignal/Vorsignal) system revised in 1969.
You can find more information in this Bahn Statistik reproduction of the SignalBuch DB DS301 from 15 December 1959.
English information available in Wolfgang Meyenberg's sh1 page.
The wikipedia has also some english information in its German railway signalling page.
For a visual description of aspects via java applets see the excellent Christoph Schmitz Signalling page.
For a historical view of signals evolution please see J. B. Calvert introduction to German and Austrian Signalling.

See the aspect page.
