JMRI: NYC-1956 Signaling Definition

This directory contains signaling definitions corresponding to the NYC-1956 rules as described in the September 1956 version of the New York Central System publication "Rules of the Operating Department Effective October 28, 1956", chapter on Signal Aspects Signal Indications and Signal Rules.

See the aspect page.

Signal Mast definitions:

  • Triple searchlight high signal
  • Double searchlight high absolute signal
  • Double searchlight high permissive signal
  • Double searchlight dwarf signal
  • Single searchlight dwarf signal
