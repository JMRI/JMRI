JMRI: Signal System Definitions

JMRI provides a number of definitions for signal systems via this directory.

For more information, please see the JMRI help page on aspect-based signaling.

The available definitions include:

basic
A non-prototypical set of aspects and appearances that most model railroaders will understand easily.
AAR-1946
American Association of Railroads 1946 rule book
BN-1989
1989 October Burlington Northern
Partial aspect mapping, not finished.
BR-2003
2003 BR rule book
CROR-2008
2008 Transport Canada TC O-0-93 rule book
CSX-1998
CSX 1998 rule book
Some errors in aspect mapping, not finished.
DB HV 1969
1969 Deutsche Bundesbahn "Einheitsbauform" Home Signal/Distant signal (Hauptsignal/Vorsignal) system revision
Missing aspect mapping, not finished.
InfraBel-2008
Current info for Belgium in 2013 taken from Wikipedia
NYCS-1937
New York Central System 1937 rule book
Missing aspect mapping, not finished.
NS-2008
Norfolk Southern Corporation Operating rules book (NORAC)
ProRail-1954
1954 Dutch rule book still in use in the Netherlands
RG-1965
1965 Rio Grande Rules of the Operating Department
SLSF-1973
1973 SLSF (Frisco) rules
Missing aspect mapping, not finished.
SPTCO-1930
Southern Pacific Railway and Transportation Company 1930 rule book
SPTCO-1960
Southern Pacific Railway and Transportation Company 1960 rule book
