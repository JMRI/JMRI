JMRI Decoder Information

The JMRI DecoderPro information on decoders has been made available in several forms on the following web pages:

  • Decoder identification CV values used by DecoderPro to identify the manufacturer and type of an unknown decoder.
  • Index to individual decoder pages, listed by manufacturer. This provides a one-page summary of the characteristics and CV contents of the decoders.
  • Selection guides listing decoder size, and by decoder electrical characteristics.

We also provide files in "comma-separated values" format via a separate index page.

Windows and Java .properties files containing some of the information are available in a separate directory.

All of these files are created automatically from the JMRI decoder files using XSLT. The toolkit to do this is available from the JMRI distribution, or our GitHub repository.

These pages are updated as people contribute decoder information to JMRI.
