Control panel look in style
Czechoslovak State Railways - Ceskoslovenske statni drahy - CSD
Unified control place - Jednotne obsluzne pracoviste - JOP
These files were contributed by Petr Sidlo

In directory template_1 and template_2 are the catalogTrees.xml files
that contains preset items for the Control Panel Editor in the style of CSD JOP. 
To use it, copy it to the profile directory.

V adresari template_1 a template_2 jsou soubory catalogTrees.xml ktere
obsahuji prednastavene polozky pro Control Panel Editor ve stylu CSD JOP. 
Pro jeho pouziti ho prekopirujte do adresare s vasim profilem.
