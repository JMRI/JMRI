Version 2


There are ten .gif files every signal mast aspect except for shunting dwarf signal:

vertical, left, right (round signal heads);
vertical squared, left squared, right squared (squared signal heads);
left Layout Editor, right Layout Editor (flashing aspects of round signal heads made for Layout Editor);
left squared Layout Editor, right squared Layout Editor (flashing aspects of squared signal heads made for Layout Editor).

Icons in Layout Editor
Kindly note that "LayoutEditor" folder icons have to be set when a signal showing flashing aspect is chosen.

Alternate flashyellow/flashgreen aspect
In order to reproduce this aspect on layout please set "Virtual Mast" and then a Logix/LogixNG is needed to drive physical outputs.

Dark Aspect
This aspect is used for single searchlight squared signal head on double-track mainline only. In the other signals it has to be disabled.

Versione 2


Sono previsti 10 files di tipo .gif per ogni aspetto, fatta eccezione per il segnale basso:

vertical, left, right (per i segnali a vela rotonda)
vertical squared, left squared, right squared (per i segnali a vela quadrata);
left Layout Editor, right Layout Editor (aspetti lampeggianti per l'inserimento dei segnali con vela rotonda in Layout Editor);
left squared Layout Editor, right squared Layout Editor (aspetti lampeggianti per l'inserimento dei segnali con vela quadrata in Layout Editor).

Icone In Layout Editor
Per cortesia scegliere l'opzione "Layout Editor" ("left" o "right") quando il segnale da aggiungere in questo pannello prevede aspetti lampeggianti.

Aspetto giallo-verde lampeggiante alternato
Per riprodurre questo aspetto sul tracciato: scegliere "Palo virtuale" e poi scrivere una Logix/LogixNG per pilotare le uscite.

Aspetto spento
Questo aspetto si usa solo per i segnali a vela quadrata singola di prima categoria/accoppiati/avviso. Questo aspetto negli altri casi va disabilitato.
