    File Access

    Directly access certain files and directories:

    /prefs
    The preferences directory for the current JMRI profile.
    /dist
    The JMRI application installation directory.
    /web
    Some web applications, and files required for the web server.
    /images
    Images included with JMRI.
    /xml/signals
    Local signal definitions, formatted to be readable.

    If a directory contains an index.html file, the index.html file will be returned instead of the directory contents.

    If a directory exists in both the user preferences and the JMRI installation directory, the directory in the user preferences is returned unless the URL path starts with /dist or /xml.
