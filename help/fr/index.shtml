JMRI Système d'Aide

Ceci est la page d'aide de haut niveau JMRI.

La page de remerciements montre que beaucoup de personnes ont été en mesure de contribuer.

Information future

La page web principale de JMRI https://jmri.org est un bon point de départ pour obtenir des renseignements sur l'état actuel, les mises à jour, etc (en fonction sur votre ordinateur, vous pourriez avoir á copier ce lien dans votre navigateur Web pour être en mesure de voir des sites extérieurs correctement)

Nous avons un groupe "JMRI utilisateurs" pour une discussion générale. Si vous souhaitez poser une question, obtenir des d'aide, ou de suggérer des améliorations, ce groupe de discussion est un bon endroit. Vous pouvez abonner, voir les messages, etc, á https://groups.io/g/jmriusers.

Nos pages techniques sont encore en cours d'écriture. C'est encore un stade précoce, et nous sommes à court de documentation. Mais si vous êtes intéressés en travaillant avec le code, nous serions ravis de vous aider á démarrer par e-mail, etc.
