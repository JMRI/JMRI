Applications JMRI®

JMRI est une suite de programmes multiplateformes basée sur Java. pour les modélistes ferroviaires.

Elle contient des applications modulaires pour :

  • Programmation des décodeurs de commande numérique (DCC). (DecoderPro®)
    • .
  • Dessiner des panneaux informatisés pour représenter un réseau de de chemin de fer (PanelPro™ et Éditeur de Réseau)
  • Contrôle du Réseau (Tous les modules)
  • Créer des manifestes de train pour dupliquer les opérations ferroviaires (OperationsPro™)
    • .
  • Contrôle des signaux sur le réseau (Tous les modules)
  • Contrôle de l'éclairage et du son dans la pièce, etc. (SoundPro™)

Toutes ces capacités sont contenues dans une . "https://jmri.org/download/index.shtml">installation.

Faites-le pas à pas

Notre conseil général est de commencer par les bases. Pour cela nous vous suggérons de commencer par mettre en place une voie simple connectée à votre centrale en premier lieu.
Lorsque le ou les trains circulent et qu'un aiguillage répond à votre régulateur tenue à la main, il est temps d'ouvrir l' Aide Matérielle (utilisez l'onglet Matériel en haut de la page).
Ensuite, il faut essayer de faire des choses basiques avec PanelPro, par exemple utiliser un régulateur JMRI pour faire fonctionner une loco.
Une fois que cela fonctionne comme vous l'attendez, vient le moment de construire un Panneau, de remplir votre Inventaire, etc.
Prenez quelques notes pour vous-même, esquissez un schéma simple de la façon dont vous concevez votre Réseau. Souvent, cela aide mieux que les mots pour mettre les choses dans l'ordre (et se souvenir 2 ans plus tard, pourquoi vous avez choisi de taper ce drôle de nom d'aiguillage').

Ne vous découragez de plonger, dans JMRI - ou toute autre chose puissante, mais complexe - une étape à la fois, vous construisez en quelque sorte une base solide qui sera rentable en temps voulu.
Construisez graduellement sur les concepts de base du programme. et prenez les éléments qui vous intéressent en suivant certains des liens en bas de la barre latérale gauche sur l'aide web du JMRI.

Si c'est votre première visite sur notre manuel, nous vous suggérons de . de faire le tour de DecoderPro® à l'adresse https://jmri.org/help/fr/html/apps/DecoderPro/Tour.shtml
pour avoir un aperçu de ce que nous allons couvrir.

Configuration du système

JMRI® nécessite que Java soit installé, que votre ordinateur soit Mac, Windows, ou Linux

La version 2.12 peut fonctionner sur n'importe quel système informatique qui exécutera Java 1.5 ou plus récent.
La version 2.14.1 nécessite Java 1.5 (ou 1.6 si vous souhaitez un système de glisser-déposer) & ; drop)
La version 3.10.1 nécessite Java 1.7 ou une version ultérieure.
La version 4.2 et plus nécessite Java 1.8.


Plus d'informations sur Java et la configuration matérielle requise.
Obtenez la dernière version de Java en téléchargement gratuit à l'adresse suivante : java.com/download.
