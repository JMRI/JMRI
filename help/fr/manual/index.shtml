JMRI® Applications

JMRI est une série de programmes multi-plateforme Java pour les modélistes ferroviairess.
Il contient des applications modulaires for:

  • Programmation de décodeurs Digital Command Control (DCC) (DecoderPro®)
  • Dessin des panneaux avec l'ordinateur pour représenter un réseau modèle de chemin de fer (PanelPro™ and Layout Editor)
  • Commande du réseau ( Tous les modules )
  • Création de manifestes des trains pour la duplication des opérations de chemin de fer (OperationsPro™)
  • Commande des signaux sur le réseau (tous les modules)
  • Commande éclairage et son dans la pièce, et ainsi de suite (SoundPro™)

Si ceci est votre première visite de notre manuel, nous suggérons que vous fassiez notre Tour DecoderPro® pour avoir une vue d'ensemble du domaine que nous couvrons.

Toute cette capacité est contenue dans une seule installation.

Système Requis

JMRI® demande que Java soit installé, si votre ordinateur est basé sur Mac, Windows, ou Linux.

  • Version 2.12: Java 1.5
  • Version 2.14.1: Java 1.5 (ou 1.6 si vous voulez glissez et déposer)
  • Version 3.10.1: Java 1.7 ou plus récent.
  • Version 4.2: Java 1.8.
  • Les Versions 4.2 - 4.25.5: Java 8 (pas release 311) ou Java 11 (recommendé)
  • Les Version 4.25.6 - 4.26: Java 8 ou Java 11 (recommendé)
  • Les Versions 4.99 - 5.x: Java 11
