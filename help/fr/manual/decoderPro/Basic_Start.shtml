La première action dans la programmation d'une locomotive avec DecoderPro® est de remplir l'écran de saisie de l'inventaire.

Un inventaire est une base de donnée des locomotives que votre installation de DecoderPro a programmée. Elle inclut l'information vue dans l'écran ci-dessous:

La plupart de ces champs sont explicites. Cependant, notez que le premier champ (ID) devient le nom de fichier dans l'inventaire pour la locomotive que vous programmez. Configurez un schéma pour identifier votre locomotive depuis son décodeur programmé. C'est à dire: Fabricant, Type de Loco, et Adresse (ex: Kato_SD40-2_5645). Les espaces dans le champ ID seront changés par des underscores (_) quand le fichier sera écrit. Dans un club , les ID commencent souvent par le nom du propriétaire, ou le numéro de membre. Ceci fait que l'Inventaire (qui est trié par ordre alphabétique).

Le Champ Décodeur Commentaire est un bon endroit pour ajouter la date d'achat de la locomotive, le prix, ou autre information que vous pensez être importante à enregistrer.

Le bouton enregistre l'information actuelle sur le disque dur de votre ordinateur et dans le répertoire où votre fichier roster est localisé. Le stockage par défaut est le même répertoire où le programme est installé.

Le bouton peut être utilisé pour retourner au fichier Roster à la condition d'un nouveau fichier pour le type et la version du décodeur listé dans l'entrée inventaire. Il ne change PAS les valeurs dans le fichier roster de votre disque dur, sauf si vous enregistrez spécifiquement après avoir utilisé ce bouton de réinitialisation. De plus, il n'est pas écrit dans le décodeur tant que vous n'avez pas sélectionné une opération d'écriture. Il a été inclus dans DecoderPro afin que, si vous vous trompez en passant d'écran à écran en sélectionnant les variables saisies, vous puissiez revenir au jeu de paramètres de base par défaut du fabricant et recommencer.


La plupart des décodeurs activent le menu Réinitialiser (à côté du menu Fichier en haut de la page). Ceci réinitialisera le décodeur aux paramètres par défaut du fabricant pour tous les CV, ou seulement quelques-uns d'entre eux si le fabricant a plusieurs routines de réinitialisation. Pour cela, il écrit directement dans un CV dans le décodeur, si le décodeur a cette fonctionnalité, CV8 dans le cas de Digitrax. Tous les décodeurs peuvent ne pas prendre en charge la réinitialisation et leur fonctionnement dépend du fabricant et du décodeur.

Vous noterez que l'adresse du décodeur est affichée, mais estompée (ou vide s'il s'agit d'une nouvelle locomotive dans la liste). Ce champ est automatiquement renseigné par le programme et est déterminé à partir de l'adresse saisie à l'aide de l'Onglet Basique auquel nous arriverons bientôt. Cette adresse est utilisée avec la fonction Ident sur la page de démarrage si vous souhaitez rappeler une locomotive qui est déjà dans votre liste.

Le mode de programmation peut être modifié en sélectionnant le mode dans la liste déroulante en bas de la fenêtre. Á gauche de la liste déroulante se trouve le mode de programmation actuel, qui est choisi

Il y a plusieurs modes de programmation. JMRI sélectionne maintenant le meilleur mode pour vous. Cela est déterminé à partir du type de centrale de commande que vous utilisez et du modèle de décodeur que vous essayez de programmer. Habituellement ce sera OK. Si vous rencontrez un problème de programmation vous pouvez essayer un mode différent.

Pour la plupart des décodeurs récents, restez en mode Paginé ou selon le mode direct pris en charge par votre Centrale de Commande. La documentation de votre décodeur doit vous faire savoir si la programmation dans un autre mode est nécessaire ... mais si vous rencontrez des problèmes, expérimentez. Si un mode n'est pas pris en charge pour le décodeur et le système sélectionnés, cette option est grisée.

Lorsque vous faites votre choix, le mode de programmation devrait maintenant s'afficher à gauche de la liste déroulante.

Maintenant que nous avons toutes les informations de l'inventaire en ordre, nous pouvons continuer avec la programmation du décodeur.

Mais d'abord, parlons un peu de la gestion de l'Inventaire. La fenêtre principale DecoderPro (celle dans laquelle nous avons sélectionné le programmeur) et l'écran initial de présentation de DecoderPro ont un menu intitulé Inventaire. Lorsque vous cliquez dessus, vous avez la possibilité de modifier, d'imprimer et de déplacer vos fichiers d'Inventaire. Exporter/Importer et Copier sont particulièrement intéressants. Supposons que vous et un ami avez deux locomotives identiques et que vous souhaitez que les vôtres soient configurées comme celles de votre ami. Maintenant, vous pouvez mettre sa locomotive sur votre voie, lire tous les CV et les mettre dans la vôtre, mais si il vit sur l'autre côté du pays, ce nâ€™est pas pratique. Alors, demandez-lui d'utiliser DecoderPro pour exporter son fichier de locomotive et l'envoyer par e-mail. Vous l'importez dans DecoderPro, changez l'adresse si vous le souhaitez, puis utilisez cette entrée d'inventaire pour programmer votre locomotive. De même, vous pouvez copier une entrée de l'inventaire pour dupliquer une de vos propres locomotives dans une deuxième entrée de l'Inventaire avec sa propre adresse.
