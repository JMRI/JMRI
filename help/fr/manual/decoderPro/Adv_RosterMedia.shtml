Le Volet Média Inventaire vous permet d'ajouter une image principale et une icône á votre inventaire machines. L'image principale peut être utilisée comme image de fond si elle est sélectionnée dans les préférences Régulateurs. L'icône image est utilisée dans la liste Régulateur et les différents menus inventaires.

Image qui doit être utilisée comme image principale pour cet inventaire: La grand cadre noir est l'espace réservé pour l'image. L'image est placée dans cet emplacement réservé en suivant ces instructions.

    1. Premièrement vous devez trouver une image de la Locomotive et la stocker sur votre disque dur.
      1. Sources
        1. Images téléchargées depuis un site web sur votre ordinateur
        2. Photos prises avec un appareil digital ou scannées depuis un film
        3. Autres sources CD, DVD etc. sauvées comme image sur votre ordinateur
      2. Format
        1. Le format GIF est préféré avec un fond transparent
        2. PNG avec un fond transparent
        3. JPG
    2. Après stockage de l'image sur votre ordinateur, ouvrez le Volet Média Inventaire puis ouvrez Windows Explorer ou le Gestionnaire de Fichiers de votre système et glissez et déposez le fichier image dans le grand cadre noir. Vous pouvez aussi sauver l'image sur votre Bureau et le glisser et déposer dans le Média Inventaire depuis lá. NE PAS essayer de glisser et déposer depuis un site web.
    3. Les images de l'Inventaire sont stockées dans le dossier <JMRI Preferences>/resources. Quand une image est actualisée ou supprimée, elle n'est pas supprimée de ce dossier.

Pour supprimer une image de l'Inventaire, Clic-Droit puis cliquer supprimer dans le menu qui apparait, donc, elle ne sera pas supprimée du dossier Resources.

L'Image devant être utilisée comme icône dans cet inventaire, doit avoir la tête á droite: similairement á l'image, il y a une petit cadre noir qui peut être rempli avec l'image á utiliser comme icône pour l'identification de l'entrée inventaire.  Les instructions sont aussi similaires:

      1. vous devez trouver une image de la Locomotive pour l'utiliser comme icône et la stocker sur votre disque dur.
        1. Sources
          1. Images téléchargées depuis un siteweb sur votre ordinateur
          2. Photos prises avec un appareil digital ou scannées depuis un film
          3. Autres sources CD, DVD etc. sauvées comme image sur votre ordinateur
        2. Format
          1. Le format GIF est préféré avec un fond transparent
          2. PNG avec un fond transparent
          3. JPG
      2. Après stockage de l'icône image dans votre ordinateur, ouvez le Volet Média Inventaire, puis ouvrez Windows Explorer ou le Gestionnaire de Fichiers de votre système et glissez et déposez le fichier image dans le petit cadre noir. Vous pouvez aussi sauver l'image sur votre Bureau et le glisser et déposer dans le Média Inventaire depuis lá NE PAS essayer de glisser et déposer depuis un site web.

Pour supprimer une image de l'Inventaire, Clic-Droit puis cliquer supprimer dans le menu qui apparait, donc, elle ne sera pas supprimée du dossier Resources.

Références web: Vous pouvez copier l'URL du site web où sont localisées les images de références

Après ajout des images dans les cadres pour image principale et icône, cliquer sur ce bouton pout enregistrer les images dans votre Inventaire.
