Quelques brefs commentaires sur les Modes de Programmation

Mode Paginé est une extension du mode Registre qui donne un accès complet à tous les CVs du Décodeur.

Mode Registre est une forme d'extension du Mode Adresse, et il est encore utilisé par quelques décodeurs agés ou bas de gamme, particulièrement quelques uns de MRC et Wangrow. Il est intrinsèquement limité dans sa capacité à accéder à tous les CV d'un décodeur.

Octet Direct est une autre méthode, pas encore prise en charge par tous les décodeurs, qui permet un accès complet. Il y a deux façons de mettre en œuvre le Mode Direct. Les méthodes Octet Direct prise en charge par votre station de commande seront activées.

Mode Adresse est une programmation obsolète. La méthode est incluse ici pour la
conformité totale à la norme NMRA DCC.

Si vous rencontrez des difficultés pour programmer un décodeur en Mode paginé, essayez Direct Byte, puis
t enfin le Mode Adresse. Le Décodeur Accessoire EasyDCC AD4 ne peut être
programmé qu'en octet direct.
