Format du Message de Collecte de Locomotive

 Vous pouvez personnaliser les messages pour l'impression de votre Manifeste et de votre liste de Manœuvres. Sélectionnez les formats dans les menus déroulants. Pour supprimer un attribut, sélectionnez vide. Les Préfixes dans les zones de texte peuvent être édités. Si vous souhaitez utiliser les options de couleur du texte, les préfixes Remise, Collecte et Déplacement doivent être uniques.

Options du Manifeste
Chaque emplacement peut avoir un commentaire facultatif. Vous pouvez imprimer ces commentaires sur votre Manifeste s'ils sont sélectionnés.

Imprimer Valide: imprime l'heure d'assemblage du train et non l'heure à laquelle le Manifeste a été imprimé.
Emplacement. permet de commenter l'emplacement de chaque Wagon sélectionnée.
Itinéraire permet de commenter l'emplacement de chaque Wagon si elle est sélectionnée
Chargés et Vides pour indiquer le nombre de wagons chargés et vides dans le train.
Le format 12 heures est défini par défaut sur une horloge de 24 heures
Heures de départ: pour imprimer les heures de départ sur le Manifeste

Afficher l'horaire indique l'horaire utilisé lors de l'assemblage du train.
Truncate réduit la quantité d'informations sur un emplacement disposant d'une liste de Manœuvres distincte.

Aperçu du Manifeste

Utiliser l'éditeur de texte

Logo
vous permet d'ajouter un logo dans le coin supérieur droit de vos Manifestes et listes de Manœuvres. Appuyez sur et sélectionnez un fichier avec l'image que vous souhaitez afficher. Chaque train de votre Inventaire peut avoir un logo différent; ceci est défini dans Configuration Trains

Commentaire sur les Wagons mal placés

Vous permet d'ajouter une notification pour que les opérateurs recherchent des Wagons égarés.

Commentaire dangereux

Vous pouvez déclarer un chargement de Wagon comme dangereux.

  • Nouveau format de Manifeste et de liste de Manœuvres: deux colonnes, une pour les tirages et une pour les taches, avec les Noms de Voie centrés dans chaque colonne.
  • Ajoutez des en-têtes dans les Manifestes et les listes de Manœuvres décrivant chacun des attributs de Wagon. Sélectionnez "Impression d'En-têtes" dans les "Options d'Impression du Manifeste". Vous pouvez également personnaliser l'intégralité du texte de l'en-tête: Panneau "Options d'Impression du Manifeste" - > Outils - > Menu Modifier le texte de l'en-tête.

Assurez-vous d'appuyer sur après avoir modifié les sélections d'Impression du Manifeste.
