Si vous souhaitez que des Wagons soient livrés chez un client dans une séquence particulière ou avec une charge spécifique pour la livraison ou l'expédition, le programme vous permet d'appliquer un Horaire (Manifeste) à une Voie d'Embranchement. Il s'agit d'une fonctionnalité facultative qui ne doit pas être utilisée tant que vous n'êtes pas familiarisé avec la façon dont le programme génère les Manifestes et les Listes de Manœuvres.

Un Horaire fournit un contrôle très précis de l'ordre dans lequel des types de Wagons spécifiques seront livrés à une Voie d'Embranchement. Vous pouvez également spécifier le nom de l'Itinéraire pour le type de Wagon livrée. Par exemple, si vous souhaitez que deux wagons SP soient livrés, suivis d'un wagon-citerne chargé de "pétrole", vous pouvez créer un Horaire qui permettra à OperationsPro de faire exactement ça.

Vous pouvez également accéder à vos horaires depuis le menu du panneau Outils Emplacements. Le tableau des Horaires fournit le statut de chaque Horaire. Etant donné que vous pouvez affecter une planification à plusieurs Voies d'Embranchements, le tableau fournit également des informations de statut supplémentaires pour chacune des Voies utilisant cette Planification particulière. Le tableau vous permet également de modifier un Horaire une fois qu'il a été affecté à une Voie d'Embranchement.

Création d'un calendrier

Ouvrez d'abord la Fenêtre Emplacements.

Appuyez ensuite sur dans la colonne à droite de l'Emplacement que vous souhaitez planifier et la fenêtre Modifier l'Emplacement s'ouvre.

Pour planifier un Emplacement, sélectionnez dans la colonne de droite pour planifier la Voie spécifique à un Emplacement donné, puis la fenêtre Modifier (Embranchement ou autre type de Voie) ouverte.

En regard de Horaire facultatif dans le panneau Modifier Voie d'Embranchement. Appuyez sur devrait ouvrir le panneau Ajouter un Horaire.

Si la Planification existe déjà, la fenêtre Modifier l'Horaire Embranchement [nom de l'Embranchement] s'ouvre pour vous permettre de la modifier (ou de la supprimer).
Si aucune planification n'est sélectionnée, appuyez sur ouvre le volet Ajouter un Horaire pour l' Embranchement [nom de l'Embranchement].

Un Horaire peut fonctionner dans deux modes:

  • Séquentiel, si vous voulez des Wagons en séquence
  • Mode de Correspondance, si vous ne vous souciez pas de l'ordre de votre Wagon

Saisissez un nom pour votre Horaire, puis appuyez sur au bas du panneau. Plusieurs Voies d'Embranchements peuvent utiliser le même horaire tant que les Voies utilisent les mêmes types de Wagons et les mêmes Itinéraires.

Ajoutez maintenant votre première livraison pour cette Voie en sélectionnant un type de Wagon et en appuyant sur . Un nouvel élément de type "Boxcar" devrait apparaître dans votre liste d' Horaire:

Un horaire peut être spécifié à l'aide des menus déroulants.
Si vous le souhaitez, spécifiez une Itinéraire particulier pour le type de Wagon que vous avez entré ou laissez le champ Itinéraire vide si vous ne vous souciez pas du nom de l'Itinéraire.
Vous pouvez également spécifier le chargement de Wagon pour la réception et l'envoi.

Le panneau Ajouter/Modifier un Horaire a une nouvelle colonne "Aléatoire" avec un menu de 8 éléments. La valeur par défaut est [Vide] (désactivé) et il existe des valeurs comprises entre 5 et 50. Si vous sélectionnez 5, le programme indique au programme de réduire de manière aléatoire le nombre de déplacements de Wagon à cet emplacement jusqu'à 5%. Donc, si vous demandez 10 mouvements, vous pouvez obtenir 10 ou 9 mouvements. Si vous avez sélectionné 50, vous pouvez obtenir un nombre quelconque de déplacements compris entre 10 et 5. N'oubliez pas d'appuyer sur "Enregistrer l'Horaire" pour enregistrer votre réglage.

Si vous souhaitez que plusieurs Wagons de ce type soient livrées consécutivement, augmentez le nombre indiqué dans le champ Nombre. Le nombre maximum est 10 et le minimum est 1. Répétez en sélectionnant les types de Wagon et en appuyant sur jusqu'à ce que votre Horaire de livraison vous convienne. Appuyez sur et fermez le panneau Ajouter/Modifier Horaire lorsque vous avez terminé.

Notez que seuls les types de Wagon et les Itinéraires acceptés par cette Voie d'Embranchement sont disponibles. Si un type de Wagon ou un Itinéraire n'est pas disponible, vérifiez les types de Wagons et d'Itinéraires configurés pour cette Voie et Emplacement dans les volets respectifs.

Si vous souhaitez une charge spécifique pour vos Wagons, vous pouvez les créer à l'aide de Modifier les attributs de Wagon pour le Wagon Charger. Chaque nouveau type de chargement personnalisé est associé au type de Wagon. Par exemple, vous pouvez avoir "Vis" pour un wagon couvert et "Huile" pour un wagon-citerne. OperationsPro livrera uniquement les Wagons avec une "charge programmée" à une Voie d'Embranchementr ayant une planification correspondante qui demande également cette charge particulière. Toutefois, les Wagons avec une charge programmée peuvent toujours être placées dans les voies de triage, de coulisse et d'échange.

Vous pouvez éventuellement utiliser vos Voies de stockage intermédiaire pour vider les wagons avec une charge "Planification". Vous pouvez également créer des chargements "Planification" à l'aide de Voies de coulisse.
