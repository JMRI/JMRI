Un Itinéraire est une série d'Emplacements où un train passera. Après avoir ajouté des Emplacements à un Itinéraire, vous pouvez décider si le train s'arrêtera à un emplacement pour manœuvrer. Les Itinéraires peuvent être point à point ou aller-retour.

Panneau de Démonstration des opérations indiquant les Emplacements et les voies de connexion sur lesquelles des Itinéraires pourraient être définis:

Tous les Itinéraires de votre répertoire OperationsPro sont répertoriés dans la fenêtre Itinéraires avec leur statut. "Orphelin" signifie qu'il n'a pas encore été affecté à un train.

Corps du Volet Itinéraires

Le volet Itinéraires répertorie tous les Itinéraires définis pour votre réseau:

colonne - Itinéraire L'identifiant est attribué par OperationsPro
colonne - Nom de l' Itinéraire
Colonne Commentaire: tous les commentaires que vous souhaitez inclure pour chaque Itinéraire
Colonne Etat: indique l'état de l'Itinéraire.
  • OK - L'Itinéraire est attribué à un Train
  • Orphelin - L'Itinéraire n'a pas été attribué à un train
bouton colonne - permet de modifier les attributs d'Itinéraire sélectionné.

Bas du Volet des Itinéraires

Trier par Nom id établit l'ordre de tri des Itinéraires dans la liste.

ouvre la fenêtre Ajouter Itinéraire vous permettant d'ajouter des Itinéraires à votre base de données Operations.
