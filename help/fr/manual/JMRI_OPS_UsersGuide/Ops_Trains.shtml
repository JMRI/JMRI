La fenêtre Trains s'ouvre. Elle vous permet de créer des Trains à exécuter sur votre réseau à l'aide des éléments que vous avez définis dans ce programme, puis de les enregistrer sur les différentes listes. Cette fenêtre est sélectionnée dans la fenêtre principale de DecoderPro® ou PanelPro™ Outils - > Opérations - > Trains ou, le cas échéant, Exploitations - > Trains.

Barre de Menu

Le corps principal
Liste de tous les Trains assemblés. Chaque Train est associé à un bouton Aperçu, Déplacement et Edition.


Heure ou identifiant (dépend de Montrer en bas de la fenêtre) indique l'heure de départ du Train

La colonne Assembler est cochée pour être incluse dans la version actuelle

Fonction, cliquez sur le bouton Assembler et le bouton Aperçu est affiché dans Fonction, cliquez sur le bouton fonction Aperçu et le bouton Aperçu avant impression, affiche le manifeste des Trains. Si Assembler est affiché. Il assemblera le Train et affichera un Aperçu.
La colonne Nom indique le nom du Train
La colonne Description décrit les itinéraires des trains
La colonne Itinéraire indique l'itinéraire actuel du Train

La colonne Départ indique le lieu de départ

La colonne Arrivée indique l'emplacement du terminal

La colonne Actuel indique l'emplacement actuel

La colonne Statut indique le statut du Train

La colonne Action indique Déplacer à chaque fois que vous appuyez dessus, avance d'un emplacement sur l'itinéraire du Train est déplacé jusqu'à ce qu'il soit terminé.
La colonne Modifier contient le Modifier permet d'ouvrir la fenêtre Modifier un Train.
