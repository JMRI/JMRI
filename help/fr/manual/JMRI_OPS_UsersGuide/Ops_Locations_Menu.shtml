      Ouvre le panneau Planifications. Si vous souhaitez que les Wagons soient livrés dans un ordre particulier ou avec un chargement spécifique pour la livraison ou l'expédition, le programme vous permet d'appliquer une planification à une voie de garage. Ceci est une fonctionnalité optionnelle et ne doit pas être utilisée avant que vous ne vous connaissiez bien la façon dont le programme génère les manifestes et les listes de manœuvres.
