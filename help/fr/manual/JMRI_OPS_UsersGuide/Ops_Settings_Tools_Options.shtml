La fenêtre Options est ouverte à partir de Paramètres > Outils et vous permet de définir plusieurs options pouvant être modifiées.

Normal Agressive Il existe deux options d'Assemblage, Normale et Agressive. Normal est recommandé lorsque vous débutez, car les Assemblages normaux garantissent que lorsqu'un wagon est livré (déposé) par un train, il y aura un espace de voie disponible pour le wagon.

Les Assemblages Agressifs permettent de disposer de l'espace disponible sur la voie dès qu'un wagon est planifié pour une prise en charge (enlèvement).

Notez qu'une fois que vous avez assemblé un train, l'option d'assemblage ne peut pas être modifiée tant que tous les trains ne sont pas terminés ou réinitialisés.
Service de Manœuvre
 Cet ensemble d'options contrôle les mouvements locaux du wagon. Normalement, il est préférable de ne pas permettre à un wagon de passer directement d'un embranchement à un autre au même endroit. Ce type de mouvement est très irréaliste.Si vous avez besoin de ces types de transferts, cochez la case appropriée.

Coulisse

Activer les restrictions de Type, de Ligne et dans le transfert

Cette option limite les trains aux voies en coulisse qui n'acceptent que les noms de type de matériel roulant, de ligne et de charge du train. Cela est utile lorsque vous souhaitez que les trains soient livrés sur des voies en coulisse en fonction des restrictions du matériel roulant du train. Toutefois, si vous souhaitez que certains types de wagons, chargement de wagons ou nom de ligne soient transportés par un train mais ne soient pas livrés à la coulisse désélectionnez cette fonction

.

Rendre la voie de départ disponible après l'assemblage du train (Grisée en mode Normal)
Lorsque cette option est sélectionnée, la voie de départ en coulisse devient disponible pour les autres trains immédiatement après que le train au départ soit assemblé. Pour les trains qui reviennent à la même coulisse, cette option permet au train de revenir sur la même voie que sur laquelle il est parti. Cette option est uniquement disponible lorsque vous utilisez le mode agressif pour l'assemblage de trains.

Demander une Voie de Départ à partir de la Coulisse

 si vous souhaitez que le programme vous demande pendant l'assemblage du train, de quelle voie un train doit partir sélectionnez cette option. Normalement, pendant l'assemblage du train, le programme sélectionnera la voie la moins récemment utilisée pour déterminer la voie de départ. Notez que le programme ne vous demandera que s'il y a deux trains ou plus disponibles pour la coulisse qui répondent aux exigences du train.
Demande le Chemin d'arrivée depuis la Coulisse
Si vous souhaitez que le programme vous demande que le programme vous demande pendant l'assemblage du train, par quelle voie un train Arrive à la coulisse , sélectionnez cette option. Normalement, pendant l'assemblage du train, le programme sélectionnerat la voie la moins récemment utilisée lors de la détermination de la voie sur laquelle il doit arriver. Notez que le programme ne vous demandera que s'il y a deux trains ou plus disponibles pour la coulisse qui répondent aux exigences du train.

Options de Routage de Wagon

Activer le Routage des Wagons
 Lorsque le routage des Wagons est activé, le programme utilise plusieurs trains pour livrer un wagon à sa destination finale. Il est recommandé de laisser l'option activée. Lorsque le routage de wagon est activé, vous pouvez utiliser un Horaire pour livrer un wagon à un emplacement spécifique. L'algorithme de routage de wagon cherchera l'itinéraire le plus efficace pour le wagon. L'algorithme essaiera de trouver un itinéraire en utilisant un maximum de cinq trains pour effectuer le déplacement du wagon. Notez que le programme n'abandonne jamais d'essayer de router un wagon. Si une "destination finale" est attribuée à un wagon, le programme essaiera de le router après chaque mouvement.
Activer le Routage de Wagon via les Triages
Vérifiez si vous souhaitez autoriser l'acheminement des wagons par l'intermédiaire de vos triages. Vous pouvez alors utiliser les triages si vous le souhaitez. route.

Options de journalisation

Si vous souhaitez que le programme d'opérations enregistre vos mouvements de Locomotive, de Wagon ou de Train dans un fichier, cochez les cases appropriées. Le programme créera un nouveau répertoire appelé logger dans le répertoire des operations. Pour chaque session du jour, le programme créera un nouveau fichier (.csv) en utilisant la date du jour.


Pour créer votre propre manifeste personnalisé à l'aide d'un fichier (.csv) séparé par des virgules, sélectionnez cette option. Vous pouvez utiliser un programme tel qu'Excel pour créer un manifeste à votre goût. Le manifeste se trouve dans le répertoire des operations dans un répertoire appelé csvManifests. Chaque ligne du fichier commence par un code de deux ou trois lettres, suivi de la description du code, puis des paramètres associés au code.

Activer le Champs d'identification ( radio-fréquence (RFID): cette option permet de renommer la RFID selon vos préférences et de définir les valeurs souhaitées.
