Manuel JMRI® OperationsPro™

Introduction

Le présent manuel traite des versions 3.10 d'OperationsPro 3.6. Aide en ligne relative à la version actuelle de JMRI® OperationsPro est ici

JMRI OperationsPro est contenu dans les modules DecoderPro, PanelPro et SoundPro et est accessible à partir de la barre de menus principale. Dans le menu Opérations, vous pourrez construire des trains avec tout votre matériel roulant, les suivre sur vos panneaux JMRI, imprimer les documents à distribuer à votre équipe et bien plus encore.

S'il s'agit de votre première visite au manuel d'OperationsPro, nous vous suggérons de de parcourir le tour d'OperationsPro® pour avoir un aperçu de ce que nous sommes sur le point de couvrir.

JMRI est une suite de programmes multi- plateformes basée sur Java destinée aux modélistes ferroviaires. Il contient des applications modulaires pour:

  • La programmation des décodeurs DCC (Digital Command Control) (DecoderPro®)
  • Dessin de panneaux basés sur ordinateur pour la représentation d'une maquette de voie ferrée (PanelPro™)
  • Contrôle de la mise en page (Editeur de mise en page)
  • Création de manifestes de train pour reproduire les opérations ferroviaires (OperationsPro™)
  • Contrôle des signaux sur la mise en page
  • Contrôle de l'éclairage et du son dans la pièce, etc. (SoundPro™)

Toutes ces fonctionnalités sont contenues dans une installation.

Remarque: la plupart des captures d'écran de ce manuel proviennent d'un ordinateur Windows utilisant l'option d'interface utilisateur "Métal". Bien que vos écrans diffèrent probablement par certains détails de mise en page, ils seront essentiellement les mêmes.

    Modifications pour JMRI® OperationsPro 3.8 et 3.10

    Nouveauté 3.8

    • Le format de la base de données OperationsPro a changé avec la v3.8. Auparavant, vous pouviez charger un fichier créé avec la dernière version et le relire dans une version t rès ancienne d'Operations. Á partir de la version 3.8, la compatibilité ascendante est limitée aux versions 3.4 et 3.6. Soyez donc vigilant si vous souhaitez utiliser une ancienne configuration avec laquelle vous souhaitez partager des fichiers OperationPro.
    • Les manifestes de train sont disponibles au format JSON (et à nouveau sur le serveur Web).
    • Nouveau format de manifeste et de liste de substitution: deux colonnes, une pour les tirages et une pour les taches, avec les noms de voie centrés dans chaque colonne.
    • Ajoutez des en-têtes dans les manifestes et les listes de commutation décrivant chacun des attributs de wagon. Sélectionnez "Impression d'En-têtes" dans les "Options d'impression du manifeste". Vous pouvez également personnaliser l'intégralité du texte de l'en-tête: Panneau "Options d'impression du manifeste" ->Outils-> Menu Modifier le texte de l'en-tête.
    • Le tri pour les déplacements locaux a changé: si vous triez par voie, le programme triera les wagons par nom de voie. Si vous effectuez un tri par destination, le programme triera les wagons par nom de voie.
    • La création automatique de locomotive est constituée. Voir les options de construction du train
    • Ajustez toutes les heures de départ de votre train selon un nombre d'heures sélectionné. Trains -> Outils -> Modifier les horaires de départ des trains.
    • Modifiez ou affichez deux emplacements simultanément.
    • La sous-fenêtre Modifier le train affiche les restrictions de route ou de charge.
    • Configurez le rapport de construction de manière à toujours prévisualiser via le panneau Trains > Outils > Menu Options du rapport de construction. Économise du papier!
    • Panneau Wagons > Outils > Afficher les cases à cocher et > Le menu Réinitialiser les cases à cocher ajoute une colonne Sélectionner dans la liste Wagons. Utile comme bloc-notes lors de l'audit des wagons.
    • Spécifiez un chargement personnalisé lors de l'utilisation de la fonction de renvoi lorsque vide (RWE) pour une wagon.
    • Avancé: la façon dont OperationsPro fonctionne avec les scripts a considérablement changé. Si vous utilisez des scripts, vous devrez les modifier.

    Nouveauté 3.10

    • Imprimer ou prévisualiser un seul lieu à partir du panneau "Modifier le lieu".
    • La voie alternative pour une voie d'embranchement fonctionne désormais sans planifications ni charges personnalisées
    • Ajoutez le numéro de locomotive de tête à la description d'un train à l'aide de la balise {1}.
    • Des déplacements aléatoires à n'importe quel endroit de l'itinéraire d'un train. Le panneau Éditer Itinéraire contient une nouvelle colonne "Aléatoire" avec un menu de 10 éléments. La valeur par défaut est "Off" ou est désactivée, et les valeurs sont comprises entre 10 et 100. Si vous sélectionnez 10, il indique au programme de réduire manière aléatoire le nombre de de wagons qui se déplace à l'emplacement jusqu'à 10%. Donc, si vous avez demandé 10 coups, vous pourriez en avoir 10 ou 9. Si vous avez sélectionné 50, vous pouvez obtenir un nombre quelconque de mouvements entre 10 et 5. Si vous sélectionnez 100, vous pouvez obtenir un nombre quelconque de mouvements entre 10 et 0. Assurez-vous d'appuyer sur "Enregistrer l'itinéraire" pour enregistrer votre paramètre.
    • Copier un emplacement, un panneau d'emplacement -> Outils -> Copier l'emplacement.
    • Sélectionnez le type de tri directement dans le volet Imprimer ou Aperçu de wagon.
    • les noms de lieux et de voie de l'inventaire de votre wagon sont imprimés entièrement au lieu de 38 caractères seulement.
    • Autres options et info-bulles pour le panneau Modifier le texte du manifeste et Modifier le texte de la liste de substitution. Affichez le nom et la description d'un train dans la plupart des messages.
    • Bouton "Vérifier les destinations" dans le panneau "Modifier les voies de destinations" pour les voies Inter/Échange. Lorsque vous appuyez sur cette touche, tous les types de wagons, itinéraires et chargements servis par la Voie sont vérifiés et vous permettent de vérifier si les destinations sélectionnées peuvent également desservir les mêmes types, itinéraire et chargements. Vérifie également qu'il existe un train pouvant desservir la voie et les destinations de tous les types de wagons. La vérification s'arrête lorsque le premier problème est détecté.
    • Panneau des emplacements -> "Modifier les emplacements par chargement de wagon". Vous pouvez maintenant voir et ajuster rapidement quelles voies d'un emplacement peuvent accepter certaines charges. Indiquez également les charges pouvant être envoyées pour les voies de préparation.
    • "Afficher les types de wagons desservis" dans la fenêtre "Trains". Sélectionnez un train dans le menu contextuel supérieur et un type de wagon à gauche pour afficher un aperçu de l'emplacement où ce type de wagon peut être traité:
    • Panneau Trains - > Outils -> "Tableau de Couleurs des Trains " pour sélectionner manuellement une couleur d'arrière- plan pour la ligne d'un train dans le panneau "Trains". Il existe également une option "Auto" qui vous permet de sélectionner une couleur d'arrière-plan lorsqu'un train est construit, que la construction échoue ou est terminée.
    • Le texte des manifestes et des listes de remplacement est entièrement personnalisable, plus de titres figés.
