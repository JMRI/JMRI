Importer des locomotives d'un Inventaire

Ce menu ouvre une boîte de dialogue d'importation du système. Si vous avez déjà un Inventaire de Locomotives DecoderPro® créé, par exemple. lors de la programmation de décodeurs, vous pouvez importer cette liste dans le répertoire OperationsPro. Toutes les Locomotives qui ne figurent pas déjà dans votre répertoire OperationsPro seront importées et un message s'affichera pour vous informer du nombre de locomotives importées:

Ainsi, vous pouvez utiliser cet outil chaque fois que vous ajoutez des locomotives à votre liste des Locomotives DecoderPro® . La liste des Locomotives DecoderPro⪚ n'a pas la longueur de la Locomotive, une longueur par défaut est donc entrée pour vous. Vous devez corriger manuellement la longueur de la Locomotive dans OperationsPro si vous souhaitez que le programme calcule correctement les longueurs de votre train.

Maintenant que nous avons importé votre liste de locomotives DecoderPro dans le répertoire OperationsPro, vous devez modifier les entrées pour ajouter les attributs requis pour OperationsPro mais non nécessaires dans le fichier de listes de locomotives DecoderPro lors de la programmation des décodeurs.

Ces Machines ont été importées de la liste JMRI, nous devrons donc les modifier pour définir la Longueur, le Couplage, l'Emplacement appropriés. Le Bouton Modifier dans la colonne Modifier ouvre la fenêtre Modifier les Locomotives et vous permet de modifier les attributs de locomotive.

Le bouton Définir de la colonne Définir de chaque locomotive de la liste vous permettra de définir l'Emplacement actuel de la locomotive.
