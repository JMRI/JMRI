Á droite de chaque attribut de Locomotive (Ligne, Modèle, Type,Longueur, Chevaux, Couplage, Propriétaire), cliquez sur pour vous permettre de modifier la liste des attributs disponibles pour les utiliser dans une définition de Locomotive. Utilisez ceci pour , ou les attributs disponibles pour une Locomotive.

Pour ajouter un nouvel attribut Locomotive, entrez le texte souhaité dans la zone de texte et appuyez sur le bouton . Lorsque vous entrez pour la première fois un nouvel attribut de locomotive, celui-ci s'affiche au début de la liste. Plus tard, lorsque vous redémarrez le programme, il sera trié dans le bon ordre.

Pour supprimer un attribut Locomotive, sélectionnez celui que vous souhaitez supprimer et appuyez sur le bouton .

Si vous devez modifier un attribut de locomotive spécifique (Ligne, Modèle, Type, etc.) dans l'ensemble de votre Inventaire de Locomotives, vous pouvez utiliser le bouton situé en bas à droite du panneau Modifier l'Attribut de Locomotive.

Par exemple, si vous souhaitez modifier l'attribut "UP" du nom de route de locomotive pour "UPS" pour tous wagons. Appuyez sur le bouton pour l'une de vos voitures dans le panneau Voitures. La boîte de dialogue Remplacer tout? s'affichera avec ou option. Entrez la nouvelle route "UPS" dans la zone de texte, sélectionnez "UP" dans la zone de menu, puis appuyez sur le bouton . Tous les Wagons dont le nom de Ligne est "UP" porteront désormais le nom "UPS".
