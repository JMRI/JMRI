vous permet de supprimer les Locomotives et les wagons affectés à un Train. Utile lorsque vous voulez essayer diverses options d'Assemblage et que vous ne voulez pas déplacer physiquement les locomotives et les wagons sur le réseau.

Appuyez sur pour enregistrer votre train. Le programme créera un fichier (OperationsTrainRoster.xml) dans votre répertoire JMRI dans un dossier appelé "operations". Un fichier de sauvegarde contenant les derniers paramètres de tous vos trains est également créé à chaque fois que vous appuyez sur .

Notez que le Train modifié est maintenant affiché dans la Fenêtre Trains .
