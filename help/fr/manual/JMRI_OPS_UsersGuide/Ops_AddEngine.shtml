La fenêtre Ajouter une locomotive s'affiche.

Les menus Fenêtre et Aide de la sous-fenêtre Ajouter une locomotive correspondent à la norme des menus JMRI® utilisés sur la plupart des fenêtres JMRI.

Sélectionnez la Ligne pour la machine. Si l'abréviation de nom de ligne souhaitée n'existe pas, appuyez sur pour ouvrir le dialogueEditer les Attributs d'une Locomotive. pour la Ligne.

Entrez le Numéro de Ligne de la Locomotive Utiliser pour supprimer votre saisie si vous tapez une erreur.

Sélectionnez le modèle de locomotive Si le modèle dont vous avez besoin ne figure pas dans la liste, appuyez sur et le La boîte de dialogue Modifier les attributs de locomotive du modèle s'ouvre.

Le programme vous permet d'organiser vos moteurs par Modèle. Chaque Modèle ne peut être associé qu'à un Type de machine, à sa longueur, à son poids et à sa puissance. Par exemple, si vous attribuez le nom de modèle "RS11" à un ensemble de locomotives, lorsque vous modifiez la longueur de l'une des RS11, celle-ci change également pour les autres.

Sélectionnez maintenant le Type de la locomotive Si le type dont vous avez besoin ne figure pas dans la liste, appuyez sur et le La boîte de dialogue Modifier les attributs de locomotive pour le Type s'ouvre.

Sélectionnez maintenant la Longueur de la locomotive. Si la longueur dont vous avez besoin ne figure pas dans la liste, appuyez sur et La boîte de dialogue Modifier les attributs de locomotive pour la Longueur s'ouvre.
Sélectionnez l'Emplacement actuel (de départ) de la locomotive et la Voie de la locomotive
Facultatif: entrez le poids de la locomotive en tonnes

Entrez maintenant la Puissance de la machine. . OperationsPro utilise la puissance de la locomotive, son poids et la pente de la voie pour calculer le nombre de moteurs nécessaires pour utiliser la fonction "Auto" lors de la construction de trains.

Synchroniser avec Couplages NCE

si vous utilisez NCE comme centrale de commande DCC, vous pouvez demander au programme de synchroniser vos numéros de Couplage avec vos locomotives. Le programme lira tous les Couplages de votre système NCE, puis recherchera les locomotives correspondantes dans votre liste OperationsPro. Le programme attribuera le Nom de Couplage nce_xxx où xxx est le numéro de couplage NCE. Pour synchroniser vos couplages, sélectionnez "Synchroniser avec Couplage NCE " dans le menu "Outils" (grisé si aucune connexion NCE n'est définie).
