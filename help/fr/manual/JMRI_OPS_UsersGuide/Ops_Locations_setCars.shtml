Définir les Wagons (plusieurs)

Vous pouvez sélectionner Définir les Wagons dans le menu Wagons du menu Outils. Cela vous permet de modifier l'Emplacement, la Destination, le Statut ou l'Affectation de Train pour un groupe de Wagons. Cet outil fonctionne de la même manière que l'outil Définir la position d'un Wagon décrit ci-dessous, à la différence que plusieurs Wagons sont modifiés en une seule action.
Sélectionnez les Wagons de votre choix dans le tableau Wagons en appuyant sur Maj/Commande + en cliquant sur leur nom, puis sélectionnez Définir les Wagons dans le menu Outils. La première Wagon sélectionné devient le Wagon par défaut. Si vous appuyez sur , tous les autres Wagons sélectionnés auront les mêmes paramètres que ceux par le wagon par défaut. Assurez-vous de dans la fenêtre Wagons si vous souhaitez conserver votre changements.

Le message suivant s'affiche lorsque vous choisissez Définir les Wagons si (l'un des) Wagons que vous avez sélectionnés font partie d'un train. Cela vous avertit que lorsque vous enregistrez les modifications à partir de la fenêtre Définir les Wagons, ces Wagons ne font plus partie d'un train précédemment assemblé.

État

Emplacement inconnu
Fait en sorte que le programme ignore le Wagon lors de l'assemblage d'un nouveau train; sur les listes des Manifestes et de Manœuvres, le programme listera le Wagon comme "égaré". Cette sélection désactive toutes les autres sélections de ce volet.

Hors service
Lorsque cette option est sélectionnée, le programme ignore le Wagon lors de l'assemblage de nouveaux trains. De nombreux opérateurs disposent d'un suivi RIP (réparation sur place) et utilisent cette fonction pour conserver le service d'entretien jusqu'à ce que les réparations soient terminées.

Emplacement

Utilisez les champs Nom de l'Emplacement/Voie pour placer le Wagon sur le réseau Auto indique au programme de placer le Wagon n'importe où dans le prochaine assemblage.

Facultatif

Retourner quand il est vide

Sélectionnez l'emplacement auquel le Wagon doit revenir lorsqu'il est vide.
Auto indique un tracé valide pour un emplacement donné.

Charger

Vous pouvez également modifier la charge du Wagon.

Définir le Wagon

Après avoir créé un Wagon, vous pouvez définir l'emplacement du Wagon et éventuellement la destination et le train du Wagon à l'aide du trouvé pour chaque Wagon dans la fenêtre Wagons. Le programme génère normalement la destination et le train attribué au Wagon, mais vous pouvez utiliser ce panneau et ses options pour forcer un Wagon vers un emplacement spécifique à l'aide d'un train spécifié.

Le message suivant s'affiche lorsque vous choisissez Définir le Wagon si le Wagon fait partie d'un train. Cela vous avertit que lorsque vous enregistrez les modifications à partir de la fenêtre Définir l'emplacement des Wagons , ces Wagons ne font plus partie d'un train.

État

Emplacement inconnu: le programme ignore le Wagon lors de l'assemblage d'un nouveau train; sur les manifestes et les listes de manœuvres, le programme indiquera le Wagon comme étant égarée. Cette sélection désactive toutes les autres sélections dans cette fenêtre.

Hors Service, lorsque cette option est sélectionnée, indique à OperationsPro d'ignorer le Wagon lors de l'assemblage de nouveaux trains. De nombreux opérateurs disposent d'une piste RIP (réparation sur place) et utilisent cette fonctionnalité pour y maintenir le Wagon jusqu'à ce que les réparations soient terminées.

Emplacement

Utilisez la définition d'Emplacement pour placer le Wagon sur le réseau.

Retourner Quand il est Vide

Entrez un emplacement pour retourner le Wagon à vide. > Affiche automatiquement un tracé valide pour un emplacement donné.
