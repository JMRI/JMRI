L'Aide actuelle sur les Préférences JMRI est ici.

Dans PanelPro ou DecoderPro, dans la barre de menus principale, cliquez sur le menu Édition -> Préférences ... (Fenêtres) ou cliquez sur Préférences ... dans le menu de l'application PanelPro (OS X). Cela ouvre le panneau Préférences (le panneau peut s'ouvrir automatiquement la première fois que vous exécutez JMRI®).

Le panneau Préférences est divisé en deux volets: à gauche, une liste des groupes de préférences pouvant être définis. En cliquant sur l'un des onglets, vous ouvrez cet ensemble d'options Préférences dans le volet de droite, où les options peuvent être cochées ou modifiées.
Remarque: N'oubliez pas de cliquer sur pour finaliser vos paramètres, puis redémarrez JMRI pour les appliquer.

Pour un utilisateur novice, la partie la plus importante au départ est l'onglet Connexions, dans lequel vous décrivez comment votre ordinateur communiquera avec votre Centrale de Commande DCC. Vous pouvez laisser la plupart des autres onglets à leurs paramètres initiaux ou "par défaut" jusqu'à ce que vous ayez besoin de modifier les paramètres à votre guise.

Le volet Préférences - > Connexions permet d'accéder aux informations de configuration de base pour connecter votre ordinateur à votre système DCC. Vous pouvez configurer JMRI OperationsPro pour qu'il fonctionne comme un générateur de liste de Manœuvres numérique utilisant un type de connexion "simulateur", ou profiter de la suite JMRI complète en connectant JMRI® OperationsPro à votre réseau DCC pour créer et exploiter des trains équipés de décodeurs DCC. et les suivres sur l'écran de votre ordinateur.

Volet Paramètres par défaut

Affiche les paramètres par défaut qui dépendent du (des) système (s) utilisé (s). Sélectionnez l'une de vos connexions comme connexion par défaut pour les régulateurs de service et les programmeurs de services. Avec une seule connexion (sous l'onglet Connexions), ce sera automatiquement la valeur par défaut et ce volet ne sera pas pertinent.
