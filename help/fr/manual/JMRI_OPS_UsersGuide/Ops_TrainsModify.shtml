Ouvert depuis le menu Trains Outils. Utilisez cet outil pour modifier rapidement le type de Wagons que votre Train utilisera. C'est utile lorsque vous ajoutez un nouveau type de Wagon à votre Inventaire.

Copier

Vous pouvez également Copiez la façon dont tous les Wagons d'un type de Wagon existant doivent être utilisés par votre Train et appliquez ces règles à un autre type de Wagon. Pour qu'un type de Wagon se comporte comme un autre, sélectionnez d'abord le type de Wagon que vous souhaitez copier. Le programme indiquera quels Trains peuvent utiliser ce type de Wagon. Ensuite, sélectionnez Copier.
Le type de Wagon apparaîtra maintenant à droite de la case à cocher Copier:

Sélectionnez maintenant le type de Wagon auquel vous souhaitez appliquer ces Trains à l'aide du menu contextuel Type et pour que les deux types de Wagon soient utilisés par les mêmes Trains.

Appuyez sur pour appliquer votre choix.
