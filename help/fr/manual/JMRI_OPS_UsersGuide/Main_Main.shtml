Toutes les fonctions JMRI, y compris le programme OperationsPro, sont accessibles via la barre de menus de DecoderPro® ou la fenêtre principale de PanelPro.

Toutes les fonctionnalités d'OperationsPro sont toujours disponibles à partir de Outils - > Opérations dans la barre d'outils principale JMRI.

L'écran principal DecoderPro® ci-dessous présente le menu facultatif Opérations (en le sélectionnant dans le volet Paramètres OperationsPro).

L'icône DecoderPro® et les informations de version DecoderPro® sont affichées. La version de Java installée et en cours d'utilisation est également affichée.

Barre de menus de la fenêtre principale

Le menu principal contient les menus qui vous permettent d'accéder à toutes les fonctionnalités de JMRI.

Ce manuel porte sur les fonctions et les menus qui s'appliquent à JMRI® OperationsPro version 3.x.
Veuillez consulter les manuels JMRI® pour d'autres fonctions.
