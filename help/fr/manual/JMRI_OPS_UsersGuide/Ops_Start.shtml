Utilisation de JMRI® pour l'Exploitation des Trains

Que fait OperationsPro?

Retour à l'Index

Utilisation de JMRI® pour l'Exploitation des Trains vous permet de créer des Manifestes de Wagons générés par ordinateur pour votre réseau. Un Manifestes ou une liste de Manœuvres détaille l'activité d'un train pendant une session d'exploitation. Le Manifeste fournit une liste de ramassage et de dépose des Wagons et indique où se trouve les Wagons et où ils pourraient être positionnés sur le réseau. Le programme vous permet de créer un Inventaire de Wagons, de définir des Emplacements sur le réseau et de créer des Itinéraires pour les Trains. L'Inventaires Wagons comprend des informations sur chaque Wagon, y compris le nom de l'Itinéraire, le numéro, le type de Wagon, la couleur, la longueur, le poids, la date de construction ou d'achat et le propriétaire. Les Trains se voient attribuer des Itinéraires qui définissent des Emplacements où les Wagons peuvent être récupérées ou placées. Les fonctions incluent la possibilité de contrôler les types de Wagons, les noms des Itinéraires et les chargements de Wagons desservis par un Emplacement, l'espace de la Voie disponible pour un Emplacement et la longueur maximale d'un train pouvant se situer entre deux Emplacements quelconques sur l'Itinéraire du train.

Les Emplacements peuvent avoir des Voies d'Embranchements, de Triages, en Coulisse et Garage Temporaire. Les Embranchements sont utilisés pour les industries de services et peuvent éventuellement recevoir des horaires personnalisées. Les Trains peuvent être configurés pour exiger un fourgon de queue. Les trains peuvent éventuellement être configurés de manière à nécessiter un Wagon de queue ou un Wagon avec un dispositif arrière clignotant (FRED). Le programme génère des Manifestes de Wagons pour chaque train et des listes de Manœuvres pour tous les lieux. Une liste de Manœuvres pour un emplacement particulier montre l'activit√© de tous les trains qui visitent cet endroit. Le programme peut également placer des icônes représentant des trains sur un panneau JMRI. Vous pouvez apprendre à l'icône à se déplacer sur le panneau avec le train d'Emplacement en Emplacement.

Le programme OperationsPro est inclus dans la suite d'outils du fichier JMRI®, donc lorsque vous installez et démarrez soit DecoderPro® ou PanelPro, vous trouverez les menus d'OperationsPro dans la barre de menus principale sous Outils> Exploitations ou en tant que menu Exploitations, si activé dans vos Paramètres d'OperationsPro.

Fichiers de Démonstration d'OperationsPro

Le programme OperationsPro fournit un ensemble de fichiers de démonstration pour vous aider à vous familiariser avec certaines des fonctionnalités de ce programme.

Chargez les fichiers de démonstration à partir du menu Outils dans la liste Opérations> Dans la fenêtre Paramètres, sélectionnez Charger les fichiers de démonstration. Les fichiers de démonstration seront chargés après le redémarrage de DecoderPro.

La sélection du chargement des fichiers de démonstration ferme DecoderPro® et il doit être redémarré pour ouvrir Demo Railroad.

Le programme créera une sauvegarde de votre configuration actuelle si vous en avez une, puis chargera les fichiers de démonstration. Redémarrez JMRI pour terminer le processus de chargement de la démo. Pour supprimer les fichiers de démonstration, utilisez la fonction Restaurer du menu Outils du Paramètres d'exploitation pour recharger la première date de sauvegarde, à savoir le formulaire d'inscription vierge.

Le programme fournit également un panneau simple pour montrer le fonctionnement des icônes Train. Chargez le Panneau avant d'assembler des trains en sélectionnant dans le menu principal JMRI, "Panels- > Ouvrir les Panneaux ...". Une fenêtre devrait apparaître. Ouvrez le répertoire "Operations" puis ouvrez le fichier xml "Operations Demo Panel".

Démarrage rapide

Dans la barre de menus de DecoderPro, sélectionnez le menu Outils. Au bas de la page, vous trouverez "Exploitations".

Il y a six éléments dans le sous-menu. Il est recommandé de commencer par Paramètres et de descendre à Trains. Nous vous recommandons également, lors de la définition de vos emplacements, d'utiliser d'abord uniquement les Voies Triage, Embranchement; essayez ensuite Coulisse et Garage Temporaire lorsque vous êtes plus à l'aise avec le fonctionnement d'OperationPro.

Chemins de fer britanniques

Le programme prend en charge un ensemble alternatif de terminologie ferroviaire pour nos modélistes britanniques. Pour utiliser les terminologies britanniques, accédez à Préférences avancées de JMRI (Edition - > Préférences ...), cochez la case Afficher les préférences avancées, puis sélectionnez Anglais (Royaume-Uni) dans la boîte de menu "Locale:". Le programme convertira également des centimètres en mètres lors de la saisie des longueurs de Wagon, de locomotive ou de lieu; ajoutez simplement "cm" à n'importe quelle longueur.
