Préparation de l'installation de JMRI®

Le progiciel JMRI comprend:

  • DecoderPro®
  • PanelPro™
  • SoundPro™

Deux versions du logiciel sont disponibles:

  • Version de production: version stable actuelle qui a été corrigée dans sa conception et pour son utilisation générale. Si vous venez de commencer, cette version est faite pour vous.
  • Version de développement: version en cours de développement pour les tests et utilisée pour valider les nouvelles fonctionnalités et les modifications apportées aux fonctionnalités existantes. Si vous connaissez JMRI et souhaitez contribuer au processus de validation et de test, utilisez cette version.

Vérifiez le matériel pris en charge pour vérifier que votre ordinateur peut exécuter JMRI.

Si possible, téléchargez le programme d'installation approprié pour votre système informatique:

  • Windows
  • Mac OS X
  • Linux

Installation de Java

Les versions de JMRI® peuvent être exécutées sur tout système exécutant Java 1.6.0 ou une version ultérieure, qu'il soit basé sur Macintosh, Windows ou Linux. Il faut que Java soit installé sur l'ordinateur. Vous pouvez obtenir Java en téléchargement gratuit (mais volumineux) à l'adresse suivante: java.com/download/
