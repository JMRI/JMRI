Options d'Assemblage
Normal Agressif
Des versions normales sont recommandées lorsque vous démarrez le programme pour la première fois. Les Assemblages normaux garantissent que lorsqu'un Wagon est livrée par un train, il y aura un espace de voie disponible pour le Wagon.

Les constructions agressives permettent de disposer de l'espace du Wagon dès qu'un Wagon doit être récupéré.

Notez qu'une fois que vous avez asssemblé un train, l'option de construction ne peut pas être modifiée tant que tous les Trains ne sont pas terminés ou réinitialisés.

Invite Voie de Départ de la Coulisse
Si cette option est sélectionnée, le programme vous demande quelle sera la voie de départ de la coulisse pendant l'assemblage du train. Notez que le programme ne vous le demandera que s'il y a deux Trains ou plus disponibles dans la coulisse qui répondent aux exigences du train.

Invite Voie d'Arrivée dans la Coulisse
Si cette option est sélectionnée, le programme vous demande quelle voie doit être utilisée pour l'arrivée lors de l'assemblage du train. Notez que le programme ne vous le demandera que s'il ya deux Trains ou plus disponibles dans l'étage qui répondent aux exigences du train.

Options de Routage des Wagons

Activer le Routage du Wagon
Lorsqu'il est activé, le programme utilisera plusieurs Trains pour livrer un Wagon à sa destination finale. Il est recommandé de laisser cette option activée. Une fois activé, vous pouvez utiliser Planifiez pour livrer un Wagon à un emplacement spécifique.
