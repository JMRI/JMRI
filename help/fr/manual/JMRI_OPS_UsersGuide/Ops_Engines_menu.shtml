Outils

  • Inventaire des Locomotives
    • Importer à partir de l'inventaire JMRI
    • Importer à partir d'un fichier
    • Exporter vers un fichier

      Enregistre le Répertoire dans votre dossier Operations\Export et affiche la boîte de dialogue Export Terminé. Le fichier est automatiquement exporté au format .csv.

    • Supprimer toutes les locomotives

      Ouvre la boîte de dialogue Supprimer toutes les locomotives?. OK supprimera toutes les locomotives de l'Inventaire dans operations; Annuler annulera la suppression.

    • Rêinitialiser le nombre de déplacements toutes les locomotives

      Permet de réinitialiser tous les compteurs de mouvements de locomotive.

    • Imprimer - Ouvre la fenêtre Imprimer de votre ordinateur et imprimera votre liste de locomotives.

    • Aperçu

      Ouvre la fenêtre Aperçu avant impression de votre ordinateur et affiche un aperçu de votre répertoire de locomotives.

  • Synchroniser avec les Couplage NCE (actif uniquement avec une connexion NCE; grisé pour LocoNet)

Opêrations: identique au menu Opérations DecoderPro®

Fenêtre: identique aux autres Menu JMRI® .

Aide: identique aux autres Menu JMRI®.
