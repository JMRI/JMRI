La fenêtre Ajouter un Train est ouverte à partir de la fenêtre Trains.

Dans le corps de la fenêtre

Nom Description
entrez un nom de Train et une description facultative, puis appuyez sur .

Les deux apparaîtront sur le Manifeste du Train. Le nom du Train s'affichera également sur l'Icône du Train. Les noms abrégés fonctionnent donc mieux.


Maintenant que toutes les autres options sont activées dans le volet, vous pouvez terminer le Train.

Sélectionnez Itinéraire Itinéraire que ce Train empruntera.

En cliquant sur à la droite de l'Itinéraire vous permet de modifier directement l'Itinéraire sélectionné. Une fois qu'un Itinéraire est sélectionné, une liste d'Emplacements que le Train visitera devrait apparaître. Vous pouvez désélectionner tout Emplacement sur lequel vous ne voulez pas que ce Train vienne.
Définissez l' Heure de Départ (hh: mm) , Heure à laquelle le Train quittera le premier Emplacement de son Itinéraire. Vous pouvez utiliser l'Heure de Départ pour trier vos Trains dans l'ordre dans lequel vous souhaitez les assembler.

Vous pouvez déterminer le type de matériel roulant utilisé par le Train. Sélectionnez le matériel roulant que le Train transportera en cliquant sur les cases à cocher appropriées. Une case cochée signifie que le Train assurera la maintenance de ce type de matériel roulant.

Sélectionnez les types de locomotive utilisées par ce Train en cochant tous les types de Machine requis.

vous pouvez contrôler le nombre de locomotives affectées à ce Train et, éventuellement, le nom du modèle et de l'Itinéraire de la Machine. Si vous souhaitez que deux ou plusieurs Machines soient affectés à un Train, ellels doivent faire partie d'un couplage. Si vous sélectionnez "Auto", le programme calculera le nombre de Machines requis pour ce Train en fonction du poids du Train, de la qualité de la voie et de la puissance des Machines.

Vous pouvez également exiger un fourgon de queue ou un wagon avec FRED (dispositif d'extrémité arrière clignotant) comme dernier wagon du Train. Vous pouvez également spécifier le nom de l'Itinéraire pour le Fourgon de Queue et le Wagon avec FRED. Si vous laissez le nom de l'Itinéraire vide, le programme essaiera de trouver un Fourgon avec le même nom de Itinéraire que la Machine.

vous permet de supprimer les Locomotives et les Wagons affectés à un Train. Cela peut être utile lorsque vous voulez essayer diverses options d'assemblage et que vous ne devez pas déplacer physiquement la locomotive et les wagons sur le tracé.

Appuyez sur pour sauvegarder votre Train. Le programme créera un fichier (OperationsTrainRoster.xml) dans votre répertoire JMRI dans un dossier appelé "operations". Un fichier de sauvegarde avec les derniers paramètres de tous vos Trains est également créé à chaque fois que est appuyé.

Notez que le Train est maintenant affiché dans la Fenêtre Trains.
