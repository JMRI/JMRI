Le menu Outil Modifier les emplacements par type de Wagon ouvre la boîte de dialogue Imprimer les emplacements de révision des types de Wagon, qui permet d'imprimer votre sortie système comme indiqué. Les emplacements sont des endroits de votre réseau où passent les trains pour récupérer ou déposer les Wagons et les Locomotives.
