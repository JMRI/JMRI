  • Liste des Wagons
    • Importer à partir d'un fichier
      Ouvre la boîte de dialogue Ouvrir, pour importer une liste de Wagons si vous en avez déjà un. une liste de Wagons dans une base de données. La fonction d'importation nécessite un fichier texte ACSII, avec une ligne pour chaque Wagon. L'importation de Wagon nécessite quatre attributs et l'ordre est le suivant: Numéro de Wagon, nom de la Ligne, Type et Longueur.
      Un exemple de format correct:

      336 GCR Flat 40
      655579 UP Wagon couvert 40

      Vous pouvez éventuellement inclure des attributs supplémentaires dans l'ordre suivant: Poids, Couleur, Nom du Propriétaire, Date de Fabrication et Lieu.
      Consultez le fichier de menu Aide du volet Wagons pour plus d'informations.
