Le Programme d'Explotation est exécuté à partir de DecoderPro® Menu activé dans les Préférences d'Exploitation.

Une fois que vous avez téléchargé et installé le logiciel DecoderPro®, il suffit d'ouvrir le programme de la manière habituelle pour votre système d'exploitation. Sur MS Windows, l'installateur placera une icône sur votre bureau. Pour plus de détails, reportez-vous à l' aide à la configuration de JMRI.

Double-cliquez sur l'icône du programme à ouvrir. Dans ce cas, nous ouvrirons DecoderPro®.

La boîte de dialogue suivante apparaîtra brièvement pendant le chargement du programme:

Vous devriez voir ceci ou un écran d'ouverture similaire affiché après que l'un des programmes JMRI® ne commencent:

Cet écran peut prendre un certain temps, en particulier sur les ordinateurs les plus anciens et les plus lents. N'oubliez pas que vous exécutez essentiellement le programme sur un émulateur J ava via le logiciel de votre système d'exploitation natif. Les nouveaux ordinateurs peuvent le faire assez rapidement, mais les plus anciens auront l'impression de prendre une éternité. Soyez patient - cela finira par arriver! Même sur des ordinateurs plus anciens, le temps de réponse est assez rapide une fois le programme terminé.

Si c'est la première fois que vous exécutez le programme après avoir installé le logiciel, l'écran Préférences s'affichera automatiquement p our vous permettre de configurer le système pour votre configuration particulière. De plus, si vous modifiez la configuration de votre système, OU si vous avez votre ordinateur portable à l'écart du réseau, mais que vous souhaitez jouer avec JMRI, vous voudrez peut-être essayer le mode simulateur. Alors cliquez sur Suivant, nous verrons "Préférences".
