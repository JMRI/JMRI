La fenêtre Définir l'Emplacement d'une Locomotive vous permet de remplacer le programme et de définir une locomotive à un Emplacement défini dans votre base de données Operations.

Le Menu est la norme pour DecoderPro®.

La Locomotive affiche la machine sélectionnée.

Le Type est affiché

Statut
Hors service
Met la locomotive hors service et ne sera pas affectée par le programme Build

Emplacement
Nom et Suivre , vous pouvez configurer la Locomotive à n'importe quel Emplacement défini de votre réseau.
Auto: limite les sélections à des Emplacements valides.

Facultatif - Normalement défini par Programme -

Destination, définissez éventuellement le Nom du lieu et de la voie .

Train définissez éventuellement le Train.

appuyez sur pour enregistrer les Emplacements de Locomotive que vous avez définis.

Fermez la fenêtre lorsque vous avez terminé.
