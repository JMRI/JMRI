OperationsPro: fenêtre Modifier les Emplacements par type de Wagon

Retour

Cet outil s'ouvre à partir du menu Outil de la fenêtre (Modifier).
Utilisez cet outil pour modifier rapidement le type de Wagons que vos Emplacements et Voies effectueront. C'est très utile pour ajouter un nouveau type de Wagon à votre liste.

Sélectionnez d'abord le type de Wagon que vous souhaitez modifier dans la liste déroulante. Lorsque le type est sélectionné, notez que tous les Wagons de ce type aux emplacements sélectionnés seront cochés.

Sélectionnez les Emplacements et les Voies que vous souhaitez desservir - ou désélectionnez ceux que vous ne souhaitez pas desservir - et appuyez sur pour modifier vos Emplacements et Voies.

Vous pouvez également Copier comment tous les Wagons d'un type existant doivent être utilisés par vos sites et appliquer les règles à un autre type de Wagon. Pour faire en sorte qu'un type de Wagon se comporte comme un autre, sélectionnez d'abord le type de Wagon que vous souhaitez copier, puis sélectionnez le Copier.

Le type de Wagon apparaîtra à droite de la case à cocher Copier.

Sélectionnez maintenant les types de Wagon auxquels vous souhaitez appliquer ces paramètres et appuyez sur pour que les deux types de Wagons soient utilisés par les mêmes emplacements et Voies.
