Le Programmeur Simple est ouvert à partir de la Page principale de DecoderPro Menu Action. Le Programmeur Simple vous permet de lire ou d'écrire des valeurs de CV dans les décodeurs DCC un par un.

Avant de lire une valeur de CV, vous devez entrer le numéro de CV (adresse) dans la zone de texte Numéro CV. Si vous écrivez une valeur de CV, vous devez entrer le numéro de CV, puis un valeur dans la zone de texte Valeur. Vous pouvez entrer la valeur sous forme décimale ou hexadécimale selon l'option sélectionné dans la zone Valeur est:.

Sur le côté gauche de la fenêtre, vous pouvez sélectionner le Mode de Programmation. Tout Mde non supporté par votre Centrale de Commande sera grisé.

Si vous sélectionnez le mode Ops, qui programmes sur la voie principale, vous devez entrer l'adresse du décodeur (locomotive) que vous programmez. Votre système DCC peut vous demander de cocher la case Adresse Longue si cette adresse est une adresse longue. La plupart des systèmes DCC ne vous permettent pas de lire sur la voie principal, le cas échéant,le bouton Lire le CV sera désactivé.

Maintenant que toutest configuré, vous pouvez lire la valeur actuelle dans la mémoire du décodeur à partir du CV en cliquant sur sur le bouton Lire le CV ou écrivez votre nouvelle valeur dans le CV en cliquant sur le bouton Écrire un CV .
