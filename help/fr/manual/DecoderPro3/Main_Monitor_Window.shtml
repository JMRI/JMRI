DecoderPro® Fenêtre Principale

Fenêtre Moniteur de Communications

Retour à l'Index

DecoderPro fournit une fenêtre de contrôle des communications vous permettant de voir ce qui se passe sur la liaison de communication entre le programme et le matériel du réseau.

Lorsque la fenêtre du moniteur s'ouvre, elle affiche automatiquement tout le trafic sur le lien de communication.

La barre de titre affiche le nom du lien de communication de votre système DCC.


Les messages envoyés et reçus via le lien de communication sont affichés dans le volet principal de la fenêtre.

Contrôles en bas de la fenêtre.

Bouton Effacer l'écran : efface la zone de journal de l'écran où les messages sont affichés.

Bouton Arrêt sur écran : arrête l'acquisition de nouvelles informations afin que les informations capturées ne défilent plus.

Afficher les données brutes : le moniteur affiche normalement l'interprétation que donne le programme de ce qu'il voit. Si vous voulez voir les octets bruts, cochez cette case.

Afficher les horodatages : normalement, le programme n'affiche que les données du lien de communication. Si vous souhaitez que chaque message soit précédé de l'heure à laquelle il a été envoyé ou reçu, cochez cette case.

La fenêtre toujours en haut - Affiche toujours la fenêtre du moniteur.

Défilement automatique : fait défiler les données dans la fenêtre en continu.

Filtrer les Octets:

Vous pouvez filtrer les messages indésirables de la fenêtre en entrant la première valeur hexadécimale de chaque message dans le champ "Filtrer les octets". Cela peut être particulièrement utile lorsque le trafic de communication est important. Séparez plusieurs valeurs par des espaces. par exemple. "B0 D0 CC". Cet exemple supprimera tous les messages dont les données brutes commencent par "B0", "D0" ou "CC". Vous pouvez déterminer les valeurs hexadécimales disponibles en activant l'option "Afficher les données brutes".

Choisissez un Bouton Fichier Journal: enregistrez les données du moniteur dans un fichier pour les récupérer à des fins de diagnostique et les partager avec d'autres.

Bouton Démarrer l'enregistrement - Lance l'envoi de données de la Fenêtre du moniteur vers le fichier sélectionné.

Bouton Arrêter l'enregistrement - Arrête l'envoi de données de la Fenêtre du moniteur vers le fichier sélectionné.

Connexion à un fichier

Parfois, il est utile de capturer les événements qui se produisent. Si un problème survient, les informations capturées dans le fichier journal peuvent être envoyées au groupe pour analyse du problème. De petites quantités d'informations peuvent être copiées à partir du volet d'affichage de la fenêtre du moniteur. Si vous avez besoin de plus d'informations que celles fournies par la fenêtre, vous devez les écrire dans un fichier journal sur le disque de votre ordinateur.

Pour ce faire:

  1. Cliquez sur le bouton Choisir le fichier journal . Cela ouvre une boîte de dialogue Enregistrer sur votre ordinateur, localisez le dossier dans lequel vous souhaitez enregistrer le fichier journal, attribuez un nom au fichier journal (si vous utilisez le nom d'un fichier existant, il sera écrasé). ). Cliquez sur le bouton Enregistrer.
  2. Lorsque vous êtes prêt à capturer les données que vous souhaitez enregistrer, cliquez sur le bouton Démarrer la journalisation dans la fenêtre du moniteur.
  3. Une fois terminé, cliquez sur le bouton Arrêter la journalisation pour stocker les données et fermer le fichier.
Bouton

Ajout d'un message

Saisissez votre message dans la zone de texte Ajouter un message, puis cliquez sur le bouton Ajouter un message pour ajouter votre message au fichier journal.
