La fonction Groupes d'Inventaire vous permet de prendre l'ensemble de votre inventaire et de le classer. Á la maison, vous voudrez peut-être faire la distinction entre le diesel et la vapeur ou la compagnie. Dans un club, il serait pratique d'avoir des catégories par nom de membre en plus de toute autre catégorie à laquelle vous pourriez penser. Une entrée de l'inventaire peut être associée à plusieurs groupes. Toujours dans un environnement de club, une entrée dans l'inventairee peut être associée aux groupes Vapeur, Union Pacific et John Smith en même temps. Il y a toujours un groupe présent qui s'appelle Global. Toutes les entrées de la liste sont dans le groupe global.

Masquer/Afficher les groupes d'Inventaires

Le contrôle Masquer/Afficher les groupes d'inventaires est utilisé pour activer/désactiver le panneau d'affichage des groupes d'inventaire dans la zone surlignée en rose. Cela doit d'abord être activé en cliquant sur l'onglet Paramètres, puis sur Masquer/Afficher les groupes d'inventaires. Si le panneau Groupes d'inventaire ne s'affiche pas lorsque vous cliquez sur le paramètre, essayez de saisir le cadre situé à gauche de la zone et de le faire glisser pour l'ouvrir (assurez-vous que le mode Masquer/Afficher est dans le bon état.)

la largeur du panneau peut être dimensionnée en faisant glisser le cadre comme indiqué par les flèches rouges. Les petites pointes de flèche vous permettent de fermer ou d'ouvrir le panneau

Les groupes de l'inventaire seront répertoriés dans le panneau de l'inventaire au fur et à mesure de leur création.

Créer un Groupe d'Inventaire

Utilisez Créer un Groupe d'Inventaire pour définir les autres groupes souhaités. Dans le menu Configuration, Créer un groupe d'Inventaire ouvre la boîte de dialogue Nouveau groupe d'Inventaire. Cette même boîte de dialogue Nouveau groupe d'Inventaires peut être ouverte à l'aide du bouton + ci-dessus.

Crée un Groupe d'Inventaire

Le bouton OK ajoutera le groupe d'Inventaire à la liste du panneau et à la liste déroulante de la barre d'outils de la fenêtre principale.

Vous pouvez maintenant ajouter des locomotives aux groupes en les faisant glisser depuis la liste de locomotives ou en utilisant le Tableau Associé, décrit plus tard.

Association de Tableaux de groupes d'Inventaire

Une fois les groupes nommés, vous "associez" les entrées d'Inventaire à leur groupe. Pour ce faire, utilisez la sélection Tableau Association. En haut du tableau, sélectionnez le groupe auquel vous souhaitez associer les entrées de l'inventaire, puis les entrées de l'inventaire que vous souhaitez associer à ce groupe.

Sélectionner un groupe d'Inventaire: le groupe d'Inventaire auquel vous voulez ajouter des entrées

Le tableau présente toutes les locomotives de votre inventaire.

Dans la colonne de droite, Inclure ... sélectionnez les entrées que vous souhaitez ajouter au groupe d'inventaires, cliquez individuellement pour sélectionner un seul élément, maintenez la touche Maj enfoncée pour sélectionner une plage d'éléments, maintenez la touche CNTRL enfoncée. pour sélectionner des éléments au hasard. Fermez la table et les entrées seront ajoutées au groupe.

Une fois les associations établies, vous pouvez sélectionner le Groupe d'Inventaire Actif. Une fois cette opération effectuée, les actions Imprimer et Imprimer le résumé ne montreront que les entrées de l'inventaire associées à ce groupe. Par exemple, vous pouvez sélectionner le groupe Santa Fe. Seules les entrées de l'inventaire de ce groupe seront imprimées. L'impression ne montre pas actuellement le groupe actif dans l'en-tête. Le groupe actif influera également sur les listes déroulantes contenant une liste d'entrées, telles que Régulateur, édition entrée, copie, etc. De plus, lors de l'ajout d'une nouvelle entrée de l'inventaire, elle sera automatiquement associée au groupe actif. Lorsque vous avez fini de travailler avec un groupe, il est recommandé de renvoyer le groupe actif à Global.


Restauration de l'Inventaire pour inclure toutes les Machines

Pour que votre inventaire affiche ttoutes les Machines, Sélectionnez le Groupe d'Inventaire dans Toutes les Entrées ou Global. Ensuite, vous pouvez être certain de travailler avec toutes les Machine de votre Inventaire.

Dissociation de l'Entrée de l'Inventaire dans un Groupe

Pour dissocier une entrée inventaire d'un Groupe d'Inventaire, ouvrez le Tableau des Groupes d'Inventaire, décochez la sélection de l'inventaire et enregistrez le nouveau Groupe en fermant le Tableau du Groupe d'Inventaire.
