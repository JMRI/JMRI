SVP Consultez Aide à la Configuration de JMRI.

JMRI® nécessite que Java soit installé, que votre ordinateur soit basé sur Mac, Windows,ou Linux .

La version 2.12 peut s'exécuter sur n'importe quel système informatique Java 1.5 ou version ultérieure.
La version 2.14.1 nécessite Java 1.5 (ou 1.6 si vous souhaitez faire glisser & déposer)
La version 3.10.1 nécessite Java 1.7 ou une version ultérieure.
La version 4.2 nécessite Java 1.8 .

Vous trouverez les instructions complètes pour l'installation sur différentes plates-formes ici:

  • Windows
  • Mac OS X
  • Linux
    • RasPi
