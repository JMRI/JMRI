Panneau Fonction Régulateur

Le Le panneau de fonctions du Régulateur vous permet de sélectionner les fonctions du décodeur de F0 à F15 sur la première page et de F16 à F28 sur la seconde. De nombreux nouveaux décodeurs de son utilisent jusqu'à 28 fonctions. Utilisez pour basculer entre les deux pages

Si vous avez entré les libellés Fonction , le texte appaitra sur les boutons du Régulateur.

Une autre façon de modifier les libellés des boutons de fonction consiste à cliquer avec le bouton droit de la souris sur le bouton que vous souhaitez modifier. Le mot Propriétés .. s'affiche et, lorsque vous cliquez sur ce bouton, la boîte de dialogue Modifier la fonction sera affichée.

Le Bouton Modifier Fonction vous permet de modifier le texte affiché sur le bouton et la fonction attribuée.

Numéro de fonction: zone de texte permettant de définir la fonction attribuée au bouton.

Avertissement !!! Veillez à ne pas attribuer le même numéro de fonction à plusieurs boutons. Dans le cas contraire, des résultats indésirables peuvent se produire

Texte: le texte à afficher sur le bouton.

Taille de la police: sélectionnez la taille de la police en points.

Verrouillable: verrouille la fonction lorsque vous cliquez sur le bouton. par exemple, fait du bouton un bouton On/Off.

Visible: rend le bouton visible. Lorsque cette case n'est pas cochée, le bouton n'apparaîtra pas dans le panneau. Pour restaurer, utilisez le menu Afficher , puis sélectionnez l'option Afficher toutes les touches de fonction .

Icône de Fonction Désactivée :

Faites glisser l'icône de votre choix dans la boîte. Pour supprimer l'icône, cliquez avec le bouton droit de la souris et cliquez sur Supprimer.

Icône de Fonction Activée:

Faites glisser l'icône de votre choix dans la boîte. Pour supprimer l'icône, cliquez avec le bouton droit de la souris et cliquez sur Supprimer.
