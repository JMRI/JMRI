    • Programme
      Ouvre le programmeur configuré dans vos préférences pour l'entrée d'Inventaire sélectionnée, Programmeur Complet , la valeur par défaut est Programmeur Basique

      Ouvre la fenêtre Etiquettes et Média avec deux onglets pour définir ces paramètres pour l'entrée d'Inventaire sélectionnée dans votre inventaire. Sélectionnez la Fonction Étiquettes pour pouvoir modifier les étiquettes de vos fonctions de régulation. Sélectionnez le Support de contenu pour modifier l'icône de votre sélection d'Inventaire.
