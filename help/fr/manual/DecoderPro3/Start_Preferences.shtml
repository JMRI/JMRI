L'aide actuelle sur les préférences JMRI est ici.

Dans le menu Edition de DecoderPro, sélectionnez Préférences. Sous Mac OS X, choisissez Préférences ... dans le menu de l'application DecoderPro. Cette fenêtre peut s'ouvrir automatiquement la première fois que vous exécutez le programme.

En tant que nouvel utilisateur, les informations les plus importantes sont l'écran Connexions, dans lequel vous décrivez comment l'ordinateur communiquera avec la Centrale de Commande. La plupart du temps , vous pouvez laisser les réglages initiaux ou "par défaut" jusqu'à ce que vous ayez besoin de modifier les choses à votre guise.

Plus important encore, vous devez cliquer sur pour finaliser vos paramètres, puis redémarrez le programme pour qu'ils prennent effet.

La fenêtre Préférences permet d'accéder aux informations de configuration de base pour connecter votre ordinateur à votre système DCC. Vous utiliserez ce système pour programmer et exploiter des trains équipés de décodeurs DCC. Un concept important à garder à l'esprit est que JMRI n'est rien de plus qu'un Régulateur très intelligent. Si vous pouvez faire quelque chose avec votre Régulateur, vous pouvez probablement le faire plus rapidement et plus facilement avec JMRI. Cependant, si votre système ne peut pas faire ce que vous voulez, soit parce que votre système n'est pas connecté correctement (c'est cassé) ou que le fabricant du système n'a pas incorporé cette fonctionnalité, alors JMRI ne pourra pas le faire non plus. Dans ce dernier cas, JMRI vous communiquera probablement un message d'erreur ou aura la commande "grisée" pour vous aider à déterminer pourquoi elle ne peut pas répondre à votre demande.

La fenêtre est divisée en deux volets. Sur le côté gauche se trouve une liste de groupes de préférences pouvant être définis. En cliquant sur l'un des éléments, vous ouvrez les options pouvant être définies dans le volet de droite. Aucune sélection n'a été faite

Nous allons maintenant configurer les préférences pour l'environnement JMRI avec votre centrale de commande.

J'utilise une centrale de commande Digitrax Super Chief (radio) reliée à l'ordinateur par LocoNet, PR3 (mode MS100) et un port USB. Je garde la liste principale sur le lecteur système.;

Volet Connexions (les onglets sont configurés en fonction du système que vous avez sélectionné)

Sélectionnez maintenant Connexions et les options de connexion s'affichent dans le volet de droite. Après avoir été sauvegardé et redémarré. Ceci est la configuration du système Digitrax Chief avec interface PR3.

Constructeur du système: Liste déroulante avec tous les fabricants DCC pris en charge, sélectionnez le fabricant de votre système.
Connexion au système: Liste déroulante avec toutes les interfaces système DCC prises en charge par le fabricant que vous avez sélectionné, sélectionnez l'interface que vous utilisez.
Paramètres: Port série: Liste déroulante permettant de sélectionner le port série auquel votre interface DCC est connectée. Si vous ne le savez pas, consultez le manuel de configuration du matériel de votre système. Une deuxième liste déroulante peut inclure une sélection s'il existe plus d'une possibilité. Volet de démarrage

Les préférences

Démarrer permettent d'ouvrir des fenêtres et des fichiers, d'exécuter des scripts et de placer des boutons supplémentaires dans la fenêtre principale au démarrage de l'application.

Cliquez sur le bouton Ajouter, puis sélectionnez le type d'action de démarrage à ajouter. La nouvelle action sera ajoutée en dernier.

L'ordre d'action au démarrage est important. Dans la plupart des cas, les scripts doivent être les derniers éléments de l'ordre de démarrage. L'ordre de démarrage peut être modifié en sélectionnant un élément de démarrage et en le déplaçant vers le haut ou vers le bas dans la liste.
