Le volet Support de Média s'ouvrira à partir du bouton DecoderPro de la fenêtre principale ou du menu Action, puis sélectionnez l'onglet Support de Média.

La sous-fenêtre Multimédia vous permet d'ajouter une image principale et une icône à la liste des Machines. L'image principale peut être utilisée pour l'image d'arrière-plan du Régulateur si elle est sélectionnée dans les préférences du Régulateur. L'image de l'icône est utilisée dans la liste des Régulateur et dans diverses zones de liste déroulante.

L'icône est l'image qui apparaîtra dans la partie Informations sur le décodeur de l'écran initial de DecoderPro, une fois que vous avez sélectionné une locomotive dans la liste. L'icône s'affiche é galement dans la colonne d'icônes de la liste des locomotives.

Image à utiliser pour cette locomotive: le grand cadre correspond au support de l'image. L'image est placée dans le porte-image en suivant les instructions ci-dessous.

  1. Vous devez d'abord trouver l'image de la locomotive et la stocker sur votre lecteur de disque.
    1. Sources
      1. Images téléchargées d'un site Web sur votre ordinateur
      2. Photos prises avec un appareil photo numérique ou numérisées à partir d'un film
      3. Autres sources, images de CD, de DVD provenant de vidéos, etc. enregistrées sous forme d'image sur votre ordinateur
    2. Format
      1. Le format GIF est préférable avec un fond transparent
      2. PNG avec fond transparent
      3. JPG
  2. Après avoir enregistré l'image sur votre ordinateur, ouvrez le volet Média de support, puis ouvrez l'explorateur Windows ou le gestionnaire de fichiers de votre système, puis faites glisser le fichier image dans le grand cadre noire. Vous pouvez également enregistrer l'image sur votre bureau, puis faire un glisser-déposer sur le média d'alignement à partir de là. N'essayez PAS de glisser-déposer depuis un site Web.
  3. Les images de la liste sont stockées dans le dossier <Préférences JMRI>/resources. Lorsqu'une image est mise à jour ou supprimée, elle n'est pas supprimée de ce dossier.

Pour supprimer une image de la liste, cliquez avec le bouton droit de la souris et cliquez sur Supprimer dans le contexte qui s'affiche. Toutefois, l'image ne sera pas supprimée du dossier Ressources.

Image à utiliser comme icône pour cette locomotive, tête à droite: de la même manière, il existe un petit cadre pouvant être rempli avec une image à utiliser comme icône d'identification de la liste. Les instructions sont également similaires:

  1. Ensuite, vous devez trouver l'image de la locomotive à utiliser comme icône et la stocker sur votre lecteur de disque.
    1. Sources
      1. Images téléchargées d'un site Web sur votre ordinateur
      2. Photos prises avec un appareil photo numérique ou numérisées à partir d'un film
      3. Autres sources, images de CD, de DVD provenant de vidéos, etc. enregistrées sous forme d'image sur votre ordinateur
    2. Format
      1. Le format GIF est préféré
      2. PNG avec fond transparent
      3. N'utilisez pas de fichier JPG.
  2. Après avoir stocké l'image de l'icône sur votre ordinateur, ouvrez le volet Multimédia, l'explorateur Windows ou le gestionnaire de fichiers de votre système, puis faites glisser le fichier image dans le petit cadre. Vous pouvez également enregistrer l'image sur votre bureau, puis faire un glisser- déposer sur le média d'alignement à partir de là. N'essayez PAS de glisser-déposer depuis un site Web.

Pour supprimer une image de la liste, cliquez avec le bouton droit de la souris et cliquez sur Supprimer dans le contexte qui s'affiche. Toutefois, l'image ne sera pas supprimée du dossier Ressources.

Référence Web: vous pouvez copier l'URL du site Web sur lequel vous avez localisé les images pour référence

.

Après avoir ajouté des images aux zones d'image pour l'image principale et l'icône, cliquez sur ce bouton pour enregistrer les images dans votre Liste. Maintenant, l'image apparaîtra dans votre liste d'inventaire dans DecoderPro.
