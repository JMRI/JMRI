Le programmeur de base utilise uniquement les onglets requis pour configurer le décodeur pour le fonctionnement DCC et la création d'une entrée d'Inventaire et utilise uniquement les deux premiers onglets. La première action dans la programmation d'une locomotive avec DecoderPro® est de remplir l'écran de saisie de l'Inventaire.

Un Inventaire est une base de données de toutes les locomotives programmées par votre installation de DecoderPro. Toutes ces informations figureront dans la liste de la page principale de DecoderPro. Il comprend les informations affichées dans l'écran ci-dessous:


a plupart de ces champs sont explicites. Cependant, notez que le premier champ (ID) devient le nom du fichier dans l'Inventaire de la locomotive que vous programmez. Configurez un schéma pour identifier votre locomotive dans son programme de décodeur. C'est-à-dire le fabricant, le type de locomotive et l'adresse (Kato_SD40-2_5645). Les espaces dans le champ ID seront remplacés par des traits de soulignement (_) lors de l'écriture du fichier. Dans un club, l'identifiant commence souvent par le nom du propriétaire ou son numéro de membre. De cette façon, la liste (classée par ordre alphabétique) conservera la liste de toutes les locomotives du propriétaire. DecoderPro vous permet également de classer les entrées de l'Inventaire dans des Groupes, afin que vous puissiez avoir un groupe séparé pour chaque membre..

L'adresse DCC sera automatiquement renseignée lors de la programmation de l'adresse. La famille et le modèle de décodeur sont entrés automatiquement à partir de la page de sélection que nous venons d'utiliser pour arriver ici..

Le champ Commentaire Decodeur est l'endroit idéal pour ajouter la date d'achat de la locomotive, son prix ou toute autre information qu'il serait important d'enregistrer.

Le bouton stocke les informations actuelles du décodeur sur le disque dur et dans le dossier de votre ordinateur où se trouve votre fichier d'inventaire. Le stockage par défaut est le même répertoire où le programme est installé.

e bouton peut être utilisé pour ramener le fichier d'Inventaire à la condition d'un nouveau fichier pour le type et la version du décodeur énumérés dans l'entrée de l'inventaire. Il ne modifie pas les valeurs du fichier d'inventaire sur votre disque dur, sauf si vous enregistrez spécifiquement après l'utilisation de ce bouton de réinitialisation. De plus, il n'est pas écrit dans le décodeur tant que vous n'avez pas sélectionné une opération d'écriture. Il a été inclus dans DecoderPro afin que, si vous vous trompez lors de la procédure écran à écran, vous pouvez revenir au jeu de paramètres par défaut du fabricant du décodeur et recommencer..


La plupart des décodeurs activent le Menu Réinitialiser (à côté du menu Fichier en haut de la page). Cela réinitialisera le décodeur aux paramètres par défaut du fabricant pour tous les CV, ou seulement à certains d'entre eux si le fabricant dispose de plusieurs routines de réinitialisation. Pour ce faire, il écrit directement dans un CV du décodeur, si le décodeur a cette fonctionnalité, CV8 dans le cas de Digitrax. Tous les décodeurs peuvent ne pas prendre en charge la réinitialisation et leur fonctionnement dépend du fabricant et du décodeur.

The Le mode de programmation peut être modifié en cliquant sur la liste déroulante

Il existe des options pour plusieurs modes de programmation . JMRI sélectionne maintenant le meilleur mode pour vous. Cela est déterminé à partir du type de centrale de commande que vous utilisez et du modèle de décodeur que vous essayez de programmer. Habituellement ce sera OK. Si vous rencontrez des problèmes de programmation, vous pouvez essayer un autre mode.

Pour la plupart des nouveaux décodeurs, restez en mode Paginé ou selon le Mode Direct pris en charge par votre centrale de commande. La documentation de votre décodeur devrait vous indiquer si la programmation dans un autre mode est nécessaire ... mais si vous rencontrez des problèmes, expérimentez. Si un mode n'est pas pris en charge pour le décodeur et le système sélectionné, cette option est grisée. Pour ce décodeur, je vais utiliser le mode Octet Direct.

Lorsque vous effectuez votre sélection, le mode de programmation doit maintenant apparaître à gauche dans la liste déroulante.

Vous trouverez ci-dessous une configuration de base de l'inventaire de programmation pour la locomotive Athearn FP-45 à l'échelle n avec le décodeur OEM (Tsunami 750). La note dans le statut en bas de l'onglet indique que le fichier a été enregistré.

Maintenant que toutes les informations de la liste sont en ordre, nous pouvons continuer à programmer le décodeur.

Maintenant il est temps de cliquer sur le bouton Enregistrer dans la liste pour enregistrer ces données. Notez qu'il y aura un bref message dans la barre d'état en bas de l'écran indiquant que la liste a été sauvegardée.

Passons au programmeur basique.
