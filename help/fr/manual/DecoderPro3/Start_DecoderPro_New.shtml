Premiers pas avec DecoderPro®

Comment démarrer le programme DecoderPro pour un nouvel utilisateur?

Retour

Pour la première fois ou une toute nouvelle installation. Une fois que vous avez téléchargé et installé le logiciel JMRI, ouvrez simplement le programme de la manière habituelle pour votre système d'exploitation. Une icône pour DecoderPro doit être installée sur votre bureau.

DecoderPro peut être ouvert en double-cliquant sur l'icône du bureau. L'écran initial de DecoderPro, présenté ci-dessous, s'affichera à la fin du démarrage pour une nouvelle installation sans aucun paramètre enregistré sur votre ordinateur.
