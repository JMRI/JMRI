La Sélection des Éléments à Imprimer est ouverte à partir de la barre de menus de la fenêtre principale de DecoderPro, Fichier-> Imprimer/Aperçu de l'Inventaire-> Entrée.

Cette boîte de dialogue vous permet de sélectionner les fonctionnalités d'un décodeur individuel de l'Inventaire que vous souhaitez prévisualiser ou imprimer.

Bouton Annuler - Annule les sélections

Bouton OK - affiche la boîte de dialogue d'impression de votre imprimante ou la fenêtre d'aperçu de l'imprimante.

Exemple: case à cocher Liste des fonctions cochée. Vous pouvez sélectionner n'importe quelle combinaison ou tous les éléments de la liste.

Exemple d'aperçu avant impression avec les options de base et de vitesse sélectionnées.
