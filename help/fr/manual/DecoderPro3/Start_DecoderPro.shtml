Une fois le logiciel JMRI téléchargé et installé, il vous suffit d'ouvrir le programme de la manière habituelle pour votre système d'exploitation. Une icône pour DecoderPro doit être installée sur votre bureau.

DecoderPro peut être ouvert en double-cliquant sur l'icône du bureau. L'écran initial de DecoderPro, comme indiqué ci-dessous, s'affichera à la fin du démarrage, sans aucune entrée dans la liste.

DecoderPro s'ouvre et affiche les entrées de votre liste si vous avez déjà établi une liste de locomotives.

Cet écran peut prendre un certain temps, en particulier sur les ordinateurs les plus anciens et les plus lents. N'oubliez pas que vous exécutez essentiellement le programme sur un émulateur Java via le logiciel de votre système d'exploitation natif. Les nouveaux ordinateurs peuvent le faire assez rapidement, mais les plus anciens auront l'impression de prendre une éternité. Soyez patient - ça finira par arriver! Même sur des ordinateurs plus anciens, le temps de réponse est assez rapide une fois le programme terminé.

Si vous exécutez le programme pour la première fois après l'installation du logiciel, l'écran Préférences apparaîtra automatiquement pour vous permettre de configurer le système pour votre configuration particulière. De plus, si vous modifiez la configuration de votre système, ou si votre ordinateur portable est éloigné du chemin de fer, mais que vous souhaitez jouer avec JMRI, vous pouvez essayer un mode simulateur. Cliquez ensuite sur Suivant, et nous examinerons les préférences. .
