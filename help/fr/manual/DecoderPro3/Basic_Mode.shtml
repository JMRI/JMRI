Quelques brefs commentaires sur les Modes de Programmation.

Mode Adresse est une méthode de programmation obsolète qui est incluse ici dans un souci de totale conformité avec la norme NMRA DCC.

Mode Registre est une forme développée du mode adresse, et est toujours utilisée par certains décodeurs plus anciens et/ou d'entrée de gamme, en particulier par MRC et Wangrow. Il est intrinsèquement limité dans sa capacité à accéder à tous les CV d'un décodeur..

Mode Paginé est une extension du Mode Registre qui donne un accès complet à tous les CV du décodeurs.

Mode Octet Direct est une autre méthode, pas encore supportée par tous les décodeurs, qui permet un accès complet. Il existe deux manières de mettre en œuvre le mode direct. Le (s) bouton (s) radio de la/des méthode (s) du mode Octet Direct (s) pris en charge par votre centrale seront activés .

Si vous rencontrez des difficultés pour programmer un décodeur en Mode Paginé, essayez le Mode Octet Direct, puis le Mode Registre, et enfin le Mode Adresse. Le décodeur d'accessoires EasyDCC AD4 ne peut être programmé qu'en Mode Octet Direct.
