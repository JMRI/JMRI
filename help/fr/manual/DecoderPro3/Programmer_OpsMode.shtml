Sélectionnez la locomotive à partir de la liste, sélectionnez Programmation principale et appuyez sur Programme.

Programmateur en mode Opérations (Piste principale) - ou Programmation sur le POM principal. Ouvre la fenêtre Mode d'exploitation (piste principale). Elle vous permet de programmer un décodeur spécifique sur le réseau, même en cours de fonctionnement. Cependant, étant donné qu'il s'adresse à une adresse de décodeur spécifique, certains décodeurs ne vous permettront pas de changer l'adresse du décodeur en utilisant cette méthode!

Certaines centrale de commande ne "diffuseront" que des commandes de programmation, ce qui signifie que chaque locomotive du réseau reçoit la même programmation! Il en va de même si vous essayez de programmer une locomotive avec une adresse zéro (ou si vous oubliez de saisir une adresse lorsque vous démarrez). En cas de doute, consultez votre documentation.

Lecture à partir de votre décodeur dans le menu principal:

Á quelques exceptions près, il n'est pas possible de lire un CV sur le réseau. Si la lecture échoue, une erreur 306 est généralement signalée (délai d'attente pour dialoguer avec la centrale de commande.)

Écrire dans votre décodeur à partir du menu principal:

Les programmes sur les écritures principales sont souvent appelés «écritures aveugles» car le décodeur ne répond pas que l'écriture a réussi. Le POM est très utile pour régler les performances de la locomotive pendant son fonctionnement et l'accusé de réception est le changement de performances.
Le décodeur est d'abord adressé par la station de commande, puis le CV est adressé, puis les données à écrire dans le CV de ce décodeur est envoyé. Une exception est si l'adresse zéro est utilisée (ou si vous oubliez d'entrer une adresse lorsque vous démarrez). Ensuite, les ALL décodeurs du disque principal écrivent ces données dans le CV approprié, ce qui constitue généralement une "conséquence inattendue".

Certaines stations de commande ne "diffuseront" que des commandes de programmation, ce qui signifie que chaque locomotive du réseau reçoit la même programmation! En cas de doute, consultez votre documentation.

Dans cette fenêtre, vous sélectionnez la locomotive dans laquelle vous souhaitez modifier les valeurs de CV et le programmeur que vous souhaitez utiliser. Une fois sélectionnés, cliquez sur le bouton Ouvrir un programmateur, comme vous le feriez avec le programmeur en mode service.
