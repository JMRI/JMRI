Avant d'utiliser le programmeur, vous devez avoir rempli la section précédente ( Mise en route), connectez votre ordinateur à votre centrale de commande/booster avec le périphérique d'interface requis. pour votre système DCC, vous êtes maintenant prêt à programmer un décodeur installé dans votre locomotive.


Lors de l'ouverture initiale decoderPro, la page principale s'affiche: (Une fois les préférences définies), de l'inventaire des locomotives programmées par DecoderPro reste vide jusqu'à ce que vous ajoutiez une locomotive. Á partir de ce moment, lorsqu'on ouvrira une inventaire de toutes les locomotives dont les décodeurs que vous avez déjà programmés dans DecoderPro3 ou DecoderPro classique seront listés.

DecoderPro fournit un accès aux outils JMRI nécessaires à la programmation des décodeurs DCC. Si vous êtes un utilisateur expérimenté et que vous utilisez PanelPro pour exécuter votre réseau, vous pouvez démarrer PanelPro puis, à partir du menu Fichier, lancer DecoderPro.

Si vous avez déjà des entrées dans de l'inventaire et que vous souhaitez commencer à programmer dans Programmeur complet , double-cliquez sur l'entrée de l'inventaire.

DecoderPro peut lui-même programmer des décodeurs DCC et également utiliser les autres outils JMRI pour faire fonctionner vos trains manuellement à partir de l'interface de l'ordinateur. Celles-ci incluent le Régulateur et le contrôle de l'Alimentation de votre réseau si votre système DCC le permet.

La fenêtre DecoderPro est divisée en cinq sections principales de haut en bas.


1 . La barre de menus.
2. La barre d'outils
3. Tableau des inventaires : inventaire de tous les décodeurs de votre inventaire faisant partie du groupe d'inventaires actif. Pour configurer un groupe de répertoires en plus du groupe par défaut de Toutes les entrées, voir Paramètres des groupes de inventaire. .

4. Zone d'informations du décodeur. Affichez des informations sur toutes les locomotives sélectionnées dans de l'inventaire d'alignement.
5. Une barre d'état qui affiche des informations sur le fonctionnement decoderPro, y compris les erreurs.

Ajouter une nouvelle locomotive

Une icône intitulée Nouvelle loco est présente dans la barre d'outils. Cliquez sur l'icône pour ouvrir le volet Créer une nouvelle locomotive. Placez une locomotive (avec le décodeur installé) sur la voie de programmation (mode service), cliquez sur le bouton lire le type à partir du décodeur (en bas de la fenêtre).


Comme dans l'écran ci-dessous, plusieurs décodeurs possibles seront probablement mis en surbrillance. Cela est dû au fait que les décodeurs sont électriquement identiques mais sont emballés différemment pour s'adapter aux locomotives dans lesquelles ils sont utilisés. Si vous savez quel décodeur est utilisé, sélectionnez-le. Sinon, dans la plupart des cas, l'un des décodeurs en surbrillance fonctionnera et vous devrez sélectionner celui comportant le moins de lettres à la fin, car les lettres sont souvent utilisées pour désigner les variations spécifiques à la locomotive. Notez que tous les décodeurs Tsunami utilisent le même matériel dans des configurations différentes.

Je sais que ce décodeur est un décodeur OEM Tsunami Diesel Genesis installé en usine par FP-45 d'Athearn N Scale. Je vais donc sélectionner ce décodeur dans de l'inventaire.


Notez que nous avons sélectionné un décodeur, qui est mis en surbrillance. Le bouton Ouvrir le Programmeur Complet est également activé pour nous permettre de saisir le programmeur.
