JMRI: Applications

JMRI offre plusieurs applications qui sont optimisées pour des objectifs spécifiques. Vous pouvez bien sûr également écrire les votres

DecoderPro

Un meilleur outil pour programmer les décodeurs DecoderPro simplifie la tâche de configuration compliquée des décodeurs DCC en fournissant des écrans sur lesquels vous pouvez sélectionner les différentes options et valeurs que vous voulez.

PanelPro

L'exploitation CRT dérivée des panneaux contrôle CTC qui reflète en temps réel l'état de votre réseau et vous permettent de le contrôler - presque comme un répartiteur prototype!

Outils LocoNet

Un ensemble d'outils LocoNet spécifiques Digitrax :
  • moniteur LocoNet
  • moniteur de Slot
  • source de paquets
  • programmateur PM4
  • programmateur BDL16
  • programmateur SE8c
  • programmateur DS64
Ils sont également disponibles dans DecoderPro et PanelPro lorsque ces derniers sont configurés pour une connexion LocoNet, mais LocoTools leur fournit une façon plus facile de les utilisés.
