Une Visite Rapide de DecoderPro

Démarrage

Lorsque vous ouvrez le programmateur, vous voyez cette fenêtre. Cliquez sur le bouton "Program locomotive " pour commencer. (Cela suppose que vous êtes déjà connecté à votre ordinateur et qu'une locomotive est sur votre voie de programmation)

Décodeur et sélection de Programmateur

Vous obtiendrez une nouvelle fenêtre pour configurer le programmateur avec l'information d'un type particulier de décodeur, ou d'une locomotive existante. Cela va devenir votre liste de locomotive quand vous programmerez vos décodeurs.

Pour commencer à travailler avec un décodeur nouvellement installé, cliquez sur le bouton adjacent "Ident"pour voir le programmateur lire le décodeur et tenter de l'identifier. (Vous pouvez aussi indiquer explicitement au programmateur le type de décodeur.

Pendant que le programmateur dialogue avec le décodeur, la situation sera affichée dans le bas de la fenêtre; "En Attente" ou "OK" indique que tout va bien. S'il réussit, il sélectionnera les modèle de décodeur dans la boîte de sélection. Habituellement, il ne sera en mesure de limiter la sélection qu'à un choix restreint. Vérifiez que le bon modèle est sélectionné dans la boîte "décodeur installé" ; mettez à jour la sélection si vous le souhaitez.

Certaines personnes préfèrent un programmateur très simple, et d'autres veulent avoir accès à toutes les fonctionnalités. Pour gérer cela, DecoderPro fournit plusieurs interfaces différentes de programmation :

  • Basic (Basique) - Une simple, avec seulement les informations les plus populaires
  • Comprehensive (Complet) - L'affichage le plus complet, avec plusieurs volets d'informations détaillées.
  • Advanced (Avancé) - Ajoute des volets avancés au "Programmer Conprehensive"
  • Registers (Registres) - Configuration complète pour un des registres d'un décodeur seulement
  • Sample Club (Exemple Club) - Un aperçu de la façon dont un club peut définir ses propres formats programmateurs qui montre leur ensemble de variables nécessaires
  • Tutoriel - Pas utile pour les décodeurs de programmation en fait, il contient des exemples de comment obtenir les différents effets spéciaux lorsque vous construisez votre propre programmateur.

La dernière fonctionnalité ajoutée est la DecoderPro3, qui rationalise simplement DecoderPro pour programmer les décodeurs et la circulation des trains avec les Régulateurs JMRI. Les principaux programmateurs disponibles sont:

  • Basic (Basique) - Une simple, avec seulement les informations les plus populaires
  • Comprehensive (Complet) - L'affichage le plus complet, avec plusieurs volets d'informations détaillées
  • Advanced (Avancé) - Ajoute des volets avancés au "Programmer Conmprehensive"
Vous pouvez même définir votre propre programmateur si vous préférez simplement voir les variables que vous utilisez. Pour cet exemple, nous sélectionnons le programmateur Comprehensive (Complet) de décodeur dans la boîte de sélection la plus basse, puis cliquez sur "Open Programmer" pour commencer à travailler avec le décodeur.

Détails Programmateur

Chacun des onglets sélectionne un écran, vous pouvez les utiliser pour configurer différents aspects du décodeur. Le volet "Basic" gère les paramètres de l'adresse dont vous aurez généralement besoin. En utilisant les boutons se trouvant sur le bas de chaque écran vous pourrez lire les valeurs ou écrire des valeurs dans le décodeur. Par exemple, pour faire une configuration de base, entrez des valeurs dans les champs et cliquez sur Écrire dans Onglet . Les champs variables sont codés par couleur pour représenter leurs états:

Edited
Vous avez changé cette valeur, ou elle contient des valeurs par défaut à partir du fichier décodeur. Ceci est montré en orange, puisque la valeur diffère de ce qu'il y a dans le décodeur.
From File
le champ contient des valeurs lues dans un fichier locomotive. Ceci est représenté en jaune, car nous ne sommes pas certains que le fichier soit en accord avec le contenu décodeur.
Read
la valeur indiquée a été lue par le décodeur. ceci est représenté en blanc pour indiquer que la valeur est digne de confiance.
Stored
affichage de la valeur qui a été écrite pour le décodeur. Cela est également montré en blanc.
Unknow
si quelque chose ne va pas avec le processus de lecture ou d'écriture,nous pourrions avoir un résultat tout à fait indigne de confiance. Il en résulte un état de la variable marqué comme UNKNOW et montré en rouge.


Exemples

Par exemple, après un "Lire", les valeurs de la table de vitesse sont indiquées en blanc sur la première image, Alors que dans la seconde image il montre que lorsque vous modifiez une valeur, elles sont marquées en jaune. Faire une écriture les remet à blanc; faire une lecture va réinitialiser la valeur à la valeur dans le décodeur, et les remettre à blanc.

L'option Digitrax éclairage FX, par exemple, exiger la configuration de deux ensembles de bits en un seul CV. Un autre exemple est la configuration de la cartographie d'un décodeur de Soundtraxx entre les fonctions et les sons. Il s'agit de bits et de valeurs réparties sur une douzaine de CV différents, mais Decoder Pro transforme cet emploi en configuration simple par contrôle de boîtes.

Chaque zone de sélection, case à cocher ou bouton radio est cartographié dans un bit(s) spécifique(s) dans les CV du décodeur. Le programmateur gère la lecture et l'écriture des CV lorsque les paramètres individuels sont modifiés. Au lieu de bits, vous travaillez avec le nom ou la description du manuel du décodeur.

Panneaux Programmateur Décodeur Spécifique

DecoderPro est conçu pour être facilement configurable pour ajouter de nouveaux types de décodeurs. Vous pouvez également ajouter des feuilles de programmateur, modifier leur disposition, et même refaire l'apparence globale. Pour ce faire, DecoderPro est piloté par des fichiers lisibles par l'homme en format XML. La documentation sur le contenu du fichier existe, et nous travaillons continuellement pour la mettre à jour.

Ces fichiers rendent commode la mise à jour de DecoderPro pour travailler avec un nouveau décodeur. Par exemple, les capacités uniques d'une DH142 Digitrax figurent sur une feuille "Digitrax" .

D'autres variables décodeur spécifiques et des feuilles peuvent être ajoutées via des fichiers de configuration. En choisissant un format industriel standard XML pour les informations du décodeur, nous espérons rendre possible la création d'une bibliothèque complète. Peut-être que nous puissions intéresser les fabricants, en les incitants à documenter leurs produits dans ce format par exemple.

Programmation Directe CV

Il est aussi possible de programmer les CV directement à l'aide du programmateur CV. Les boutons "Lire" et "Ecrire" vous permettent de travailler avec le contenu CV individuel. Les changements effectués ici mettront également à jour les valeurs associées trouvées sur les écrans d'autres programmateurs .

Listes Locomotive

DecoderPro tient une liste de vos locomotives. Vous pouvez enregistrer les configuration du décodeur, et ensuite travailler avec les valeurs stockées dans les CV au lieu d'avoir à les lire à partir du décodeur. Cela permet d'économiser le temps qu'il faut pour lire un décodeur complet, qui peut être assez long. La liste peut également stocker des informations de locomotive pour vous. La feuille "Entrée Liste" affiche cette information et vous permet de la modifier.

Cliquez sur "Enregistrer" pour stocker les informations d'un fichier sur le disque dur de votre ordinateur, vous serez invité à le faire si vous fermez le programmateur avec des informations non mémorisées.

L"ID " dans la partie supérieure est le nom que vous utiliserez plus tard pour sélectionner cette locomotive.

Sélection d'une Entrée Liste

Lors du démarrage de DecoderPro, vous pouvez sélectionner une locomotive spécifique pour travailler avec en utilisant l'option "Utiliser les paramètres de locomotive pour ..." du menu déroulant.

Le bouton "Ident" utilise l'adresse DCC pour sélectionner une locomotive spécifique dans le fichier. Lorsque vous cliquez sur "Ouvrir programmateur" avec une locomotive spécifique sélectionnée, le programmateur ouvre chaque feuille avec les valeurs actuelles déjà présentes.

Vous pouvez imprimer les informations de la locomotive actuelle depuis les fenêtres de programmation. Dans le menu "Fichier ...", sélectionnez "Imprimer ...", et remplir la boîte de dialogue qui est présentée.
