JMRI: Téléchargement et installation DecoderPro

DecoderPro est distribué dans le cadre des téléchargements libres de JMRI. Les étapes à suivre pour ce travail sont:

Assurez-vous que DecoderPro peut s'exécuter sur votre ordinateur
Le logiciel DecoderPro fonctionne sur Windows, Linux et Mac OS X. Il pourra probablement aussi travailler sur d'autres systèmes d'exploitation modernes qui ont Java d'installer; nous contacter pour vous aider si vous souhaitez l'essayer sur un autre système. Vous avez besoin d'avoir un port série disponible, soit sur l'ordinateur lui-même, ou via un adaptateur USB.
Assurez-vous que JMRI fonctionne avec votre système DCC
La page matériel dresse la liste des systèmes DCC compatibles avec DecoderPro, avec les limitations connue.
Téléchargez et installez JMRI
La page de téléchargement sur le site Web principal JMRI fournit des liens pour télécharger les différentes versions JMRI, ainsi que des liens pour les notes, etc Nous vous recommandons fortement d'abord de télécharger et d'installer la version de production la plus récente et assurez-vous que cela fonctionne, avant d'essayer quelques-unes des versions de développement plus récentes.

Notez qu'il existe différents téléchargements pour les différents types d'ordinateur (Macintosh, Windows, etc), et les procédures de l'installation du programme diffèrent. Vous trouverez tout cela sur la page de téléchargement.

Démarrer et configurer le programme
Les instructions d'installation obtenues par la page de téléchargement vous auront dit comment démarrer le programme et le configurer. Lorsque c'est fait, vous devriez avoir une installation fonctionnelle DecoderPro.
