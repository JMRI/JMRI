Détails des Fichiers Liste

Les informations de liste présentées dans la boite de sélection, et utilisées pour la sélection automatique d'une locomotive, sont stockées dans le fichier roster.xml et dans le répertoire JMRI Preferences. Vous pouvez trouver l'emplacement de votre répertoire Preferences en choisissant "Emplacements" dans le menu Aide JMRI. Cliquez sur la bouton "Ouvrir Emplacement Inventaire" pour ouvrir le répertoire dans le Moteur de recherche/Explorateur. Une information complémentaire est disponible sur: Page d'aide "Configuration Fichier" Vous pouvez ouvrir votre fichier Inventaire et regarder pour voir le format. Vous devez utiliser le menu Inventaire pour le manipuler le fichier Inventaire afin de s'assurer qu'il reste auto-cohérent.

Pour déplacer le fichier entier sur un autre ordinateur, choisir "Exporter Inventaire" depuis le menu, sélectionnez un emplacement pour sauver le fichier et ensuite déplacer le jeu de fichier vers un autre ordinateur avant que vous choisissiez " Importer Inventaire" depuis le menu pour l'importer là.

Pour faire ceci manuellement au niveau du fichier, dans le répertoire Preferences il doit y avoir un fichier appelé "roster.xml", et un répertoire appelé "roster". Vous avez besoin de déplacer le fichier "roster.xml", le répertoire "roster", avec tous les contenus. Plus de renseignement Configuration Fichiers JMRI
