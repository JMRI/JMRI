Guide Utilisateur DecoderPro - Détails des Fichiers Liste

Les informations de liste présentées dans la boite de sélection, et utilisées pour la sélection automatique d'une locomotive, sont stockées dans le fichier roster.xml et dans le répertoire "prefs". Vous pouvez ouvrir ce fichier et le regarder pour voir le format. Vous devez utiliser le menu Fichier pour le manipuler afin de s'assurer qu'il reste auto-cohérent.

Pour déplacer le fichier entier sur un autre ordinateur, d'abord trouver le répertoire des préférences en utilisant les informations dans la page "fichiers de configuration".

Dans le répertoire des préférences, il y aura un fichier nommé "roster.xml", et un répertoire appelé "roster". Vous avez besoin de déplacer le fichier "roster.xml", le répertoire "roster", et tous les contenus qui vont avec.
