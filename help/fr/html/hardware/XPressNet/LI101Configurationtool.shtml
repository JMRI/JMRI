Outil de Configuration XPressNet LI101F

>

L'outil de configuration LI101 est utilisé pour configurer le LI101F.

En partant du haut:

La première liste déroulante permet de sélectionner l''adresse que l'appareil utilisera sur le XPressNet. Cette adresse doit être différente de celle de tous les autres périphériques sur votre XpressNet.

La seconde liste déroulante affiche les boites fixant la vitesse du port LI101F. Cette vitesse devrait correspondre à ce que vous avez utilisé pour le paramètre "Port Speed" dans le volet Préférences

Boutons

A partir du côté gauche de l'écran, le bouton "Read from LI101" (Lire depuis LI101), va lire les paramètres actuels de LI101F.

Le bouton "Write to LI101"(écrire dans LI101), écrira les paramètres sur l'écran de LI101F. Si vous avez changé le réglage de la vitesse LI101, vous aurez également besoin de changer le réglage de la vitesse dans vos Préférences.

Le bouton "Reset to Factory Defaults" (Remettre aux paramètres par défaut) réinitialise l'écran aux réglages d'usine par défaut. Ceci n'envoie pas d'instruction de réinitialisation à LI101F à ce moment.

Le dernier bouton, étiqueté "Close" va fermer l'écran, sans envoyer d'information à LI101F.
