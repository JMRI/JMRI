Structure Matérielle: Outil Information Système XPressNet

L'outil d'Information Système XPressNet est utilisée pour déterminer quel matériel est connecté au système.

Les deux premières lignes indiquent le type de centrale et les version des logiciels qui sont connecté à votre ordinateur. Cette information est lue dans le cadre du processus de configuration automatique lorsque JMRI débute la première fois, mais peut être relu à l'aide du bouton "Get System Info" (Obtenir Infos Système).

Sous le champ Interface, la version du matériel et des logiciels d' Interface ordinateur sont affichées. Ces dernières sont obtenues par le "Get System Info" (Obtenir Infos Système).

L'image montre les résultats obtenus en appuyant sur le bouton "Get System Info "avec une version du logiciel 2 LI101F connectée à un Atlas Commander avec le logiciel 3.2.

Cliquer sur "Close" fermerera l'écran.
