Outil deCconfiguration XPressNet LZV100

L'outil de configuration LZV100 est utilisé pour configurer la LZV100 travers une interface informatique LI100 ou LI101 .

Commençant par le haut, nous avons des informations de configuration pour la partie Alimentation de la Centrale LZV100.

Le menu déroulant sélectionne la tension voie.

La seconde liste déroulante indique si le "E-Line" doit être Active ou Inactive. Quand elle est active l'Alimentation des Centrales utilise cette interface pour couper le réseau quand il y a un court-circuit, ou comme un arrêt d'urgence.

Boutons

A partir du côté gauche de l'écran dans la première rangée de boutons, le bouton "Write to LZV100"(Ecrire à LZV100), écrira les réglages au-dessus de cette ligne sur le LZV100.

Le bouton "Reset to Factory Defaults"(Reset aux paramètres par défaut)" réinitialise l'écran aux réglages d'usine par défaut. Ceci n'envoie pas d'instruction de réinitialisation à la LZV100.

Le dernier bouton, étiqueté "Close" va fermer l'écran, sans envoyer d'information à la LI101F.

La deuxième rangée de boutons contient des options pour la Centrale.
Ces options travaille aussi avec la LV100.

Le bouton "Reset Command Station"(Reset station de commande) envoie à l'adresse de locomotive 0 des instructions de fonction indiquée dans le manuel LZV100 pour une commande de réinitialisation de la Centrale. Ceci permet de réinitialiser la Centrale aux parmàtres d'usine. Voir le manuel LZV100 pour plus d'informations pour savoir exactement ce que cela fait.

Le bouton "Auto/Manual mode toggle" envoie à la locomotive à l'adresse 0 des instructions fonction indiqué dans le Manuel LZV100 pour basculer entre le mode de démarrage manuel et automatique. Dans le mode Manuel , la LZV100 fonctionnera sur toutes les fonctions des locomotives qui étaient en service la dernière fois que la LZV100 a été allumée. En mode Automatique, les locomotives reprendront également la vitesse et les réglages de direction précédents .
