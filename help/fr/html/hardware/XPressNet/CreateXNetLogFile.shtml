Comment Créer un Fichier de Connexion XPressNet

Démarrer Decoder Pro (ou la démo JMRI) Ouvrez le Moniteur XpressNet ( qui est dans le sous-menu du menu "Système" de XPressNet)

Vous verrez une fenêtre comme la suivante:
Ensuite cliquez sur le bouton Â«Choisir le fichier Rapport" , et choisissez un fichier Rapport à utiliser (vous pouvez utiliser le nom par défaut, la chose importante est de savoir où est le fichier )
Une fois que vous avez cliquez sur Enregistrer dans la boîte de dialogue, cliquez sur le bouton "Démarrer Connexion" sur le moniteur.
Maintenant, vous pouvez effectuer toutes les actions que vous souhaitez effectuer.
Enfin, retournez à la fenêtre Moniteur XPressNet , puis cliquez sur le bouton "Stop Connexion"
Maintenant vous pouvez quitter l'application ou continuer à travailler.
