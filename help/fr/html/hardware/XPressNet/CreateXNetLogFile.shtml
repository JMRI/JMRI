Structure Matérielle: Création d'un Fichier Journal

Démarrer Decoder Pro (ou la démo JMRI)

Ouvrez le Moniteur XpressNet ( qui est dans le sous-menu du menu "Système" de XPressNet)

Le volet suivant va s'ouvrir:
h4>Boutons

  • "Clear Screen" enlève tout l'historique du panneau
  • "Freeze Screen" Arrête le chargement de nouvelle donnée pour vous aider à étudier le contenu actuel de l'affichage
  • Cliquer sur le bouton "Choisir fichier journal", et choisir le nom d'un fichier de connexion à utiliser ( vous pouvez garder le nom par défaut; l'important est de savoir où le fichier est stocké )
  • Ensuite vous cliquez "Save" dans la boite de dialogue fichier, cliquez le bouton "Start logging" sur le panneau.
    Maintenant, vous pouvez effectuer toutes les actions que vous souhaitez effectuer dans JMRI. Un journal est conservé de tout le trafic sur la connexion XPressNet.
  • Cliquez le bouton "Stop Logging" sur le volet XPressNet Monitor, pour - bien - arrêter l'enregistrement.
