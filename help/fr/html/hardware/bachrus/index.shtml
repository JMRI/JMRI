Guide Matériel JMRI: Bachrus

Matériel Supporté

JMRI soutient le Bachrus MTS-DCC Speedometer pour train miniature, qui est un périphérique d'entrée unique, pour capturer les informations de vitesse à partir d'un module de lecteur sur un stand de course Bachrus "Saddle" . Pour plus d'informations sur les produits Bachrus voir le site web Bachrus.

Lorsque le Bachrus MTS-DCC est défini comme une préférence de connexion de réseau, l'utilitaire Speedo peut être utilisé pour afficher l'échelle de vitesse de course aux échelles de Z à O et au-delà.

Lorsque le Bachrus MTS-DCC est défini comme une connexion mise parallèlement à un système matériel DCC l'utilitaire Speedo permet la mesure automatique d'une décodeur DCC équipé en profil de vitesse de Loco.
