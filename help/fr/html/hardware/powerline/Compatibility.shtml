JMRI: Controleurs CPL (Powerline)pris en charge

Cette page vous permet d'identifier le type de contrôleur que vous avez, et vous donne des informations nécessaires pour configurer JMRI.

Voir aussi la page principale sur configuration JMRI pour les contrôleurs CPL .

Adaptateurs X10


Dernière Edition: 4 Février 2008
Dernière Edition: 10 Mars 2008
Fabricant Modèle Image Aide identification Status JMRI
X10.com CM11A
RCA HC60CRX 		Cet adaptateur peut étre identfié par la connection RJ-11 4 pins dans le bas.
Pour l'utiliser avec JMRI, sélectionner "Powerline Controller" dans les préférences JMRI, le bon port série, et "CM11" pour l'adaptateur. 		L'aide est disponible depuis la version JMRI 2.1.3
X10.com CP290 C'est un adaptateur de type console. Il peut aussi fournir des boutons pour des opérations sur des équipements X10 .
Pour l'utiliser avec JMRI, sélectionner "Powerline Controller" dans les préférences JMRI, le bon port série, et "CP290" pour l'adaptateur. 		Le support est disponible depuis la version JMRI 2.1.5
X10.com CM15A C'est l'adaptateur fourni avec ActiveHome Professionnal. Bien identifié par la connexion USB sur le fond et une antenne sur le côté. Non Supporté (encore). Nous espérons un prochain support, en attendant le support USB et que quelqu'un veuille bien le tester.
Dernière Edition: 17 Janvier 2008
X10.com CM17A Ce dispositif repose sur un autre appareil X10 qui est vrai contrôleur, généralement l'appareil TM751. Tandis qu'il est un équipement série, il utilise les lignes RTS/DTS au lieu des lignes de données . Ceci permet à l'équipement de travailler alors que d'autres matériels série travaillent à travers lui. Non supporté. Nous pouvons étre capable de le supporter éventuellement, mais nous travaillons sur le plus simple, première version basée sur les caractères.
Dernière Edition: 17 Jan 2008
