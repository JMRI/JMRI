Guide Matériel JMRI: Système Wangrow One

Matériel Supporté

  • Wangrow System One

Limitations

Certaines centrales de commande Wangrow ont un« circuit de contrôle analogique en option . Dans ces dernieres, les broches 5, 6 et 9 sont utilisées avec un convertisseur de tension analogique, tension comme un régulateur. Si vous avez un de ces centrales de commandes un câble normal série à 9 broches ne fonctionnera pas! Vous aurez besoin de faire un câble personnalisé qui relie simplement les broches " Envoyer données(Dtx)", "Recevoir données(Rtx) »et« masse ». Consultez votre manuel Wangrow pour les détails.

Congiguration

  1. Pour connecter votre ordinateur à un système Wangrow, vous pouvez habituellement connecter la centrale de commande au port série de l'ordinateur en utilisant un câble série à 9 broches.

  2. Puis allez dans le panneau des préférences d'un programme basé sur JMRI. Cela s'ouvre automatiquement la première fois qu'un programme est exécuté, ou vous pouvez le sélectionner dans le menu "Édition"

  3. Sélectionnez "Wangrow" dans la boîte de sélection en haut. Vous pouvez alors sélectionner le port série approprié dans la seconde boîte de sélection.

  4. La boîte de sélection des vitesse de transmission sera fixée à 9600 bauds.

  5. La série de boutons radio étiquetés «style graphique" vous permet de sélectionner la façon dont le programme va apparaitre. "Metal" est l'apparence native Java chercher disponible sur tous les types. D'autres choix peuvent varier avec votre type d'ordinateur. "Motif", "Windows", "MacOS" et "MacOS adaptative» apparence native sous Linux, Windows, MacOS "Classic" et Mac OS X respectivement.

  6. Dans la boîte de sélection en bas, vous pouvez choisir l'option par défaut du format de programmateur. Vous pouvez le remplacer à chaque fois que vous ouvrez le programmateur, ce n'est donc ici juste que par commodité.

  7. Cliquez sur "Enregistrer". Il vous sera demandé si vous voulez quitter le programme, cliquez sur "Oui".

  8. Redémarrez le programme. Vous devriez être opérationne.
