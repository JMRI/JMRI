Prise en Charge Matériel : DCC++ (déprécié)

Le projet original DCC++ n'est plus maintenu.
Pour la documentation actuelle sur l'utilisation de DCC-EX avec JMRI, voir la page EX-CommandStation (DCC-EX).

DCC++ était une Centrale DCC Open Source basée sur Arduino. Il a été remplacé par l'EX-CommandStation de l'équipe DCC-EX, qui continue de fonctionner avec JMRI et fonctionne sur le matériel DCC++ existant.

JMRI continue de prendre en charge les anciennes Centrales DCC++ via la même connexion DCC-EX utilisée pour les EX-CommandStations. Depuis les Préférences JMRI, sélectionnez DCC-EX comme Fabricant du Système.
