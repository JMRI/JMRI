Guide Installation JMRI: Connexion Serveur Terminal

Un serveur Terminal est une boîte qui connecte un ou plusieurs ports série à un réseau afin qu'ils puissent être accessibles à partir d'un ordinateur distant. JMRI s'exécutant sur un ordinateur avec une connexion réseau peut se connecter à une centrale de commande ou EasyDCC NCE par l'une de celle-ci. Cela vous permet vous de mettre l'ordinateur et la centrale commande aussi éloigné que vous le souhaitez.

Pour faire ce travail, vous devez configurer à la fois JMRI et le serveur terminal, et de faire les connexions appropriées.

Cette utilisation a été initialement créée par Zuhn David, qui a une page web décrivant en détail comment il l'a mis en place pour EasyDCC. Ce qui suit est juste un aperçu de la présente. Le processus des NCE est similaire.

  • Pour configurer JMRI:
    • Sélectionnez "Préférences "dans le menu "édition"
    • Sélectionnez l'option protocole "EasyDCC sur le serveur terminal "
    • Entrez l'adresse IP et le numéro de port de connexion correspondant au port série sur le serveur terminal
    • Cliquez sur enregistrer, puis quittez le programme
    • Configurer le serveur Terminal pour se connecter à la station de commande: 9600 bauds, sans parité, 8 bits.
    • Connectez un câble série à partir du serveur terminal à la centrale de commande. Selon le type de serveur terminal, cela pourrait exiger que vous fassiez les fils sur mesure pour obtenir les bonnes connexions. Notez que vous voulez que le fil "émette des données" à une extrémité pour se connecter à la "réception de données" sur l'autre. Faites de même pour les deux directions, et il comprend également un fil de terre.
    • Faire les connexions de réseau nécessaires.
    • Démarrez le programme en place. Si tout est OK, JMRI va démarrer et vous montrer qu'il est correctement connecté
