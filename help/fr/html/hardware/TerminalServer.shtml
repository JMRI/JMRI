Guide Installation JMRI: Connexion Serveur Terminal

Un Serveur Terminal est une boîte qui connecte un ou plusieurs ports série à un réseau afin qu'ils puissent être accessibles à partir d'un ordinateur distant. JMRI s'exécutant sur un ordinateur avec une connexion réseau peut se connecter à une centrale EasyDCC ou NCE ou des nœuds C/MRI par l'une de celle-ci. Cela vous permet de mettre l'ordinateur et la centrale aussi éloigné que vous le souhaitez.

Matériel prit en charge

Cette utilisation a été initialement créée par Zuhn David, qui a une page web décrivant en détail comment il l'a mis en place pour EasyDCC. Pour NCE et C/MRI le processus est similaire.

Connexion

Pour faire ce travail, vous devez configurer à la fois JMRI et le Serveur Terminal, et de faire les connexions appropriées.

Réglages

  • Pour configurer JMRI:
    • Sélectionnez "Préférences "dans le menu "édition"
    • Sélectionnez "EasyDCC" comme System Manufacturer
    • Sélectionnez "EasyDCC over terminal server"comme System Connection
    • "Entrez l'adresse IP et le numéro du Port TCP/UDP correspondant au port série sur le serveur terminal
    • Cliquez sur enregistrer, puis quittez le programme
  • Configurer le Serveur Terminal pour se connecter à la centrale: 9600 bauds, sans parité, 8 bits.

    Câblage

    • Connectez un câble série du Serveur Terminal à la centrale. Selon le type de Serveur Terminal, cela pourrait exiger que vous fassiez des fils sur mesure pour obtenir les bonnes connexions. Notez que vous voulez que le fil "émette des données" à une extrémité pour se connecter à la "réception de données" sur l'autre. Faites de même pour les deux directions, et il comprend également un fil de terre.
    • Faire les connexions de réseau nécessaires.

    Démarrez JMRI. Si tout est OK, l'application va démarrer et vous montrer qu'il est correctement connecté
