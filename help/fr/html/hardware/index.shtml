JMRI : Support matériel

JMRI prend en charge un large éventail de produits de type "Centrale" DCC de divers vendeurs d'automatisation de modèles réduits ferroviaire. Nous prenons également en charge les accessoires et les dispositifs des réseaux non-DCC qui se connectent via une variété de réseaux et de protocoles. Cette page énumère les matériels et les réseaux avec lesquels JMRI fonctionne, et fournit des liens vers des informations sur la manière de configurer JMRI pour qu'il fonctionne avec chaque type particulier.

IPOCSMR
IPOCSMR est une implémentation de logiciel libre d'une interface basée sur IP pour contrôler les objets du chemin de fer miniature tels que les moteurs d'aiguillages , signaux et détecteurs. L'IPOCSMR est conçu en utilisant la norme ESP8266 comme la liaison Wi-Fi vers les contrôleurs d'objets et Arduino UNO pour réaliser le contrôle direct de l'objet. Les interfaces IPOCS JMRI utilisent MQTT et comprend un logiciel d'interface fonctionnant dans un logiciel JMRI de communication IP fonctionnant dans un environnement ESP8266 et un logiciel de contrôle d'objets fonctionnant dans Arduino. Les autres informations, descriptions des systèmes et les logiciels peuvent être trouvés sur GitHub.

Configurer : Connexion à un système IPOCSMR .

IPOCS
IPOCS, abréviation de Internet Protocol Object Controller System, est un protocole Open Source qui est basé sur le protocole FFIS TCC-OCS pour la communication entre les objets ferroviaires en bordure de voie et un Centre de contrôle du trafic . La spécification FFFIS TCC-OCS fait partie de l'ERTMS-R.

Informations générales sur le matériel

Rétrosignalisation Aiguillage

Si votre mise réseau utilise un mécanisme de rétrosignalisation pour détecter les positions réelles des aiguillages, par exemple un microcontact connecté aux barres de de commande ou à un décodeur qui renvoie une confirmation, JMRI peut l'utiliser pour suivre ce qui se passe réellement sur le réseau. Nous appelons cela "Feedback", et comment le configurer et l'utiliser est décrit sur le page sur la rétrosignalisation aiguillage .
