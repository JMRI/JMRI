Structure Matériel: CBUS - Désignation

Cette page explique comment JMRI utilise les Noms Systèmes pour accéder aux ressources liées à CBUS.

Noms Systèmes

JMRI fournit une interface au réseau réel via des objets internes, tels que les Aiguillages, les Capteurs, etc qui sont étroitement "mappés" pour correspondres à des périphériques physiques. Ce "mapping" est réalisée via les "Noms Systèmes", qui à la fois identifie comment accéder à un dispositif particulier, et sont la clé de la recherche d'un objet correspondant dans JMRI.

Événements CBUS

Le CBUS MERG communique par messages avec un format fixé: Un octet de commande et d'information sur la longueur, suivi éventuellement par des octets de données supplémentaires. Et ceci est la forme la plus simple, il est utilisé pour envoyer des "événements" identifiables. Á leur tour, les événements sont de deux types: "ON" et "OFF".
