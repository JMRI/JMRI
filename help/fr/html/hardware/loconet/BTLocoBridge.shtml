Support Matériel: connexion a travers un Adaptateur Bluetooth LocoBridge

Le LocoBridge vous permet de vous connecter à distance à travers Bluetooth à un adaptateur, qui vous connecte à un LocoNet pour commabnder votre réseau.

Pour avoir plus d'information sur le LocoBridge lui-même, voir ces pages.

Installation

Sur les systèmes récents Mac et Windows, tous les logiciels nécessaires sont fournis par le téléversement de JMRI.

Sur Linux et autres sustèmes, vous aurez besoin d'avoir une libbluetooth installée. Dépendant de la distribut Linux utilisée, vous pouvez avoir besoin de commandes telle que "sudo apt-get install libbluetooth" ou peut être "sudo apt-get install libbluetooth-dev". Pour avoir plus d'information,voir ces pages, qui décrivent les exigences d'utilisation de la bibliothèque Blue Cove que JMRI utilises pour accéder à Bluetooth.

Configuration

Pour configurer une application JMRI pour utiliser le:

  1. Démarrez JMRI
  2. Ouvrez la fenêtre Préférences depuis le menu Édition et configurez la Connexion comme:
    • Sélectionnez System manufacturer pour: Digitrax
    • Sélectionnez le System connection pour: BT LocoBridge
    • Sélectionnez votre LocoBridge pour: the Bluetooth adapter
    • Sélectionnez votre type de Centrale
  3. Cliquez "Enregistrer" au bas de la fenêtre Préférences, quittez JMRI et redémarrez.
