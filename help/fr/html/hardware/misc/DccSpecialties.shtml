JMRI: Programmation Poduits DCC Specialties

DCC Specialties fait plusieurs contrôleurs puissants pour les chemins de fer modèle. Leurs deux contrôleurs de moteur switch sont appelés les Hare et Wabbit. Ils font aussi un détecteur de canton intelligent appelé Block Watcher, et les puissants contrôleurs d'Alimentation PSX etPSX-AR .

Ceux-ci peuvent faire beaucoup de choses, mais ils peuventt exiger du travail pour les configurer.

Brett Scott et Denny Whitaker ont écrit des scripts JMRI pour aider à les programmer, et ont fourni un beau document , (lien cassé le 17/01/2012) sur la façon de les utiliser. Le documentp couvre également comment il documente leurs réseaux, et discute les avantages et inconvénients de plusieurs manières de connecter le Hare et le Wabbit.

vous pouvez télécharger leurs scripts depuis le site web JMRI, et il est aussi disponible dans des exemples de répertoires de scripts dans le Téléchargements JMRI .
