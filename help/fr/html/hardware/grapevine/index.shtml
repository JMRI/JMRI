      Connexion

      Réglages

      Pour configurer la connexion:

      • Ouvrez les préférence JMRI dans le menu "édition"
      • Dans l'onglet Connexion, sélectionnez "Protrak"comme System Manufacturer
      • Sélectionnez "Grapevine (ProTrak) Layout Bus" comme System Connection (la seule possibilité)
      • Cliquez "Enregistrer" & redémarrez

      Avant que vous accédiez au Nœeud Gravpine dans JMRI, Vous devez le définir pour le programme. Vous pouvez le faire soit depuis la fenêtre Préférences dans le menu "édition du menu principal ou en utilisant l'outil Configurer les Nœuds dans le menu Grapevine.

      Les Noms Systèmes Gravrpine commencent avec un système de lettre "G". Après ceci , il y a une Lettre type, puis un numéro de nœud et un numéro de broche.
      La dénomination et les connecteurs entrée/sortie sont décrits dans une page séparée.
