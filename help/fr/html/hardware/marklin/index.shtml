Documentation Matériel : Marklin Centrale CS 2

JMRI peut se connecter à une centrale Märklin CS2 en utilisant l(interface ethernet intégrée.
Il est possible pour plusieurs ordinateurs de se connecter à l'aide de CS2 IP/UDP..

La CS2 utilise son propre protocole de BUS CAN fontionnant sur UDP, les réponses de la CS2 sont envoyées sous forme de message de diffusion et par conséquent, les ordinateurs qui se connectent au CS2 doit être dans le même sous-réseau IP sinon les messages de réponse ne seront pas vus.

Cette mise en oeuvre du protocole CAN CS2 a été développé et testé sur la version du firmware 2. Cette interface est encore en développement, si vous trouvez des problè mes alors s'il vous plaît poser une question sur le groupe Utilisateurs JMRI

Le Support à la Marklin CS2 était d'abord disponible en JMRI 3.1.1.

Limitations

Cette interface est encore en développement; si vous trouvez n'importe quel problème, alors s'il vous plait envoyez un message sur la liste des Utilisateurs JMRI mailing list.

Le Couplage et la Programmation ne sont pas encore supportés.

Connexion

Pour configurer la connexion

  • Ouvrez les "Préférences" JMRI dans le menu "Edition"
  • Sélectionnez "Märklin" comme le manufacuturer, puis le type de connexion qui est Commande Station 2,
  • Entrez l'adresse attribuée à la CS2.
  • cliquez sur "Enregistrer".

La mise en oeuvre pour les Aiguillages, Capteurs, Régulateurs et Alimentation est disponible pour une unité CS2 connectée à JMRI

La prise en charge des régulateurs inclut la possibilité de sélectionner un protocole spécifique (Märklin, DCC & MFX)
