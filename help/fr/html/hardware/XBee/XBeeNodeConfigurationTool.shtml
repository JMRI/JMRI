Configuration de Nœud XBee

L'outil de Configuration de Nœud XBee peut être utilisé pour ajouter des nœud XBee à la configuration JMRI.
Vous pouvez accéder à l'outil depuis le menu quand une connexion XBee est active.

Au sommet se trouvent trois boîtes déroulantes pour:

  • l'adresse des nœuds en 16 bit
  • le GUID 64 bit
  • le champ d'identification du Nœud (NI) .

La sélection d'une adresse dans la première fenêtre apportera l'information pour le sélectionné dans la moitié moitié du volet outil

Tableau

Il y a trois colonnes listant la configuration actuelle des broches E/S du nœud

  • La première colonne liste les numéros des pin DIO.
  • La seconde colonne liste le nom système associé avec la pin ou - free - si la pin n'est pas actuellement affectée à un équipement.
  • La troisième colonne liste les Noms Utilisateur associés aux Noms Systèmes.

Boutons

Le bouton libellé "Ajouter Nœud" ajoutera le nœud associé avec les 16 bits d'adresse à la liste des Nœuds.

Le bouton libellé "Découvrir Nœuds" qui lancera pour le Xbee une routine de découverte de nœud et cherchera de nouveau nœud sur le réseau.

Le bouton libellé "Modifer Nœud" vous autorisera a modifier l'information nœud pour le nœud sélectionnée.

Le bouton libellé "Effacer Nœud" enlévera le nœud actuellement sélectionné.

Le bouton libellé "Terminé" fermera l'outil de Configuration de Nœud XBee.
