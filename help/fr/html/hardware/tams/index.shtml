JMRI: Support TAMs

JMRI peut se connecter à une centrale MasterControl TAMS en utilisant l'interface USB intégrée et/ou l'interface série.
L'interface USB est livré avec les pilotes pour émuler un port série.
Le port USB fonctionne sous Linux et Windows - les deux ont été testés
. Les connecteurs USB et interface série peuvent être utilisés en même temps.

Le TAMS MC utilise le protocole P50X (Uhlenbrock/Märklin).

Cette interface est encore en développement, si vous rencontrez des problèmes alors s'il vous plaît interroger le groupe d'utilisateurs Groups.io

Pour configurer la connexion:

  • Ouvrez les "Préférences" JMRI dans le menu "Edition"
  • Sélectionnez "TAMS" comme fabricant, puis le type de connexion,
  • Sélectionnez l'interface série approprié.
  • La vitesse de transmission configurée dans la MC sur le port USB est fixé à 57600 bps (8n1: 8 bits, sans parité, 1 bit d'arrêt).
  • Sur le port série de la vitesse par défaut est fixée à 2400 bps. Ceci peut être changé sur le MC dans les étapes habituelles entre 1200 et 57600 bps
  • cliquez sur "Enregistrer".

Le support pour les Aiguillages, Régulateurs et alimentation est disponible pour une unité de TAM connectée JMRI.

Consistant, capteurs et RailCom ne sont pas encore en charge.

Turnout, Programming, Throttle and power support is available for a JMRI-connected TAMs unit.

Couplage, Capteurs et RailCom ne sont pas supportés.

Le Support pour TAMS MC était d'abord disponible sur la version JMRI 2.99.7.
