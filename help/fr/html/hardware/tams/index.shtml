Support matériel : Tams MasterControl

La fonctionnalité suivante est disponible pour une Unité Tams connectée au JMRI :

  1. Assistance à la commande et à la programmation des locomotives
  2. Définir et commander les aiguillages connectés via le DCC
  3. Définir et obtenir un retour d'information des capteurs connectés au S88
  4. Surveiller les commandes Tams
  5. Envoyer les commandes Tams
  6. commande de l'alimentation
  7. Lire la base de données Tams Loco
  8. Lire la version du micrologiciel et le numéro de série de la centrale Tams
  9. Connexion au simulateur tams

Le soutien à Tams MC a été disponible pour la première fois pour la version JMRI 2.99.7.

Limitations

Limitations connues :

  1. Une seule Tams MasterControl peut être connectée
  2. Régulateurs : En raison de limitations techniques dans l'ancien protocole P50, JMRI ne peut contrôler et afficher que 14 fonctions et il ne peut saisir que les changements de statut provenant de l'extérieur de JMRI pour f1 à f8. Après le le premier changement de Régulateur JMRI les 14 fonctions s'afficheront à nouveau correctement en cas de divergence. La vitesse, la direction et la lumière devraient être pris en compte. Une amélioration du protocole P50 a été réalisée de manière à ce que les fonctions f15 à f24 peuvent également être correctement contrôlées, mais cela n'est pas encore implémentée dans cette version.
  3. Moniteur de commande tams : Les messages binaires ne sont pas vraiment lisibles par l'homme
  4. Envoyer les commandes Tams : Seul un ensemble limité de réponses aux commandes est interprétée par JMRI, bien qu'elles soient toutes correctement affichées dans le moniteur de commande

Cette interface sera encore améliorée, si vous trouvez un problème veuillez alors le signaler au groupe des utilisateurs de JMRI.

Connexion

Câblage

JMRI peut se connecter à une commande Tams MasterControl (MC) Station utilisant l'interface USB et/ou série intégrée.
Les interfaces USB et série peuvent être utilisées en même temps .

L'interface USB est livrée avec des pilotes pour émuler un port série
Le port USB fonctionne sous Linux et Windows - les deux ont été testé.

Réglages

Le Tams MC utilise le protocole (Uhlenbrock/Märklin) P50X .

Pour configurer la connexion:

  • Ouvrir les "Préférences" de JMRI dans le menu "Edition"
  • Sélectionnez "Tams" comme fabricant
  • Sélectionnez l'interface série appropriée comme Type de connexion
  • Le débit en bauds configuré dans le MC sur le port USB est fixé à 57600 bps (8n1 : 8bit, pas de parité, 1 stop bit)
  • Sur le port série, la vitesse par défaut est fixée à 2400 bps. Cela peut être modifié sur le MC en suivant les étapes habituelles entre 1200 et 57600 bps
  • Cliquez sur "Sauvegarder" et redémarrez JMRI.

Outils JMRI Tams

  • Le trafic du réseau Tams peut être visualisé avec le Moniteur Tams
  • Les messages peuvent être envoyés aux appareils Tams avec Envoyer des paquets Tams
  • Le L'outil LocoData vous permet de lire et d'écrire données de la locomotive dans la Centrale Tams.
