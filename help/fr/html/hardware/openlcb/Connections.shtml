Documentation Matériel: OpenLCB - Connexions

Cette page explique comment JMRI peut être connecté à OpenLCB

Connexion

Via un TCH Tech CAN=USB Connector

Via un adaptateur GridConnect

Via une interface Réseau GridConnect

Via a Lawcell CAN USB

Via a MERG CAN-RS/-USB

Vous pouvez simuler une connexion à distance OpenLCB, par exemple pour déboger des choses, sur beaucoup d'ordinateurs en utilisant la commande "nc": 

nc -l 12022
