Support matéériel : Fleischmann/Roco z21/Z21

La Roco z21/Z21 est une centrale qui soutient LocoNet, R-Bus et XPressNet, et il comprend un port Ethernet. L'appareil Z21 est également vendu sous la marque Fleischmann.

Pour se connecter à l'appareil via le LocoNet ou le XPressNet veuillez consulter les pages de connexion de l'interface LocoNet et XPressNet .

Cette page précise la connexion au z21/Z21 via le Port Ethernet tel que documenté sur le site web Z21.

Matériel supporté

Roco a deux appareils qu'elle vend comme la Z21 Blanche et le Z21 noir. Le Z21 Blanche est aussi parfois appelé le z21. Les deux appareils prennent en charge les connexions via le Port Ethernet , aux appareils R-Bus et aux appareils XPressNet. Le Z21 Noire prend également en charge les connexions via LocoNet et le bus CAN de Zimo .

Limitations

JMRI

JMRI supporte actuellement la Z21 XPressNet sur la mise en œuvre d'Ethernet. JMRI fournit également un soutien pour la configuration des drapeaux de connexion et de l'interface RailCom Z21 .

Z21 Blanche

La Z21 Blanche ne donne pas accès à l'interface LocoNet .

Le z21 blanc n'inclut pas de sortie de voie de programmation. Il n'est pas clairement indiqué si elle peut ou pas lire les valeurs du décodeur en mode service .

Z21 Noire

Il n'y a pas de limitations connues de la Z21 Noire cette fois-ci.

Connexion

Connexion au Roco z21 ou Z21 via la connexion Ethernet

e

Note : La première version de JMRI soutenant La connexion Ethernet Lenz LIUSB était v 3.11.7.

  1. Pour connecter votre ordinateur au z21 ou au Z21 via le Port Ethernet, connectez votre z21/Z21 à votre ordinateur en utilisant le câble Ethernet fourni.
  2. Une fois l'appareil connecté, se référer à la documentation de l'appareil pour obtenir des informations sur la définition de l'adresse IP. L'adresse IP par défaut est 192.168.0.111. Prenez note de l'Adresse IP à utiliser lors de la configuration de l'appareil.
  3. Démarrez un des programmes basés sur JMRI, puis allez à la page du panneau de préférences. Celui-ci s'ouvre automatiquement la première fois qu'un programme est exécuté, ou vous pouvez le sélectionner dans le menu "Editionr". .
  4. Sélectionner "Roco" dans la sélection du fabricant du système et "Roco Z21" dans la boîte de sélection supérieure. Il y a deux options de configuration disponibles pour le LIUSB-Ethernet.
  5. Entrez l'adresse IP que vous avez enregistrée précédemment dans la case intitulée "Adresse IP/Nom d'hôte"
  6. Cliquez sur "Sauvegarder". On vous demandera si vous voulez le faire pour le pour arrêter, cliquez sur "Oui".
  7. Redémarrer le programme. Vous devriez être opérationnel.

Connexion au simulateur Roco Z21

  1. Démarrez un des programmes basés sur JMRI, puis allez à la page du panneau de préférences. Celui-ci s'ouvre automatiquement la première fois un programme est exécuté, ou vous pouvez le sélectionner dans le menu "Edition". .
  2. Sélectionner "Roco" dans la sélection du fabricant du système et "Z21 Simulator" à partir de la boîte de sélection connexion au système . Il n'y a pas d'étapes de configuration supplémentaires nécessaire pour utiliser le simulateur Z21.
  3. Cliquez sur "Sauvegarder". On vous demandera si vous voulez le faire pour le pour arrêter, cliquez sur "Oui".
  4. Redémarrer le programme. Vous devriez être opérationnel.

Outils JMRI z21/Z21

Les outils suivants sont disponibles dans le menu lorsque cette connexion est active:

  • OUvrir le moniteur Z21 - Cet outil vous permet de surveiller le trafic à destination et en provenance de la Z21.
  • Envoyer la commande Z21
  • Outil de configuration Z21 - Cet outil vous permet de vérifier et de mettre les drapeaux Z21 liés à cette connexion. Les drapeaux sont fixé automatiquement par certains services que JMRI démarre, tels que la connexion XPressNet Z21.

En plus du menu spécifique Z21, vous trouverez également le menu Z21XPressNet, qui contient les même outils XPressNet trouvés sur n'importe quel système XPressNet.
