Matériel supporté

Interfaces informatiques

JMRI soutient le SPROG à la fois en tant que Programmeur de décodeur, et comme mini-station de commande. Pour en savoir plus ces capacités, voir le Manuel SPROG sur le site web SPROG.

Notez qu'il existe plusieurs versions différentes du matériel SPROG, et encore plus de versions du software SPROG software.
Les modèles SPROG suivants fonctionnent avec la version actuelle de JMRI:

  • SPROG II
  • SPROG II USB
  • SPROG 3 (USB)
  • SPROG Nano (uniquement en tant que centrale, en utilisant un booster externe)
  • Pi SPROG One
  • Pi SPROG Nano

Connexion

Câblage

Les SPROG originaux étaient connectés par un câble série à 9 broches.
Les SPROG actuels ne sont disponibles qu'en USB.
Connectez votre SPROG à l'aide du câble approprié et mettez-le sous tension.

Configuration de SPROG

SPROG utilise un "Mot Mode" pour définir un certain nombre de modes de fonctionnement.

Dans tous les SPROGs récents, la valeur par défaut du Mot Mode est correcte pour les modes "programmeur" et "centrale". Pour les SPROG plus anciens, vous pouvez vérifier ou modifier le mot mode comme suit:

Ouvrir le Moniteur de commande et lui Envoyer la commande outils du menu SPROG. Envoyer une commande "M" utilisant Envoyer Commande dans le panneau de commande .
Vous devriez voir une réponse dans la fenêtre du moniteur de commande, quelque chose comme:

P>M=h800

Le préfixe h indique une valeur hexadécimale. Si le préfixe valeur que vous voyez est différente, alors il faut le changer en envoyant la commande "M h800". Faites suivre cette commande de la commande "W" pour enregistrer le changement dans la mémoire EEPROM de SPROG afin qu'il soit efficace à chaque fois que vous mettez le SPROG sous tension. Sinon, la nouvelle valeur sera perdue lorsque vous éteignez le SPROG.
Ne soyez pas tentés d'expérimenter avec d'autres valeurs de Mode Word car cela pourrait entraîner un comportement erratique de votre SPROG.
