JMRI: Outils Généraux

JMRI fournit plusieurs outils qui sont optimisés à des fins spécifiques. Outils généraux:

Capteurs

Un capteur est une façon pour JMRI de représenter une information venant du réseau.

Aiguillages

JMRI appelle "Aiguillage" à peu près n'importe quelle sortie vers le réseau.

Régulateurs

Vous pouvez ouvrir plusieurs régulateurs logiciels pour piloter vos locomotives DCC.

Couplage (UM)

JMRI fournit les outils pour gérer les couplages de locomotives.

Signaux

Signaux (ou plus formellement Têtes Signal) sont comment JMRI représente et contrôle les Têtes de signal sur un réseau. Ils sont un type spécifique de périphérique de sortie qui peut être défini pour différentes couleurs: Rouge, Jaune, Vert, Jaune clignotant, etc

Lumières

Les lumières sont les bits de sortie numériques qui fonctionnent comme des interrupteurs marche/arrêt. Les lumières ont été créées pour fournir une commande facile pour les lumières d'un panneau de commande et l'éclairage des paysages. Les sorties lumières peuvent aussi être utilisées à des fins qui nécessite un interrupteur marche/arrêt. La sortie lumière peut être utilisée, par exemple, pour activer une découpleur électromagnétiques pour un temps pré-déterminé en réponse à un bouton du TCO

Itinéraires (Routes)

Les itinéraires sont un outil pour le pilotage de multiples aiguillages lorsque quelque chose se passe sur le réseau. Vous pouvez avoir un panneau d'interrupteur pour tous les aiguillages de vos accès au triage, etc. Ils peuvent également changer l'état des capteurs, lire des fichiers audio, et invoquer des scripts. Ils ont certaines capacités logiques simples pour décider quand régler les sorties de l'itinéraire.

Logix

Un Logix est un outil puissant pour entrer des expressions logiques qui peuvent contrôler les éléments d'un réseau. Chacun peut suivre de multiples entrées et des opérations logiques contenant un certain nombre d'expressions conditionnelles, dont chacune peut prendre des mesures simples. Pour un contrôle plus puissants, combiner les raisonnements logiques d'un Logix avec la puissante sortie de commande d'Itinéraires.

LItinéraires

LItinéraires étend les capacités des Itinéraires par leur mise en oeuvre en tant que Logix. LItinéraires peut être défini avec un nombre illimité de capteurs, d'aiguillages, de Lumières, et/ou Tête Signal pour des actions ou des contrôles. LItinéraire peut fournir un point de départ facile pour la création de nouveaux Logix car ils peuvent être étendus pour inclure toute les Capacités de Logix.

Horloges Rapides

JMRI peut à la fois fournir différents types d'horloges rapides sur l'écran de l'ordinateur, et aussi travailler avec les horloges rapides déjà installées sur votre réseau.

Tachymètre

Un outil pour déterminer la vitesse à l'échelle de vos locomotives . Vous pouvez l'utiliser pendant leurs réglages, pour éviter les excès de vitesse, ou pour tout ce que vous voulez.

Dispatcher

L'outil offre des fonctions de répartition et organise l'information relative à l'expédition des trains sur un réseau en modélisme ferroviaire. Sa principale fonction est l'allocation des sections de voie pour que des trains différents puissent circuler sur le réseau.

Exploitations (Opérations)

L'outil d'Exploitation vous permet de produire par ordinateur les manifestes et les listes de manoeuvre de votre chemin de fer.

Outils spécifiques LocoNet

Un jeu d'outils spécifiques Digitrax LocoNet :
  • Moniteur LocoNet
  • Moniteur Slot
  • Source de paquet
  • programmateur PM4
  • programmateur BDL16
  • programmateur SE8c

Écriture de scripts

S'il n'y a pas un outil de pré-programmés pour faire ce que vous voulez, vous pouvez presque certainement écrire un script qui fera exactement ce qu'il faut.

Au lieu de définir notre propre langue, les scripts JMRI utilisent le language "Python" qui est couramment utilisée dans le développement web. Bien qu'il vous prendra un peu de temps pour vous y habituer, à la fin il fournit beaucoup de puissance pour contrôler votre réseau de la façon que vous souhaitez..
