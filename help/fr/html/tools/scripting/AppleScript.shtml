JMRI : Architecture Ouverte de Script

Sur macOS, vous pouvez utiliser l'Open Scripting Architecture (OSA) avec JMRI pour envoyer des instructions à OS X ou autre applications.

Vous pouvez intégrer ou appeler des scripts OSA dans les scripts JMRI, les invoquer pour que votre Mac exécute des fonctions en dehors de JMRI. Un exemple de cela se trouve dans le AppleScript.py exemple de script distribué avec JMRI. Il exécute un simple AppleScript:

  indiquer à application "Finder " de cr√©er un nouveau dossier à la fin du bureau

L'exemple de script montre comment faire, et peut facilement de base pour exécuter vos propres scripts OSA à partir de JMRI. .

JMRI 4.5.2 ou plus récent peut appeler directement AppleScript sans l'intégrer dans un autre script en ajoutant JMRI AppleScript Support à votre installation JMRI. Notez que cela pourrait ne pas fonctionner dans les versions futures de macOS ou de Java.
