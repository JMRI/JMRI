JMRI : Horloges rapides

JMRI offre deux types de possibilités d'horloge rapide:

  • Interne - l'horloge des ordinateurs fournit l'information temps
  • Externe - le système DCC (ou un objet sur un bus de contrôle de configuration) fournit les informations sur le temps

Cette page d'aide décrit la structure générale.

Il y a deux parties différentes à considérer : Une "base de temps fournit les informations de base sur le temps, tandis qu'un ou plusieurs les écrans vous montrent ce qui se passe.

La Base de Temps de l'Horloge Rapide

Pour créer une base de temps, utilisez PanelPro > Outils > Horloges > Configuration Horloge rapide... Cela vous donne une fenêtre où vous pouvez :

  • Régler le temps
  • Démarrer et arrêter l'horloge
  • Définir le rythme auquel l'horloge tourne
  • Sélectionner la source de base de temps pour la commande (généralement, c'est l'horloge interne de l'ordinateur)

Affichage de l'horloge rapide

Une fois que vous avez créé une base de temps en la configurant vous pouvez ouvrir des horloges qui s'afficheront sur votre écran, le tout avec un bouton optionnel "pause/exécution".

Toutes les horloges peuvent être redimensionnées en faisant glisser les bords de la fenêtre

  • Outils > Horloges > Horloge analogique - L'horloge circulaire traditionnelle avec aiguilles.

  • Outils > Horloges > Horloge LCD - une horloge numérique.

  • Outils > Horloges > Horloge Nixie - une horloge numérique qui ressemble aux Nixie à l'ancienne

  • Outils > Horloges > Horloge Pragotron - horloge à bascule utilisée dans le réseau des chemins de fer tchécoslovaques &ndash ; ČSD depuis 4.19.2

  • Horloge du panneau - vous pouvez également intégrer une horloge dans un panneau de l'éditeur du panel.
    Les horloges analogiques peuvent être intégrées dans les panneaux de l'éditeur de réseau (au lieu d'une fenêtre séparée) via Fenêtre du panneau > Options > Ajouter > Ajouter une horloge rapide

Quand une horloge est en marche, JMRI crée un capteur interne et une variable mémoire que vous pouvez utiliser pour contrôler et accéder l'horloge. Le capteur ISCLOCKRUNNING contrôle si le l'horloge est en marche (ACTIVE) ou arrêtée (INACTIVE). La mémoire IMCURRENTTIME contient l'heure actuelle de l'horloge rapide, et est facilement accessible depuis Logix et scripts.

Les lumières peuvent être un moyen facile de mettre en place une logique d'horloge rapide, voir Contrôleur de lumière - Suiveur d'horloge rapide.
