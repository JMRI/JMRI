Possibilités de l'Horloge Rapide LocoNet JMRI

Un LocoNet Digitrax peut contenir une base de temps propre pour son horloge rapide. Ceci peut être assurée soit par une centrale de commande DCS100/DCS200, ou par un module séparé d'horloge rapide telles que ceux faits par les technologies ferroviaires Logic. Cette page explique comment JMRI peut soit utiliser ou contrôler les horloges rapides LocoNet.

Contrôle de l'horloge LocoNet

L'outil "Moniteur Horloge" dans le menu "LocoNet" fournit un moyen pratique pour:
  • Lire le contenu actuel de l'horloge rapide LocoNet
  • Régler la date et l'heure
  • Réglez la vitesse. Notez que vous ne pouvez pas explicitement mettre en pause une horloge LocoNet, mais vous pouvez l'arrèter en fixant le taux de vitesse á zéro.

Si vous utilisez une centrale de commande DCS100 ou DCS200 pour créer votre horloge rapide LocoNet, vous avez sans doute remarqué qu'elle n'indique pas particulièrement le bon temps. Si vous sélectionnez le "Corriger le Maitre Horloge Rapide LocoNet", JMRI va ajuster le rythme de l'horloge LocoNet afin qu'il conserve au mieux le temps.

Utilisation l'Horloge LocoNet

Vous pouvez également avoir JMRI qui utilise l'horloge LocoNet pour sa propre base de temps interne. Cela permet de maintenir les horloges que JMRI affiche sur l'écran synchronisées avec l' Horloge LocoNet, d'oó l'affichage sur les régulateurs DT400, etc

Pour ce faire, sélectionnez l'option "Horloge Rapide LocoNet pilote l'Horloge Interne" dans la Fenêtre surveiller l'horloge LocoNet. Notez que vous devez faire ceci avant d'ouvrir les fenêtres horloge de sorte que lorsque vous ouvrez les fenêtres horloge elles se connectent á la base de temps correct. Chronométrage heureux!
