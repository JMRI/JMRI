JMRI: Mâts de Signalisation

Informations générales sur les Mâts Signal JMRI

Les objets JMRI "SignalMast" vous permettent d'afficher des aspects réalistes sur les signaux. Pour plus d'information, consultez la page Aspect Signalisation .

Les Mâts de Signalisation sont configurés via le Tableau Mâts de Signalisation Vous pouvez en ajouter de nouveaux avec le bouton "Ajouter ..." sur le tableau, qui vous emmène à la fenêtre "Ajouter Mâts de Signalisation" .

Du tableau, vous pouvez définir des Mâts de Signalisation individuels avec tous les aspects qu'ils sont capables d'afficher en cliquant dans la colonne "Aspect".
