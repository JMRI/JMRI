JMRI: Mâts Signal

Informations générales sur les Mâts Signal JMRI

Les objets JMRI "SignalMast" vous permettent d'afficher des aspects prototypiques sur les signaux. Pour plus d'information, consultez la page Aspect Signalisation .

Les Mâts Signal sont configurés via le Tableau Mât de Signal Vous pouvez en ajouter de nouveaux avec le bouton "Ajouter ..." sur le tableau, qui vous emmène à la fenêtre "Ajouter Mât de Signal" .

Du tableau, vous pouvez définir des mâts de signal individuels avec tous les aspects qu'ils sont capables d'afficher en cliquant dans la colonne "Aspect"
