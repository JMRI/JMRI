JMRI : Feux de Signalisation

Les Feux de Signalisation sont la façon dont JMRI représente les lumières/LED individuelles dans un signal sur votre réseau. JMRI s'occupe des Feux de Signalisation comme un type spécifique de sorties qui peuvent être fixées à différentes couleurs : Rouge, Jaune, Vert, Jaune Clignotant, etc. En fonction du matériel utilisé pour connecter le signal de votre modèle au système DCC, le programme déploie une technique différente pour activer ou désactiver chacune de ces couleurs..
Les Feux de Signalisation peuvent être utilisés pour construire des mâts de signalisation à l'aide de l' Outil Mâts de Signalisation. Selon la façon dont vos signaux sur le réseau sont connectés au Système DCC, les Feux de Signalisation font normalement partie d'un mât de signalisation, mais certains matériels de signalisation sont capables de prendre en charge plusieurs aspects directement, et ne nécessiteront pas l'utilisation d'un Feu de Signalisation individuel.

La Table des Feux de Signalisation

Toutes les Feux de Signalisation connus par JMRI peuvent être référencés en utilisant l'outil "Table des Feux de Signalisation" dans le menu Outils de la plupart des programmes JMRI.

Pour plus d'informations sur la configuration des Feux de Signalisation avec le Table des Feux de Signalisation, veuillez consulter la Page d'aide de la Table des Feux de Signalisation.

Créer une Feu de Signalisation

Pour définir un nouveau Feu de Signalisation, cliquez sur le bouton Ajouter... en bas de la Table des Feux de Signalisation. Cela permettra de vous demander le type de Feu de Signalisation et tout autre renseignement, ces informations sont nécessaires pour le type sélectionné.
Plus d'informations sur la Page d'aide Ajout de Feux de Signalisation .

Modifier un Feu de Signalisation existant

Cliquez sur le bouton Modifier. à droite de l'en-tête de la Table Feu de Signalisation pour ouvrir la fenêtre d'édition du Feu de Signalisation.

Une fois qu'un Feu de Signalisation a été défini, son type de connexion et le nom du système ne peut être modifié. Vous pouvez ignorer les Feux de Signalisation que vous avez définis et que vous n'utilisez pas ; ils ne font pas de mal. Si vous le préférez, vous pouvez supprimer un Feu de Signalisation de la liste en cliquant sur le bouton Supprimer. Enregistrez une copie du Fichier de configuration JMRI comme sauvegarde, lire attentivement le message d'avertissement qui apparaît en cas d'utilisation du Feu de Signalisation par d'autres parties de JMRI !

Commande d'un Feu de Signalisation

Autre que le réglage manuel l'apparition d'un Feu de Signalisation à partir de la table de Feu de Signalisation en cliquant sur l'état actuel d'un Feu dans la colonne Apparence et en sélectionnant un état dans la boîte combinée qui s'ouvre, Les Feux de Signalisation peuvent être commandés automatiquement par JMRI de différentes manières, notamment :
  • Aspect Signalisation, la méthode la plus polyvalente pour afficher les différents types de règles de signalisation de nombreux chemins de fer américains et étrangers, automatiquement en tenant compte de l'état des Aiguillages et des (Occupation) Capteurs;
  • Via Logix, une manière graphique de conmmander les Feux de Signalisation avec des énoncés logiques simples
  • By Scripts, utiliser un langage de programmation complet pour faire ce que vous voulez;
  • Utilisation du Logique Signal Simple Outil, pour la signalisation de base de BAL dutype nord-américain (plus facilement accessible via le première méthode).

Sauvegarde et rechargement de la configuration

Les informations du Feux de Signalisation sont conservées dans votre configuration de réseau, ainsi que les aiguillages, les feux, les capteurs, configuration du panneau de commande, etc.
Sauvegarder ces informations sur le disque, permettant de les recharger la prochaine fois que vous lancez JMRI, utilisez Sauvegarder Configuration... dans le menu Fichier en haut de la Table Feux de Signalisationx (ou autres tableaux dans le menu Outils), ou sélectionnez Sauvegarder Panneau... dans le menu Panneau...

Pour plus d'informations sur la création et l'affichage des panneaux, y compris la façon de montrer les signaux sur vos panneaux, veuillez consulter les Pages Panneaux.
