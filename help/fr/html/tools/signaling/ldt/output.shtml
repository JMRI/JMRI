JMRI: Exemples de Signalisation LDT

Ce fichier contient quelques exemples de configuration pour le module signal LDT LS-DEC. Il y a des exemples de signaux de plusieurs types de systèmes de signalisation:

  • Signaux type L de la SBB (Chemin de Fer Fédéraux Suisse)
  • Signaux DB (Chemin de Fer Fédéraux Allemande )
  • Signaux DR (Chemin de Fer National Allemand)
  • Signaux NS (Nederlandse Spoorwegen, Dutch National Railways)
  • Signaux OEBB (Chemin de Fer Fédéraux Autrichien)
  • Signaux CSD/CD/ZSR
Les exemples sont basés sur des exemples de connexion publiés sur site du fabricant.

Configuration JMRI

Vous devez ajouter chaque signal LDT à l'aide du bouton Ajout/Edition des têtes de Signal depuis le Tableaux Signaux. Sélectionnez "LDT LS-DEC" pour le type de connexion de signal et la fenêtre affichera sept lignes pour la configuration de la commande d'activation envoyée par JMRI au décodeur pour chacune des apparences de signaux JMRI.

Les décodeurs LDT présentent un ensemble particulier de voyants de sortie en fonction de la commande d'aiguillage reçue, donc ce panneau vous permet de connecter ce que JMRI veut (par exemple, "vert", indiquant clair pour continuer) avec l'apparence spécifique du chemin de fer sur le Décodeur LDT ("1 dévié", qui allume l'aspect "1-procéder" pour les signaux à 7 lampes SBB, par exemple).

Les exemples de liens en haut de la page vous donnent des exemples sur la façon de remplir ce formulaire pour obtenir les bonnes apparences pour divers chemins de fer.

Décodeurs pris en charge

Les signaaux JMRI LDT LS-DEC peut être configurée pour décodeurs suivants.

Littfinski DatenTechnik (LDT)

Pour plus d'informations, s'il vous plaît voir http://www.ldt-infocenter.com/

LS-DEC-DR 8 adresses 2 - 4 têtes de signaux

LS-DEC-OEBB 8 adresses 4 têtes de signaux

LS-DEC-SBB 4 ou 8 adresses 2 têtes de signaux

LS-DEC-NS 8 adresse 4 têtes de signal

Modèle Marathon
