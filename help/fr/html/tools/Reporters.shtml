Qu'est ce qu'un Rapport

Un rapport est le moyen pour JMRI de représenter l'information retournée par le réseau.

Un journal peut contenir des informations arbitraires, contrairement à un capteur qui n'est qu'ACTIFf ou INACTIF. Chaque rapport est associé à une source d'information spécifique sur le réseau, et fait que les informations périphériques sont disponibles pour le reste de JMRI.

La mise en oeuvre initiale est pour le transpondage LocoNet (Digitrax) , où le rappost donne accès à l'information à partir de cartes Digitrax BDL168 avec RX4 href="http://www.digitrax.com/prd_statdec_rx4.php">. Lorsque le BDL168 rapporte qu'un décodeur a été vu dans un canton transpondeur ou n'est pas plus visible dans le canton, le rapport correspondant affichera des messages comme "1234 entrer" et "1234 sortie". Il y a aussi un rapport LocoNet mis en oeuvre pour l'affichage des messages des détecteurs Lissy .

Comment Configurer les Rapports

Le Tableau Rapport

Le Tableau Rapport fournit un accès facile au contenu de tous les rapports que JMRI connaît. Il y a aussi un bouton "Ajouter ..." pour créer vos propres Rapports internes; JMRI crée automatiquement Objets Reporter pour les appareils sur le réseau.

Affichage des Valeurs Rapport sur les Panneaux

La valeur d'un rapport peut être affichée sur un panneau via une icône. Pour ce faire:
  • Sur la fenêtre éditeur Panneau de votre panneau, trouver le bouton "Ajouter Rapport" .
  • Dans la boîte de texte à côté de ce bouton, tapez le nom utilisateur ou système du Rapportr désiré.
  • Cliquez sur le bouton. L'icône Rapport sera placée sur le panneau. Si elle n'est pas visible, ce pourrait être parce que la valeur est actuellement vide; utiliser le tableau Rapport pour changer la valeur à quelque chose qui sera visible.
  • Vous pouvez faire glisser l'icône à l'endroit où vous voulez dans la manière habituelle.
  • Le menu pop-up sur l'icône vous permettra de modifier le formatage.
