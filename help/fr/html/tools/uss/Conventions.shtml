JMRI : Recommandations sur les outils de l'USS CTC

Cette page expose certaines conventions qui pourraient être utile lors de la mise en place d'un panneau CTC à l'aide des outils CTC de l'USS. Voici, en gros, comment nous avons réalisé les exemples, et ce pourrait être une approche utile pour vous permettre d'organiser toutes les informations nécessaires.

Les noms utilisateur des appareils :

Les indicateurs du panneau sont pilotés par les Aiguillages appelés tels que"CTC 31 N" ou "CTC 31 R". pour un levier d'aiguillage, ou "CTC 32 R", "CTC 32 C" ou "CTC 32 L" pour un levier de signalisation.

Les Capteurs qui sont connectés pour commuter les entrées du panneau utilisent les mêmes noms, bien que celui du centre sur un signal (c'est-à-dire "CTC 32 C") soit souvent omis.

Les Capteurs détecteur d'Occupation sont étiquetés comme "Circuits de Voie", c'est-à-dire "TC 12". La lampe correspondante sur le panneau est commandée par un Aiguillage du même nom, "TC 12".

Le code du bouton et le code de la lampe sont respectivement appelés "Boutoncode 32" et "Lampe code 32".

L'appel de maintenance des Aiguillages et des Capteurs sont étiquetés par exemple "MC 88".

Le véritable Aiguillage sur le réseau est connu sous le nom de "Aiguillage CTC 3 ".

Les signaux se trouvant autour d'une section de l'OS sont connus par le numéro de la section et son emplacement : "CTC 4 principal supérieur", "CTC 4 principal inférieur", "CTC 4 principal simple", "CTC 4 voie d'évitement".
