JMRI:Outils USS CTC

JMRI contient quelques outils pour aider à l'automatisation des panneaux USS CTT . Ceux-ci peuvent être utilisés soit pour les panneaux à base d'écran , ou à piloter un panneau physique avec de véritables leviers et des voyants.
Suiveur
Les esclaves d'un dispositif de sortie d'Aiguillage pour suivre l'état d'un dispositif de capteur d'entrée. Cela pourrait, par exemple, être un aiguillage du réseau suivi par un commutateur du panneau de contrôle relié à un Capteur.
OS Indicateur
Piloter les voyants d'occupation sur le panneau. Cela prend l'entrée d'un capteur pour montrer la section de voie occupée, et aussi éventuellement peut allumer le voyant si la section a été débloquée.
Levier d' Aiguillage
Contrôle un aiguillage depuis un levier situé sur le panneau, avec les deux voyants. (Pas encore sorti)
levier signal
Contrôles d'un ensemble de signaux provenant d'un levier situé sur le panneau, avec les trois voyants. (Pas encore sorti)
Bouton code
Implémente le bouton de code et une lampe de code. (Pas encore sorti)
