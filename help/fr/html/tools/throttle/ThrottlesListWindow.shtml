Fenêtre Liste des Régulateurs

Cette liste est accessible depuis la barre fenêtre régulateur.

 La liste affiche tous les régulateurs actuellement ouvert en JMRI, si une icône est disponible pour les listes dans leur DB , elle sera automatiquement utilisée.

Cliquer sur une ligne amènera jusqu'au régulateur correspondant.

Nouveau bouton
