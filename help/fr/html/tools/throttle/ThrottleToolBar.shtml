Barre d'outils Régulateur



La barre d'outils régulateur est activable depuis le Panneau des Préférences Régulateurs .

Lorsque elle est activée, une Fenêtre Régulateur peut accueillir un ensemble de Régulateurs, ces Régulateurs peuvent être listés en utilisant le:

bouton Précédent

Déplace vers le Régulateur précédent pour cette Fenêtre de Régulateur.

bouton Suivant

Déplace vers le Régulateur suivant pour cette Fenêtre de Régulateur.

Bouton Nouveau

Ajouter un nouveau Régulateur dans cette Fenêtre de Régulateur.
Une fois que le nombre de Régulateurs dans une Fenêtre Régulateur est supérieure à un, ensuite les boutons Précédent et Corbeille sont activés.

Bouton Corbeille

Retirez le Régulateur actuel de cette Fenêtre de Régulateur.
Lorsque le nombre de Régulateur dans une Fenêtre de Régulateur est égal à un, les boutons Suivant, Précédent et Corbeille sont désactivés

Bouton d'Arrêt Urgence

Envoie un message d'arrêt à tous les Régulateurs gérés par JMRI.

Le bouton d'alimentation

Permet la commande de l'alimentation du réseau si possible (rouge = Arrêt, vert = Marche , le jaune est le statut inconnu).

Bouton Liste

Afficher/Masquer la liste des Régulateurs.
