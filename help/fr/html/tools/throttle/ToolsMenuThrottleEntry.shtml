Fusionner des fenêtre Régulateurs

Les fichiers de configuration Régulateur peuvent consister en une fenêtre de régulateur unique (pour commander une seule locomotive ou une adresse de couplage), ou elle peut contenir des multiples panneau de régulateur. Dans tous les cas, lors de l'ouverture d'un fichier de configuration de régulateur, JMRI vérifie pour voir si des panneaux existants de Régulateurs actifs sont déjà ouverts. Si aucune n'est trouvée, le fichier est ouvert, comme une ouverture de configuration simple (expliqué ci-dessus).

Cependant, quand une ou plusieurs panneau de Régulateur sont déjà ouverts à l'écran, le programme va ouvrir une boîte de dialogue suivante:


Figure 4-5: Merge Dialog Box

avec 3 boutons
  • Fusionner — quitte les panneaux régulateur actifs actuellement ouverts en place et ouvre le régulateur(s) contenu dans le dossier de configuration choisi.
  • Remplacer — ferme le bas du panneau actif du régulateur ouvert (s) et ouvre le panneau de régulateur (s) définit dans le fichier de configuration choisi.
  • Annuler —annule la demande d'"Ouverture" et ferme cette fenêtre de dialogue .

Remarque: Ne confondez pas l'action qui se produit ici avec toute tentative d'écraser vos fichiers sauvegardés de configuration de régulateur. Les actions ci-dessus n'ont aucun effet sur la configuratrion des fichiers sauvegardés de régulateur eux-mêmes. La seule action "fusionner" et "remplacer" qui se produit est tout simplement sur votre écran.
