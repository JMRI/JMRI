Où puis-je trouver les Régulateurs?

Par chance , si vous lisez ce texte , alors vous avez probablement vu des référence aux "Régulateurs" dans les messages du groupe d'utilisateurs JMRI. Et la première question qui vous est sans doute venue à l'esprit fut: "Où diable vais-je trouver ces Régulateurs?"

Eh bien vous avez commencé au bon endroit. La première chose que cette section va faire est de vous montrer comment accéder aux Régulateurs. Cette section vous fera également voir les options de menu qui sont disponibles.

Premièrement, il est important de comprendre que le projet JMRI est un groupe de nombreux"outils" pour vos chemins de fer avec de nombreuses contributions de différents programmeurs. Il y a quatre principales applications de démarrage disponibles

  • DecoderPro (le morceau le plus largement et communément appelé le projet)
  • JMRI démo
  • Outils Loco
  • PanelPro

Pour le sujet de ce guide, l'application de démarrageJMRI démo a été utilisée aux fins de la rédaction de ces pages. Toutefois, vous pourrez constater que de nombreux outils "cross over" ont de multiples applications de démarrage. L' Outil Régulateur est un bon exemple de cela. Peu importe les applications mentionnées qui précèdent avec lesquelles vous commencez, vous serez capable de trouver les régulateurs.



Menu Principal (ou primaire)

Dans chacune des applications de démarrage notée ci-dessus, il y a toujours une fenêtre principale "primaire" qui s'affiche sur votre écran (similaire à celle montrée ci-dessous dans la figure 1-1). Fermez cette fenêtre et toute l'application sera terminée. Cette fenêtre principale contient un ensemble d'options de menu principal. C'est l'endroit où vous trouverez la plupart des outils et des options disponibles y compris les Régulateurs.

  • Cliquez sur"Outils".
  • Cela va ouvrir un sous-menu.
  • Déplacez le pointeur de votre souris jusqu'à ce que vous localisiez ""Régulateurs"".
  • Cela va ouvrir un autre sous-menu.
  • C'est le menu des Régulateurs(voir figure 1-2).


Figure 1-1: structure menu principal JMRI





Le Sous- menu Régulateurs

Le sous-menu régulateurs (comme le montre la figure 1-2 ci-dessous) contient les options suivantes, qui sont définies comme suit:

  • Nouveau Régulateurs... - ouvrir une nouvelle fenêtre régulateur simple avec des paramètres régulateur par défaut est activée.

  • Enregistrer Régulateurs ...-sauve une ou plusieurs fenêtres régulateur actuellement ouvertes dans un fichier utilisateur nommé sur votre disque dur.

  • Charger Régulateurs ...-ouvre un fichier régulateur enregistré précédemment. Ce fichier peut contenir une ou plusieurs fenêtres régulateur selon la façon dont la mise en page a été sauvée .

  • Enregistrer les Préférences Régulateurs ...-Permet de configurer votre ensemble préférences pour toutes les fenêtres régulateur.
