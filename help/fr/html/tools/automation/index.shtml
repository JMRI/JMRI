JMRI Automatisation Réseau

L'application JMRI fournit un grand nombre de façons d'automatiser votre réseeau, allant du simple remplissage de formulaires à des scripts à part entière et des capacités de programmation. Il s'agit notamment de:

Logique Signal
JMRI fournit une logique intégrée de signalisation qui peut être utilisée pour conduire vos tête signal dans de nombreuses applications.
Entrée Sortie (NX)
Le routage NX crée un chemin valide dynamique entre deux emplacements sur un Éditeur de Réseau.
Bulletin d'Ordre
Les Bulletins d'Ordre (Warrants) vous permettent de faire circuler automatiquement des trains sur le réseau avec un Éditeur Panneau de Commandes.
Dispatcheur
Dispatcheur vous permet d'expédier et (automatiquement) de faire circuler des trains sur le réseau avec Éditeur de Réseau.
Itinéraires
Un simple forme d'automatisation est définir des Itinéraires et controler vos aiguillages et signeaux.
LItinéraires
Les LItinéraires sont comme les Itinéraires mais offrent plus de types de contrôle. Lorsqu'un LItinéraire est défini, il crée automatiquement des instructions Logix (voir ci-dessous) qui peuvent être modifiées pour implémenter encore plus de contrôles d'itinéraire.
CTC (Centralized Traffic Control)
Un ensemble d'outils pour aider à l'automatisation des panneaux CTC.
Logix
Utilisant JMRI Logix, vous pouvez spécifier les actions à réaliser quand diverses conditions se produisent sur le réseau.
LogixNG *)
La nouvelle génération de fonctionnalités Logix avec une structure modulaire pour l'extension et la possibilité de créer une logique plus complexe sans script.
Écriture de Scripts
JMRI fournit un outil de script très puissant. Basé sur un véritable langage de programmation, il vous donne accès à toutes les fonctionnalités JMRI.
Écrire votre propre code en Java
Les bibliothèques JMRI simplifient grandement l'écriture d'un programme pour que votre réseau fasse exactement ce que vous voulez.
