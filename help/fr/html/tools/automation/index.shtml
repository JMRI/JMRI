JMRI Automatisation Réseau

L'application JMRI fournit un grand nombre de façons d'automatiser votre réseeau, allant du simple remplissage de formulaires à des scripts à part entière et des capacités de programmation. Il s'agit notamment de:
Logique Signal
JMRI fournit une logique intégrée de signalisation qui peut être utilisée pour conduire vos tête signal dans de nombreuses applications.
Itinéraires
Signalisation
JMRI fournit une logique de signalisation de simple à réaliste qui peut être utilisée pour conduire vos signaux dans de nombreuses applications.
Logix
Utilisant JMRI Logix, vous pouvez spécifier les actions à réaliser quand diverses conditions se produisent sur le réseau.
écriture de Scripts
JMRI fournit un outil de script très puissant. Basé sur un véritable langage de programmation, il vous donne accès à toutes les fonctionnalités JMRI.
écrire votre propre code en Java
Les bibliothèques JMRI simplifient grandement l'écriture d'un programme pour que votre réseau fasse exactement ce que vous voulez.
