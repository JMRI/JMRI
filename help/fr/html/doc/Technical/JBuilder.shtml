JMRI: Construire avec JBuilder

JBuilder est une plate-forme viable pour le développement et le débogage du code JMRI. JBuilder étend le cadre open source Eclipse et peut être téléchargé gratuitement. Cependant beaucoup de développeurs vous recommande d'utiliser NetBeans> , XCode , Eclipse , ou même simplement Ant .

A télécharger une copie d'essai gratuite de JBuilder est disponible à partir de http://www.embarcadero.com/products/jbuilder.

Après le téléchargement et l'activation de votre copie de JBuilder vous pouvez utiliser les directives depuis l'installation Eclipse pour configurerJBuilder
