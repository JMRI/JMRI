JMRI: Visualisation des fichiers XML

JMRI enregistre diverses informations dans des fichiers XML. Bien qu'ils soient en format texte qui est fondamentalement lisible par l'homme, c'est un format compliqué.

Pour les rendre plus faciles à lire, à imprimer et à comprendre, les fichiers XML de JMRI peuvent être affichés dans un navigateur Web. Une technique appelée «feuilles de style XSLT" est utilisé pour les formater dans le navigateur web. Pour un exemple de la façon dont cela ressemble, s'il vous plaît voir le panneau échantillon fichier XML sur les principaux Web site JMRI. (Peut prendre quelques secondes à charger)

Pour afficher un fichier XML que JMRI a créé, il suffit d'ouvrir le fichier avec votre navigateur web favori. Dans de nombreux cas, vous devriez être en mesure de simplement double-cliquez sur sur lui, ou faites le glisser sur l'icône du navigateur Web. Sur certains types d'ordinateurs, vous aurez besoin de l'ouvrir à partir du menu "Fichier".

Malheureusement, cette approche simple ne fonctionne pas toujours, parce que tous les navigateurs ne mettent pas correctement en oeuvre les algorithmes pour trouver les fichiers de mise en forme nécessaires à cet affichage. Si le premier navigateur que vous essayez n'affiche pas le fichier XML (habituellement montrant une page blanche, bien que parfois ils vont afficher un message d'erreur), S'il vous plaît essayez les autres que vous avez installés. Internet Explorer est particulièrement problématique. Mozilla, Safari et Opera semblent pour obtenir ce droit, tandis que Firefox semble fonctionner correctement sur certaines plateformes, Linux entre autres, et pas sur d'autres. (Il peut aussi y avoir des options qui l'affectent s'il fonctionne)

Si aucun des navigateurs qui s'offrent à vous ne peut afficher le fichier, vous pouvez contourner ce problème en utilisant le mini serveur Web JMRI dans la version JMRI 2.9.4 et plus tard.

Pour ce faire:

  • d'abord, démarrer JMRI.
      • Soit DecoderPro ou PanelPro peut être utilisé, et aucune connexion réseau n'est nécessaire.
      • Vous devez utiliser JMRI 2.9.4 ou version ultérieure.
      • Le fichier que vous souhaitez afficher doit avoir été entreposé dans JMRI 2.9.4 ou version ultérieure. S'il est plus vieux que cela, il suffit de charger dans JMIR 2.9.4 ou version ultérieure et de le stocker à nouveau.
    • Démarrez le serveur web JMRI. Dans le cadre du menu débogage , sélectionnez "Démarrer Mini Web Server".
    • Ouvrez votre navigateur Web et saisissez l'URL " http://localhost:12080/index.html "
    • Vous devriez maintenant voir le logo de JMRI sur une page d'index. Si votre fichier XML se trouve dans le répertoire de distribution JMRI, cliquez sur le lien "dist" . Si c'est dans le répertoire des préférences, cliquez sur le lien "prefs" .
    • Enfin, cliquez sur un des noms de répertoires nécessaires pour accéder à votre fichier, puis cliquez sur votre fichier.
    Vous pouvez ensuite utiliser votre navigateur pour visualiser, imprimer, enregistrer, etc comme vous le souhaitez.
