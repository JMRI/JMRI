Installation JMRI: Fichiers de Configuration

Cette section essaye d'expliqueroú sont stockés sur votre ordinateur les fichiers de Configuration ( réglages ) de JMRI et quelleinformation ils contiennent .

Emplacements Fichier de Configuration
Il y a deux endroits oú nous pouvons trouver des Fichiers de Configuration JMRI:
  1. Les versions des fichiers de configuration JMRI se trouvent dans un répertoire "xml" dans le package d'application JMRI et ses sous-répertoires. Notez que tout ce que vous modifiez ici sera perdu lorsque vous mettez á jour JMRI vers une nouvelle version.
    L'application JMRI est installée par défaut dans votre dossier Programme.
  2. Le répertoire des préférences JMRI de l'utilisateur et ses sous-répertoires sont également importants. JMRI cherche ici en premier les fichiers dont il peut avoir besoin.
    Vous devriez mettre ici n'importe quelle version de fichiers modifiée localement afin qu'elles ne soient pas écrasées par une mise á jour de JMRI. Tous les fichiers que le programme écrit pour contenir des informations locales, par Ex: Les entrées Inventaire et les fichiers Panneau, sont également écrits ici.
    L'emplacement par défaut des préférences utilisateur varie selon le système d'exploitation de l'ordinateur. Normalement, il est situé:
    • sur une machine Linux chercher un répertoire .jmri dans dans votre répertoire personnel.
    • Sur un un Mac OS X, les Préférences sont chargées dans un dossier "JMRI" dans le dossier /Users/me/Library/Preferences/.
    • Pour Windows, Cela peut se faire dans un certain nombre d'endroits selon la version de Windows que vous exécutez et si vous avez plusieurs utilisateurs configurés.. Commencez par consulter "Documents" ou "Documents utilisateur" ou "Utilisateurs et documents". Vous recherchez un dossier appelé "JMRI".

    La façon la plus simple de le trouver est de lancer un programme JMRI, p. DecoderPro ou PanelPro et sélectionnez "Emplacements ..." dans le menu "Aide".. Un traitement spécial pour les utilisateurs de Windows: si vous accédez au menu Démarrer de Windows, dans la section JMRI, vous pouvez sélectionner "Préférences" pour ouvrir ce répertoire.

    Vous pouvez définir ou modifier les scripts de localisation et les fichiers utilisateur dans les Emplacement des fichiers Preferences... -> .Notez que le changement de "User Files Location" changera également l'emplacement par défaut pour Panels, Rosters etc. etc.

Types des Fichiers de Configuration
Dans votre dossier JMRI Preferences, vous trouverez un certain nombre de types de fichiers de configuration utilisés par DecoderPro:
  • Définitions de décodeur - Enregistré dans le sous-répertoire decodeurs, un pour chaque décodeur personnalisé; ceux-ci Définissent comment une famille spécifique de décodeurs est configurée..
  • Configurations de locomotive - Enregistré dans le sous-répertoire roster, un pour chaque Locomotive; Ils enregistrent la configuration spécifique de chaque locomotive dans votre Inventaire
  • Définitions de programmateur - Enregistrées dans le sous-répertoire programmer lorsque vous avez utilisé au moins un programmateur; Définissent les formats des différentes fiches de programmation qui sont disponible dans DecoderPro .
  • Le dossier DecoderPro, contenant vos profils.
Fivhiers de Configuration Générales
Il ya beaucoup plus de fichiers de configuration JMRI, certains partagés par les applications, qui sont dans le répertoire des préférences de l'utilisateur:
  • roster.xml - un index de vos locomotives
  • decoderIndex.xml - un index des fichiers de définition de décodeurs disponibles. Il contient également les informations sur quel modèle de décodeur utilise le fichier, comment les identifier et quelles options il met en œuvre.
  • names.xml - Pas utilisé en fonctionnement normal, il s'agit d'une Liste des noms de variables standard á utiliser dans les définitions de programmateur..
