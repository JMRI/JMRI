Installation de JMRI : Fichiers de configuration

Cette section tente d'expliquer oú sont stockés les fichiers de configuration (paramètres) de JMRI sur votre ordinateur et les informations qu'ils contiennent.

Emplacements des fichiers de configuration
Il existe deux endroits oú les fichiers de configuration JMRI peuvent être trouvé :
    Notez que tout ce que vous modifiez ici sera perdu lorsque vous mettrez à jour JMRI avec une nouvelle version. Ce n'est pas le bon endroit pour faire des changements !
  1. Le répertoire des préférences JMRI de l'utilisateur et ses sous-répertoires est également important. JMRI cherche d'abord les fichiers dont il a besoin.
    Vous devriez mettre ici toute version modifiée localement des fichiers afin qu'ils ne soient pas écrasés par une mise à jour de JMRI. Tous les fichiers dans lesquels le programme écrit et qui contiennent des informations locales, par exemple les entrées de l'Inventaire et l es fichiers de paneau, sont également écrit ici.
    L'emplacement des préférences de l'utilisateur par défaut varie selon le système d'exploitation de l'ordinateur. Normalement, il est localisé :
    • Sur une machine Linux, recherchez un .jmrirépertoire dans votre répertoire personnel.
    • Sur macOS, les préférences sont stockées dans un dossier "JMRI" dans le répertoire /Users/me/Library/Preferences/folder.
    • Sur Windows, cela peut être à plusieurs endroits en fonction de la version de Windows que vous utilisez et si vous avez configuré plusieurs utilisateurs. Démarrez en cherchant sous "Documents" ou "Documents utilisateur" ou "Utilisateurs et documents". Vous cherchez un dossier appelé "JMRI".
    Le moyen le plus simple de trouver l'emplacement des préférences est de lancez un programme JMRI, par exemple DecoderPro ou PanelPro, sélectionnez "Emplacements..." du menu "Aide" et cliquez sur [Emplacement Fichiers utilisateur].
    Une particularité pour les utilisateurs de Windows : si vous allez dans le menu de démarrage de Windows, dans la section JMRI, vous pouvez sélectionner "Préférences" pour qu'il ouvre ce répertoire pour vous.
  2. Les fichiers utilisateurs peuvent être stockés sous l'emplacement des préférences, mais JMRI peut également être configuré pour les placer ailleurs.
    Vous pouvez définir ou modifier les scripts d'emplacement et les fichiers utilisateurs dans le fichier Panneau Préférences... -> ; Emplacements des fichiers. Notez que la modification de l'emplacement des fichiers utilisateur modifiera également l'emplacement par défaut des panneaux, des serveurs etc.
Types de fichiers de configuration
Dans votre dossier Préférences JMRI, vous trouverez un certain nombre de types de fichiers de configuration utilisés par DecoderPro :
  • Définitions des décodeurs - stockées dans le sous-répertoire decoders, une pour chaque décodeur personnalisé ; elles définissent comment une famille spécifique de décodeurs est configuré.
  • Configurations des locomotives - Stocké dans le sous-répertoire roster, un pour chaque Locomotive ; celles-ci enregistrent la configuration spécifique de chaque locomotive de votre liste.
  • Définitions du programmeur - stockées dans le sous-répertoire programmers lorsque vous avez utilisé au moins un programmeur ; celles-ci définissent les formats pour les différentes fiches de programmation qui sont disponibles dans DecoderPro.
  • Le dossier DecoderPro, contenant vos profils.
Fichiers de configuration générale
Il existe de nombreux autres fichiers de configuration JMRI, dont certains sont partagés par les applications, qui vivent dans le répertoire des préférences de l'utilisateur :
  • roster.xml - un index de vos Locomotives
  • decoderIndex.xml - un index des fichiers de définition des décodeurs disponibles. Il contient également les informations sur le modèle de décodeur spécifique qui utilise tel ou tel fichier, la manière de les identifier et les options qu'ils mettent en &œuvre.
  • names.xml - non utilisé en fonctionnement normal, c'est une liste des noms de variables standard à utiliser dans les définitions des programmeurs.
