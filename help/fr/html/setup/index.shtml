JMRI® nécessite l'installation de Java:

  • La version 2.12 peut fonctionner sur n'importe quel système informatique qui exécute Java 1.5 à Java 8 (également connu sous le nom de Java 1.8).
  • La version 2.14.1 nécessite Java 1.5 (ou Java 1.6 pour le mode glisser-déposer) via Java 8.
  • La version 3.10.1 nécessite Java 1.7 à Java 8.
  • La version 4.2 nécessite Java 8.

A partir de la version 4.20 de JMRI, JMRI n'est pleinement pris en charge que lorsqu'il fonctionne avec la mise à jour la plus récente possible des deux dernières versions de Java Long Term Support (LTS) (8 ou 11 à partir de juillet 2020) et recevra un support "au mieux" pour la dernière version de Java (14 à partir de juillet 2020). Certains problèmes rencontrés lors de l'utilisation d'une version de Java "best effort" ne peuvent être résolus que par l'utilisation d'une version supportée.

Le projet JMRI recommande Java de AdoptOpenJDK ou le fournisseur du système d'exploitation sauf si vous utilisez Windows 8 ou plus ancien, auquel cas Java de Oracle est recommandé. Voir le guide d'installation spécifique au système d'exploitation pour plus de détails.

Démarrage du programme

Une fois que vous avez téléchargé le logiciel JMRI® et que vous l'avez installé, il vous suffit d'ouvrir le programme de la manière habituelle pour votre système d'exploitation. Windows affichera une icône sur le bureau (avec les options d'installation par défaut sélectionnées).

Double cliquez sur l'icône pour que le programme s'ouvre. Dans ce cas nous ouvrirons DecoderPro®

Vous devriez voir s'afficher cet écran ou un écran d'ouverture similaire après le démarrage du programme:

Si c'est la première fois que vous exécutez le programme après avoir installé le logiciel, l'écran Préférences s'affichera automatiquement pour vous permettre de configurer le système pour votre configuration particulière. De même, si vous changez de système OU si vous avez votre ordinateur portable loin de la chemin de fer, mais si vous voulez jouer avec JMRI, il y a un simulateur vous pouvez essayer ce mode.
