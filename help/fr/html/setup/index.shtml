Installation

Des instructions complètes pour l'installation de JMRI sur diffrentes plateforme peuvent être trouvées ici:

  • Windows
  • Mac OS X
  • Linux
    • RasPi

JMRI® nécessite au préalable l'installation de Java:

  • Version 2.12: Java 1.5
  • Version 2.14.1: Java 1.5 (ou 1.6 si vous voulez glissez et déposer)
  • Version 3.10.1: Java 1.7 ou plus récent.
  • Version 4.2: Java 1.8.
  • Les Versions 4.2 - 4.25.5: Java 8 (pas release 311) ou Java 11 (recommendé)
  • Les Version 4.25.6 - 4.26: Java 8 ou Java 11 (recommendé)
  • Les Versions 4.99 - 5.x: Java 11

Démarrage du programme

Une fois que vous avez téléchargé et installé le logiciel JMRI®, simplement ouvrir le programme de la manière habituelle pour votre système. Windows montrera une icône sur le bureau (avec les options sélectionnées et installées par défaut).

Double clic sur l'icône pour ouvrir le programme. dans ce cas nous ouvrirons DecoderPro®

Ce dialogue apparaitra brièvement pendant le chargement du programme:

Vous pouvez voir celui-ci ou un écran d'ouverture similaire après le démarrage du programme:

Si c'est la première fois que vous lancez le programme après l'avoir installé, l'écran des Préférences s'affichera automatiquement, vous permettant de régler le système pour une configuration particulière. Aussi si vous changez votre configuration OU si vous avez votre ordinateur portable loin du réseau, mais que vous vouliez jouer avec JMRI, il y a une méthode similaire que vous poyvez essayer.
