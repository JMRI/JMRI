Installation JMRI: Configuration des Profils

Profils de Configuration permettent de lancer une seule application JMRI avec l'une des configurations spécifiées par plusieurs utilisateurs. Un Profil de Configuration contient toutes les Préférences telles qu'elles sont définies dans le dialogue des préférences JMRIP, et peut - en option - inclure des fichiers utilisateur, un Inventaire

Par défaut, un seul Profil est utilisé et il n'est pas très visible pour l'utilisateur. Mais il peut être utile d'avoir plusieurs Profils :

  • Par exemple, vous pouvez avoir un Profil dans votre ordinateur portable qui définit toutes vos connexions matérielles, et une autre qui définit les mêmes comme des connexions "simulées". En choisissant l'un ou l'autre au démarrage, vous pouvez à la fois travailler sur le réseau avec des connexions réelles et travailler également hors ligne, sans le réseau matériel connecté.
  • Un autre exemple serait un Profil pour votre propre réseau, et un autre pour l'utilisation de DecoderPro dans votre club local.

Lorsqu'une application JMRI (version 3.9 et plus) est lancée ou la première fois, elle vérifie un Profil de configuration. Si aucun Profil n'existe, l'application créera un Profil par défaut et l'utilisera sans vous le demander. Si un seul fichier est trouvé, l'application utilisera ce Profil sans vous le demander. S'il existe plusieurs Profils, l'application vous demandera de les utiliser. Vous pouvez spécifier que l'application doit toujours commencer par le même Profil.

Si l'utilisateur ne sélectionne pas un nouveau Profil et ne clique pas sur OK, le programme continuera avec le Profil par défaut après un court délai. Vous pouvez cocher une case dans les préférences pour contourner cette étape et passer directement au dernier Profil utilisé.

Gestion des Profils

Les Profils de configuration peuvent être gérés dans les préférences de l'application, sous la configuration du Profil tab.

La ligne marquée "Profil actif actuel" dans la colonne Statut est utilisée par ce lancement de l'application JMRI, et c'est également là que les modifications des préférences seront enregistrées.

Pour utiliser ou configurer un Profil différent, sélectionnez-le, cliquez sur "Activer", sauvegardez et redémarrez l'application.

Pour plus d'informations sur la façon de configurer JMRI pour partager ses fichiers de configuration entre plusieurs ordinateurs, voir le page d'aide sur l'utilisation de Dropbox avec JMRI.

Nous recommandons que les noms des répertoires de Profils se terminent par .jmri afin que d'autres applications puissent utiliser cette extension pour reconnaître un Profil par extension. JMRI ajoute automatiquement .jmri au nom du répertoire pour les nouveaux Profils. Si .jmri est ajouté au nom d'un répertoire de Profils existant, JMRI devrait automatiquement reconnaître et gérer ce changement.

Partage des Profils

Exportation

Les Profils peuvent être exportés à partir de la section "Profil" des préférences de l'application. Un Profil exporté est une archive compressée contenant le contenu du Profil, et contiendra éventuellement les fichiers et la liste des utilisateurs s'ils sont stockés en dehors du Profil.

Importation

JMRI ne prend pas encore en charge directement l'importation de Profils.

Un Profil peut être importé en le décompressant à l'aide de l'utilitaire de gestion des archives dans un nouveau répertoire, puis en l'ouvrant au lancement de JMRI ou à partir des Préférences de l'application en cliquant sur le bouton "Ajouter un Profil existant...".

Notez que l'utilisation de la capacité d'importation/exportation de Profil pour copier les données JMRI dans les deux sens entre deux ordinateurs n'est ni supportée ni recommandée.

Mise à jour des versions antérieures de JMRI

Lorsque vous exécutez une application JMRI pour la première fois après avoir mis à niveau une version antérieure de JMRI, l'application copie la configuration existante dans un nouveau Profil et utilise Profil. Si vous utilisez une application JMRI que vous n'aviez pas utilisée avant la mise à niveau, l'application s e comportera comme décrit ci-dessus.
