JMRI:Fenêtre Préférences

La fenêtre de préférences vous configure le fonctionnement de base de votre application JMRI.

L'écran est divisé en deux sections, sur le côté gauche est une liste des groupes de préférence qui peut être réglé. Cliquer sur chaque préférence va ensuite remplir le côté droit, avec les différentes options qui peuvent être configurées.

Sans aucun doute la commande la plus importante se trouve au bas de cette section: Le bouton "Enregistrer"!

Soyez sûr de cliquer sur "Enregistrer" lorsque vous quittez cette fenêtre. Vos modifications des préférences ne prennent pas effet avant que le programme soit arrêté et redémarré, de sorte que le programme vous invite à le faire après avoir sauvé les préférences.

Cette page est en construction. En attendant, il y a quelques informations supplémentaires sur les préférences dans certaines pages de tutoriels .

Connexions

Vous pouvez sélectionner l'une des diverses connexions JMRI mise ici.

Chaque connexion a son propre onglet unique, pour commencer, choisissez d'abord le fabricant de votre système DCC, suivi par le type de connexion que vous utilisez. Il vous sera alors présenté avec une série de paramètres les plus souvent requis qui doivent être fixés. Si vous devez modifier certains des paramètres les plus avancés, puis connexion, puis cochez la case "Connexion Paramètres supplémentaires" et cela va vous fournir de plus amples détails.

Pour ajouter une connexion supplémentaire cliquez sur l'onglet avec un "+", ce sera alors un onglet connexion raffraichi. Pour supprimer une connexion, dans la liste déroulante des frabricants du système, sélectionnez (aucun sélectionnés) et ceci va enlever la configuration de cette connexion.

Pour plus d'informations sur comment configurer cela pour votre matériel, consultez les pages matériels .

Rappelez-vous de presser Enregistrer!

Démarrage

 Ceci traite de toutes les préférences qui sont invoquées au démarrage, comme le chargement des fichiers de panneau et de l'exécution de scripts.
Actions
Cette section vous permet d'avoir le programme qui exécute automatiquement les sélections de menu différents au démarrage. Par exemple, vous pouvez avoir la Table Aiguillage qui s'ouvre automatiquement, ou ouvrir automatiquement une fenêtre pour la programmation sur la voie principale.

Par exemple, pour ouvrir automatiquement la Table Aiguillage, cliquez sur le bouton "Ajouter une action" dans cette section des préférences avancées. Dans la liste qui est présentée, sélectionnez "Ouvrir Table Aiguillage". N'oubliez pas de sauvegarder vos préférences à jour avec le bouton au bas de l'écran!

Boutons
Cela vous permet d'ajouter des boutons supplémentaires à la fenêtre principale JMRI qui sont connectés à des choix de menu spécifique.

Par exemple, pour ajouter un bouton qui va ouvrir la commande d'alimentation lorsque vous cliquez dessus, cliquez sur le bouton "Ajouter une action" dans cette section des préférences avancées. Dans la liste qui est présentée, sélectionnez "Ouvrir Table Aiguillage". N'oubliez pas de sauvegarder vos préférences à jour avec le bouton au bas de l'écran! Dans la liste qui est présentée, sélectionnez "Ouvrir Commande Alimentation ". N'oubliez pas de sauvegarder vos préférences à jour avec le bouton au bas de l'écran! Le bouton apparaîtra au prochain démarrage du programme

Fichiers
Cette section vous permet de charger automatiquement les fichiers de panneaux lorsque le programme démarre. Les sections précédentes ont déjà exécutées, de sorte que toute action configurée pour cela aura eu lieu. Vous pouvez charger autant de fichiers de panneaux que vous voulez, chacun sera complètement chargé avant que la prochaine soit démarré.

Cliquez sur "Ajouter un fichier", qui va ouvrir une fenêtre de section fichier dans laquelle vous pouvez sélectionner le fichier désiré. Si vous souhaitez plus tard arrêter le chargement du fichier au démarrage, cliquez sur le bouton "Supprimer" à côté de son nom.

scripts
Cette section vous permet d'exécuter automatiquement un fichier de script lorsque le programme démarre. Les sections précédentes ont déjà exécutées, de sorte que toute les actions qui sont configurées pour ça ont eu lieu et tous les fichiers panneaux ont été chargés. Vous pouvez exécuter les fichiers script que vous souhaitez, chacun sera dirigé à la fin du fichier avant le prochain démarrage.

Cliquez sur "Ajouter un fichier", qui va ouvrir une fenêtre de section de fichier dans lequel vous pouvez sélectionner le fichier désiré. Si vous souhaitez plus tard pour arrêter l'exécution d'un script particulier au démarrage, cliquez sur le bouton "Supprimer" à côté de son nom.

Affichage
Interface Graphique Utilisateur (GUI)
Ces boîtes radio vous permettent de sélectionner la façon dont le programme va être afficher. Les choix disponibles dépendent du type d'ordinateur que vous avez, et tous les éléments suivants peuvent être disponibles:
  • Métal - le style par défaut de Java, utilisable sur tous les ordinateurs
  • / Theme/CDE - Une forme Linux/Unix qui fonctionne bien avec les gestionnaires écran x Windows. Il est également disponible sur (presque) tous les types d'ordinateurs.
  • Windows - uniquement disponible sur les systèmes Microsoft Windows, ce qui rend l'apparence du programme, autant que possible comme une application normale de Windows.
  • Mac OS X - disponible uniquement sur MacOS, ce qui rend l'apparence du programme comme une application Macintosh
Locale
Beaucoup de gens ont contribué à l'"Internationalisation" de JMRI. Ceux-ci ont convertit (beaucoup) de texte à l'écran dans une langue autres que l'anglais. Si vous sélectionnez votre pays ou votre région et votre langue ici, puis enregistrez et redémarrez les préférences, JMRI va utiliser une internationalisation pour ce pays et la langue si elle est disponible. Les éléments qui n'ont pas été traduits continueront à paraître en anglais, de sorte que vous ne manquerez rien. pour plus d'informations sur la façon de le mettre en place, et en particulier la façon de contribuer à des traductions nouvelles et les améliorées, s'il vous plaît voir notre Page d'internationalisation.

Liste
Normalement, la liste est stockée dans un emplacement standard dans le cadre du répertoire préférences . Cette option permet de vous déplacer quelque part d'autre

Le bouton "Réinit" restaure l'emplacement par défaut. Notez que cela change juste où le programme regarde. Si vous avez stocké la liste d'informations ailleurs et et voulez la voir revenir à l'emplacement par défaut, vous devez déplacer le fichier vous-mêmee.

Vous pouvez également entrer une valeur par défaut pour le champ "propriétaire" dans les fiches de la Liste DecoderPro ici. Ceci comble la valeur pour les nouvelles entrées, ce qui peut vous faire économiser un peu de frappe. Elle n'a aucun effet sur les entrées existantes, même si elle change.
