JMRI: Ancien Panneau Préférences

Cette page décrit un ancien panneau des Préférences qui a été remplacé par une nouvelle mise en page dans JMRI 2.9.4. L'Information sur la nouvelle version est sur cette page.

Le panneau des Préférences est l'endroit où vous configurez le fonctionnement de base de votre application JMRI.

Préférences Basiques

Lorsque le panneau s'ouvre en premier, il affiche les Préférences de bases.

Connexion Réseau
Vous pouvez sélectionner l'une des diverses connexions réseau JMRI ici. Pour plus d'informations sur comment la configurer pour votre matériel particulier, voir la page d'aide Matériel.
Style Interface Graphique Utilisateur( GUI)
Ces boîtes radio vous permettent de sélectionner la façon dont le programme s'affiche. Les choix disponibles dépendent du type d'ordinateur que vous avez, et tous les éléments suivants peuvent être disponibles:
  • Métal - le style par défaut de Java, utilisable sur tous les ordinateurs
  • CDE/Motif - Une forme Linux/Unix qui fonctionne bien avec les gestionnaires écran X Windows. Il est également disponible sur (presque) tous les types d'ordinateurs.
  • Windows - uniquement disponible sur les systèmes Microsoft Windows, ce qui rend le programme aussi ressemblant que possible qu'une application normale de Windows.
  • Mac OS X - uniquement disponible sur MacOS, ce qui rend l'apparence du programme ressemblant à une application Macintosh
Programmateur par Défauts
Le programmateur DecoderPro vous permet de choisir l'affichage de beaucoup ou peu de contenu du décodeur en utilisant un "programmateur" configurable . Vous pouvez en sélectionner un dans le même temps, vous sélectionnez votre décodeur de locomotive , mais cela fournit un endroit commode pour définir un décodeur par défaut ainsi vous n'avez pas à le sélectionner à chaque fois que vous ouvrez un décodeur.
Sans aucun doute la commande la plus importante se trouve au bas de cette section: Le bouton "Enregistrer"!

Soyez sûr de cliquer sur "Enregistrer" lorsque vous quittez cette fenêtre. Vos modifications des préférences ne prendront effet qu'après redémarrage du programme, de sorte que le programme vous invite à le faire après avoir sauvé les préférences.

Préférences Avancées

La case "Montrer Préférences Avancées" vous permet de personnaliser votre connexion au réseau. Vous n'avez pas besoin d'utiliser les préférences avancées pour obtenir votre première connexion au réseau et et exécuter l'application, et nous vous recommandons de les laisser seuls jusqu'à ce que la connexion à votre centrale soit opérationnelle. Pour paramétrer vos préférences avancées, cochez la boite "Montrer Préférences Avancées".

Connexions auxilliaires au réseau
Les connexions auxiliaires vous permettent d'avoir plus d'une connexion réseau active à un moment. La plupart des utilisateurs n'ont besoin que d'une connexion, donc sélectionner une connexion auxiliaire seulement si vous avez besoin de plus d'un raccordement au réseau. Utilisez la "Connexion Réseau:" en haut du menu Préférences de votre connexion principale.
Programmateur par défaut
Cette section a une case à cocher qui vous permet de décider si le programmateur DecoderPro va afficher des volets vide (onglets). Certaines personnes aiment les voir, même si elles sont vides, d'autres préfèrent ne pas les voir. Nous fournissons cette préférence afin que vous puissiez choisir.
Affichage
Beaucoup de gens ont contribué à l' "Internationalisation" de JMRI. Nous les remercions d'avoir convertit (beaucoup) de texte à l'écran pour une autre langue que l'anglais. Si vous sélectionnez votre pays ou votre région et votre langue ici, puis enregistrez et redémarrez les préférences, JMRI va utiliser une internationalisation pour ce pays et la langue si elle est disponible. Les éléments qui n'ont pas été traduits continueront à apparaître en anglais, de sorte que vous ne manquerez rien. Pour plus d'informations sur la façon de le mettre en place, et en particulier la façon de contribuer ventaire; de nouvelles traductions et les améliorées, s'il vous plaît voir notre Page d'internationalisation .
Actions à faire au démarrage
Cette section vous permet d'avoir le programme qui exécute automatiquement les sélections de menu différents au démarrage. Par exemple, vous pouvez avoir la Table Aiguillage ouvrir automatiquement, ou d'ouvrir automatiquement une fenêtre pour la programmation principal.

Par exemple, pour ouvrir automatiquement la Table Aiguillage, cliquez sur le bouton "Ajouter une action" dans cette section des préférences avancées. Dans la liste qui est présentée, sélectionnez "Ouvrir Table Aiguillage ". N'oubliez pas de sauvegarder vos préférences avec le bouton Enregistrer au bas de l'écran!

Créez des boutons sur l'écran principal
Cela vous permet d'ajouter des boutons supplémentaires à la fenêtre principale JMRI qui sont connectés à des choix de menu spécifique.

Par exemple, pour ajouter un bouton qui va ouvrir la commande d'alimentation lorsque vous cliquez dessus, cliquez sur le bouton "Ajouter une action" dans cette section des préférences avancées. Dans la liste qui est présentée, sélectionnez "Ouvrir Tableau Aiguillage ". N'oubliez pas de mettre à jour vos préférences avec le bouton au bas de l'écran! Dans la liste qui est présentée, sélectionnez "Ouvrir Commande Alimentation ". N'oubliez pas de mettre à jours vos préférences avec le bouton au bas de l'écran! Le bouton apparaîtra au prochain démarrage du programme.

Chargement des fichiers de panneau au démarrage
Cette section vous permet de charger automatiquement les fichiers de panneaux lorsque le programme démarre. Les sections précédentes ont déjà été exécutées, de sorte que toute action configurée aura eu lieu . Vous pouvez charger autant de fichiers de panneau que vous le souhaitez, chacun sera complètement chargé avant le prochain démarrage.

Cliquez sur "Ajouter un fichier", qui va ouvrir une fenêtre de la section du fichier dans lequel vous pouvez sélectionner le fichier désiré. Si vous souhaitez plus tard arrêter le chargement du fichier au démarrage, cliquez sur le bouton "Supprimer" à côté de son nom.

Exécuter des scripts au démarrage
Cette section vous permet d'exécuter automatiquement un fichier de script lorsque le programme démarre. Les sections précédentes sont déjà exécutées, de sorte que toutes les actions configurées auront eu lieu et tous les fichiers panneaux ont été chargés. Vous pouvez exécuter les fichiers script que vous souhaitez, chacun sera éxecuté à la fin du fichier avant le prochain démarrage

Cliquez sur "Ajouter un fichier", qui va ouvrir une fenêtre de la section du fichier dans lequel vous pouvez sélectionner le fichier désiré. Si vous souhaitez plus tard arrêter l'exécution d'un script particulier au démarrage, cliquez sur le bouton "Supprimer" à côté de nom de fichier .

Définissez l'emplacement de l'Inventaire
Normalement, l'inventaire est stockée dans un emplacement standard dans le cadre du répertoire preferences . Cette option vous permet de le déplacer autre part.

Pour le configurer, cliquez sur le bouton "Paramétrer ..." et suivez les instructions.

Le bouton "Réinit" restaure l'emplacement par défaut. Notez que cela modifie juste où le programme regarde. Si vous avez stocké des informations I ailleurs et voulez le faire revenir à l'emplacement par défaut, vous devez déplacer les fichiers vous-même.

Vous pouvez également entrer une valeur par défaut pour le champ "propriétaire" dans les entrées de registre DecoderPro ici. Ceci remplit avec la valeur les nouvelles entrées, ce qui peut vous faire économiser un peu de frappe. Il n'a aucun effet sur les entrées existantes, même s'il a changé.
