L'éditeur Panneau de Commande

éditeur Panneau de Commande est tout simplement une vue alternative et un contrôleur avec exactement le même contenu de données que dans l'éditeur Panneau. Vous pouvez modifier l'affichage et les commandes de votre panneau à chaque fois et aussi souvent que vous voulez. Le Stockage et le chargement peut être fait de l'un ou l'autre éditeur. L'éditeur utilisé pour stocker le panneau sera l'éditeur utilisé au prochain chargement.

Modification de la Vue entre les éditeurs

Dans l'un ou l'autre des éditeur en mode édition le dernier élément dans le menu Fichier est Modifier la vue pour "...". ("..." étant l'autre éditeur). Sélectionner cette option de commutation de la vue d'un mode éditeur pour un autre mode éditeur.

Différences dans la Vue entre les éditeurs

L'éditeur de Panneau a une fenêtre séparée pour ses fonctions d'édition. Pas l'éditeur Panneau de Commande . Dans l'éditeur de Panneau le menu: Modifier ouvre une fenre d'édition. dans l'éditeur de Panneau de Commande, le menu Modifier ajoute trois menus additionnels au menus du panneau : Fichier, Options, Zoom, Add Items...

Menus éditeur de Panneau de Commande

Menu Fichier

Le menu Fichier change ces éléments entre le mode d'édition et le mode de fonctionnement. Dans le mode d'édition les articles sont les mêmes que les éléments dans le menu Fichier de la fenêtre d'édition de l'éditeur de Panneaux avec quelques articles supplémentaires. Ils sont les suivants:
  • Nouveau Panneau.
  • Panneaux Stockés....
  • Stockage Index Images.
  • Renommer Panneau.
  • Créer/Modifier Index des images.
  • Changement Vue de l'éditeur de Panneau.
  • Changer vue du Générateur de Circuit.
  • Supprimer Ce panneau....
  • Fermer l'éditeur.

En mode opérationnel le menu Fichier de l'éditeur de Panneau de Commande, le menu Fichier n'a qu'un seul élément, Ouvrir éditeur

Menu édition

Ce menu soutient Couper, Copier et Coller avec le système presse-papier. Couper et Copier placera une copie d'un article de la sélection ou un groupe dans le presse-papiers. Cette fonctionnalité permet des copies de sélections d'un panneau à un autre panneau.

Il y a également un élément de menu pour sélectionner tous les éléments du panneau.

Options du Menu

Les éléments de ce menu sont similaires aux cases à cocher dans la fenêtre d'édition de l'éditeur de Panneau avec quelques éléments et caractéristiques supplémentaires. Comme avec l'éditeur de Panneau, l'éditeur de Panneau de Commande peut définir globalement le repositionnement etcontrôler les attributs. Cependant CPE permettra de préserver le repositionnement et le commande individuel des attributs d'éléments lors de la substitution globale est retirée.

Les éléments Remplacement de la position individuelle et Commande des réglages active ou désactive, les deux éléments de menu suivants: (Tous les éléments du panneau peuvent être repositionnés et Tous les éléments du panneau commande le réseau). Lorsque le Remplacement de la position individuelle et Commande des réglages n'est pas cochée, le verrouillage ou déverrouillage de la position des icônes se fait comme dans l'éditeur de Panneau - comme est activée ou désactivée la commande d'un élément du réseau. Lorsque le Remplacement de la position individuelle et Commande des réglages est cochée, les deux prochains articles de menu fonctionnent comme ils le font dans l'éditeur de Panneau. Toutefois, le verrouillage individuel et les paramètres de commande pour chaque élément sont mémorisés.

Décocher l'élément Remplacement de la position individuelle et Commande des réglages rétablira les paramètres des élément individuel à leurs paramètres précédents. Dans l'éditeur de Panneau, cochez Remplacement de la position individuelle et Commande des réglages et Tous les éléments de commande du réseau fixeront définitivement tous les éléments du panneau.

Les définition des préférences de la barre de défilement se trouvent également dans ce menu.

Menu Zoom

L'élément zoom (pourcentage )... va agrandir ou réduire la taille du panneau.

l'élément Zoom pleine Page va agrandir ou réduire l'affichage de tous les éléments du panneau dans le panneau plein écran dans la fenêtre sans l'utilisation de barres de défilement. Cela peut être utile si votre panneau de conception se trouve être légèrement plus grand que ce que vous pouvez voir à l'écran. Plutôt que d' utiliser les barres de défilement pour voir tous les éléments sur le panneau, "Zoom Plein écran" permettra de réduire pour adapter l'affichage à la fenêtre du panneau. Inversement, il élargira l'affichage pour s'adapter aux fenêtre du panneau. Si vous redimensionnez la fenêtre du panneau, alors vous devriez répéter "Zoom Plein écran" pour redimensionner l'écran pour s'adapter aux nouvelles dimensions de la fenêtre du panneau.

Menu Ajt Articles

Le menu Article Palette ouvre une fenêtre à onglets qui vous permet d'ajouter des éléments au panneau. Chaque type d'élément a un onglet dans la fenêtre. L'ajout d'éléments se fait par glisser-déposer. Des éléments iconiques peuvent avoir plusieurs ensembles d'icônes à choisir. Les jeux d'icônes peuvent être personnalisés et ajoutés ou supprimés en fonction de l'utilisateur. Les détails peuvent être trouvés au La Palette élément. Notez que l'article de la palette peut également être ouvert avec la touche de raccourci, ctrl P.

La Palette d&ns léditeur de Panneau de Commande ne supporte les icônes aiguillages TJ/3-Voies/Croisement qui sont disponible depuis l'éditeur de Panneau.

L'élément menu Tableau Liste des éléments ouvre la liste des tableaux appareil. L'élément du menu peut également être appelé avec la touche de raccourci, Ctrl T.

Autorisations Restrictives

Les éléments dans ce menu sont les fichiers d'aide pour définir l'Occupation Cantons et en les utilisant pour définir les Autorisations Restrictives. Lorsque deux ou plusieurs Occupation de Canton sont définies des éléments de menu additionnels pour définir les Autorisations Restrictives.

Constructeur de Circuit

Les éléments dans ce menu fournissent une interface utilisateur graphique (GUI) pour définir l'Occupation Cantons, Portails et Chemins. Chaque élément possède le fichier d'aide pour cette construction des circuits de voie.

Utilisation de la souris dans l'éditeur de Panneau de Commande

L'éditeur Panneau de Commande n'implémente pas la mise en évidence de renversement trouvé dans l'éditeur de Panneau.

Sélection

Dans l'éditeur de Panneau de Commandela sélection est faite avec un simple clic. Ainsi la sélection peut être glissé d'un simple clic enfoncé.

Lorsque des éléments sont empilés, Maj-clic permet de sélectionner l'élément le deuxième plus en avant dans la couche. Control-clic sélectionne l'article le plus en bas de la pile. Si les articles sont empilés sur un élément de fond, Control-clic sélectionne le point le plus en bas au-dessus du fond. Contrôle-cliquez sur seulement le fond vous sélectionnez l'icône de fond.

Sélection multiple

Comme avec l'éditeur de Panneau, une sélection de groupe peut être faite en utilisant la "zone de la sélection"- un rectangle en pointillés rouges va sélectionner tous les articles au sein de la sélection. Si la touche Maj est maintenue enfoncée tout en créant une zone de sélection, alors tous les éléments qui croisent le rectangle en pointillés rouges sont sélectionnés. Cela peut être une fonction très utile lorsque les éléments sont situés à la frontière en haut et à gauche du panneau.

Une autre caractéristique commune est l'ajout ou la suppression d'éléments dans un groupe de sélection par l'appui sur la touche Ctrl quand vous sélectionnez l'article.

Glisser la Sélections

Glisser une sélection ou un groupe de sélection peut être faite soit avec bouton de la souris dans l'éditeur de Panneau de Commande. Une icône de fond

(c'est à dire une icône au niveau 1) peut être déplacée en la faisant glisser avec la touche Ctrl enfoncée.

Duplication Sélections Dans éditeur de Panneau de Commande

La copie d'un élément ou un groupe sélectionné est faite par l'utilisation du menu contextuel pour l'un des éléments sélectionnés. Lorsque l'élémentCopier du menu contextuel est cliqué, une copie du groupe sélectionné se fait sur le dessus de la sélection originale. Faites glisser cette copie à l'endroit où vous souhaitez placer la copie. La sélection initiale est maintenant désactivée et reste à sa position initiale. La procédure de copie peut être répétée sur la nouvelle position copie pour continuer à faire plus de copies.

Menus contextuel d'Article dans l'éditeur de Panneau de Commande

Faites un clic droit (méta-cliquez sur sur Mac ou Linux) fait apparaître le menu contextuel de l'élément.

Attributs du Texte Edité

Les Attributs du Texte Edité des éléments contextuel ouvre une boîte de dialogue pour définir l'un des attributs de texte de l'élément.

Modification Sélection Multiple

Lorsqu'un élément de menu contextuel est invoqué à partir d'un élément qui est un membre d'un groupe sélectionné , toutes les valeurs modifiées qui ont été faites dans le menu contextuel sont appliquées à tous les membres du groupe sélectionné si cette valeur est pertinente pour les membres. Par exemple, si la taille de la police est modifiée sur l'un des éléments du groupe sélectionné, ensuite la taille de police est modifiée dans tous les éléments du groupe. Si par exemple, une horloge rapide a été incluse dans ce groupe, alors aucune modification ne serait apportée à l'horloge.
