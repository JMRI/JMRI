L'Éditeurd'Icône Multicapteurs

La façon dont les icônes sont modifiées dans l'Édteur d'Icône Multicapteurs est la même que celles qui les modifie dans les autres Éditeurs d'Icône. Le bouton Changer l'Icône... et le Catalogue de fonction sont identiques avec les autre Éditeurs d'Icône. Le Menu Trouver des Icônes et ses éléments sont également identiques.

Voir la pages Éditeurs d'Icône et l'image Page Éditeur d'index pour plus de détails sur leur utilisation.

Contenu de l'Éditeur d'Icône Multicapteurs

Les icônes de l'Éditeur d'Icône Multicapteurs ne diffèrent que par l'existence de plusieurs icônes "actives". Chacune correspond à un commutateur ou à la position d'un levier. Dans le Panneau, les positions s'enchaînent dans l'ordre. Une position peut être supprimée en appuyant sur la touche Supprimer sous l'icône. D'autres positions peuvent être ajoutées en appuyant sur le bouton Ajouter une autre icône.

Chaque icône active doit avoir un capteur qui lui est associé. Pour attribuer un capteur à une position de l'icône, faites glisser une ligne du Tableau Capteur à l'endroit dans l'onglet icône à un emplacement intitulé "indéfini". Lorsque la ligne du capteur est tombé sur l'étiquette, il affiche le nom du capteur.

Lorsque toutes les icônes actives sont affectées" aux capteurs, le bouton Ajouter au Panneau devient actif et il peut être appuyé pour installer le multicapteurs dans le panneau.
