Réglage Largeur Voie

La boîte de dialogue Réglage Largeur de Voie permet de changer les largeurs des deux types de voies utilisés dans les schémas de voies dans le TCO. La voie peut être désignée comme voie principale ou voie d'évitement lors de la création ou de l'édition de Segments de Voie. Les parties de type de voie d'aiguillage et de croisement sont déduits à partir du type des Tronçons de Voie connectés. Toutes les voies sont initialement voie d'évitement par défaut. La spécification du type voie principale pour des segments spécifiques de la voie est une option que vous pouvez ou non choisir d'en profiter.

Chaque type de voie peut être dessiné en utilisant une largeur de voie différente. La modification de la largeur d'un type de voie modifie les largeurs de toutes les voies de ce type dans le schéma de voie.
