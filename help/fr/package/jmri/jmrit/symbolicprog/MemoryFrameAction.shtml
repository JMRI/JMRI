JMRI: Outil Moniteur de Mémoire

L'Outil Moniteur de Mémoire vous permet de voir combien de mémoire JMRI utilise dans votre ordinateur.

Il ya trois colonnes à l'écran en haut:

  • "Mise à Jour" - : met une nouvelle série de valeurs dans l'affichage
  • "Recueillir la mémoire" - réduire la quantité de mémoire utilisée au minimum possible en demandant à Java d'exécuter un nettoyage de cMemory Monitor Toolollection
  • "Test" - Exécuter un test de l'allocation de mémoire et processus de collecte (pas nécessaire pour les utilisateurs réguliers)

La partie supérieure affiche des informations sur la mémoire. Chaque fois que vous appuyez sur le bouton "Mise à Jour", la ligne de fond obtient de nouvelles valeurs et les valeurs anciennes sont déplacés vers le haut.

Memory Monitor Tool

  • La colonne de gauche est la quantité de mémoire actuellement en usage.
  • La colonne du milieu montre la quantité de mémoire actuellement affectées à JMRI, mais que vous n'utilisez pas. Celle-ci est libre pour une utilisation ultérieure.
  • La colonne de droite est la quantité totale de mémoire disponible pour JMRI.

Au sommet de la colonne de gauche, la mémoire maximum qui peut être disponible pour JMRI est affichée.The Memory Monitor Tool lets you see how much memory JMRI is using in your computer.
