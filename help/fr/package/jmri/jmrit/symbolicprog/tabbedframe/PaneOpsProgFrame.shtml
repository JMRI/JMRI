onglet Liste

Une liste est une base de données de toutes les locomotives que votre installation de Decoder Pro a programmées.Elle inclut les informations vu dans l'écran ci-dessous:

Le premier champ "ID" devient le nom de fichier dans la Liste pour la locomotive que vous programmez.

Le champ Adresse est l'adresse DCC active pour cet loco, voir le Volet Basique .

La Vitesse Limite Régulateur définit le pourcentage du régulateur Maximum autorisé.

Vous pouvez mettre tout ce que vous voulez dans les autres domaines.

Le bouton Enregistrer stocke les informations actuelles du décodeur dans votre dossier Liste. Le stockage par défaut est dans le même répertoire que celui des préférences.

Le bouton "Réinitialiser par défaut" pas un "reset usine", retourne le CV décodeur à la valeur par défaut qui a été programmée dans le fichier xml décodeur. Elle ne modifie PAS les valeurs dans le fichier liste sur votre disque dur, à moins que vous y Enregistriez les valeurs qui ne sont pas écrites dans le décodeur jusqu'à ce que vous sélectionniez de les écrire.

La plupart des décodeurs vont activer le menu Réinit (à côté du menu Fichier en haut de la page). Il s'agit de la "Réinit d'Usine", certaines construCeci est la page d'aide help/en/package/jmri/jmrit/cteur ont plusieurs routines de réinitialisation. Elle le fait en écrivant directement dans un CV dans le décodeur, CV8 dans le cas de Digitrax. Tous les décodeurs peuvent ne pas supporter la Réinit d'Usine et comment ils fonctionnent dépendent du fabricant et du décodeur.

Onglet Basique

Le volet Basique est l'endroit où vous définissez l'adresse du Décodeur, vous pouvez définir une adresse courte et longue , puis sélectionnez celui qui est Actif.

Les éléments qui s'affichent peuvent être différents pour chaque décodeur. Cela dépend de la façon dont le fichier de définition de décodeur a été écrit et quelles sont les options de fabrication incluses dans le décodeur.

Vous pouvez sélectionner la Direction de la Locomotive. Si vous voulez que la loco aille dans l'autre direction sélectionner Inverse.

Les Crans de Vitesse de la Loco peuvent être sélectionnés, généralement 28 crans.

Vous pouvez sélectionner ralentir si la locomotive répond à la tension DV ou non.

Vous pouvez entrer n'importe quel numéro que vous voulez dans le "ID Utilisateur"

L'ID Fabricant et l'ID de version Fabricant sont définies dans le décodeur par le fabricant. JMRI utilise ceux-ci pour identifier le décodeur.

Autres Onglets

Les autres onglets dépendent de ce que le fabricant a installé dans le décodeur, si les options n'ont pas été installées les volets sont soit grisés ou pas activés.

Onglet Table de Vitesse

Ce panneau peut être différent selon la façon dont le décodeur xml a été construit. Pour utiliser les CVs de la table de vitesse habituellement l'option de vitesse basique doit être éteinte. Les petites cases sous chaque curseur sont très utiles. Pour voir comment ils travaillent cocher une dans le milieu de la table, puis faites glisser le dernier curseur vers le bas . Notez que les valeurs ne changent pas dessous celle que vous avez cochée. Ils ne changent pas pour les crans au-dessus, si un cran inférieur est modifié.
