Groupes Capteur

La fenêtre de l'outil Groupe de Capteur vous permet de définir des groupes de capteur, un groupe de capteurs dont plus d'un ne peut pas être actif. Quand un membre du groupe devient actif, tous les autres sont mis à l'état inactif.

Généralement, vous pouvez utiliser quelque chose comme cela pour simuler un commutateur rotatif sur un panneau de contrôle réel. Le commutateur réel serait d'activer une seule entrée quand il a été tourné de position en position, l'entrée précédente deviendra inactive quand l'interrupteur sera tourné vers la position suivante. Sur un panneau de commande PanelPro, chaque position peut être représentée par une icône de capteur séparée sur laquelle vous cliquez pour l'activer. Mettre ce jeu de capteurs dans un groupe de capteurs ferait alors devenir le capteur de la position précédente inactif lorsque vous avez cliqué pour modifier le commutateur rotatif simulé.

Pour définir un groupe:

  • Cochez la case "inclus" à côté de chaque capteur que vous voulez inclure dans le groupe. Vous pouvez cliquer sur les titres au sommet de la colonnes pour les trier si vous avez beaucoup de capteurs à parcourir.
  • Mettez un nom de capteur unique au groupe dans la case en bas. Pas de format particulièrement requis, mais il ne peut pas reproduire celui que vous avez déjà défini.
  • Cliquez sur ajouter/mise à jour

    Pour voir un groupe de capteurs que vous avez déjà défini, mettre son nom dans la Zone Nom du groupe, et cliquez sur Afficher. Vous pouvez ensuite modifier les cases pour ajouter ou supprimer des capteurs du groupe, et cliquez sur Ajouter/Mettre à jour pour faire vos modifications.

    N'oubliez pas de sauvegarder vos modifications d'une nouvelle configuration ou d'un fichier du panneau avant de quitter le programme!
