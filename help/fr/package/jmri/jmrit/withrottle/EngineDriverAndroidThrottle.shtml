JMRI: Pilote de Régulateur de Machine pour Android

Cette application gratuite peut se connecter à un serveur RégulateurWi JMRI s'exécutant sur un ordinateur, et prend tour à tour le contrôle de vos locomotives. Vitesse, direction, et jusqu'à 29 fonctions DCC sont soutenues pour une ou deux locomotives. Vous pouvez également contrôler l'alimentation du réseau, les aiguillages et les itinéraires.

Pour les captures d'écran, l'installation et l'utilisation en détails, visitez [page d'accueil pour EngineDriver].

Prérequis :

Vérifiez que votre appareil Android peut se connecter à votre réseau PC via le WiFi et obtenir une adresse IP.
Assurez-vous que vous utilisez l'une des dernières versions de JMRI (2.8 fonctionne, 2.10 fonctionne encore mieux) sur votre PC, relié au réseau. Démarrer JMRI et vérifiez que vous pouvez utiliser l'application comprenant "Throttle" pour contrôler une locomotive.

Opération:

Démarrez la Régulateur Wi fonction de JMRI, située dans DecoderPro sous Outils,Régulateur. (Ou sous Débogage dans les anciennes versions de JMRI).

Sur votre téléphone, lancez l'application Pilote Machine et attendez que l'adresse de votre PC se montre dans Â«Services Découverts".
Cliquez sur l'adresse découverte, (ou vous pouvez entrer l'adresse et le port de votre serveur WiThrottle directement).

help page

Sur l'écran du régulateur, cliquez sur le bouton loco "Appuyez sur pour sélectionner" pour chaque loco
Sur l'écran suivant, entrez l'adresse loco (à vérifier si courte ou longue), et appuyez sur "Acquérir". Vous serez renvoyé à l'écran du Régulateur.
Répétez en utilisant le 2ème bouton "Appuyez sur pour sélectionner"pour acquérir un 2ème loco.

help page

Faite fonctionner votre loco en utilisant le curseur de vitesse, et les boutons inverse, arrêt et en avant boutons pour contrôler la direction.
Pour libérer toutes les locomotives, il suffit d'appuyer sur le bouton Retour. Ce sera également arrêtera votre loco (peut être substituée dans les préférences).

Appuyez sur le bouton Menu de votre téléphone pour accéder aux écrans de contrôle pour les Aiguillages, les Itinéraires et l'Alimentation, ainsi que régler les paramètres de configuration par défaut et les étiquettes de fonction. </ p>

Configuration:

Sous Préférences, définir le nom que vous voulez donner à votre régulateur. (Doit être unique si de multiples EngineDrivers sont en usage) Vous pouvez désactiver la fonction "stop à la libération" si désiré, et vous pouvez définir une valeur Maximale pour les curseurs d'accélérateur. Si vous prévoyez d'utiliser le contrôle d'Aiguillage, assurez-vous de sélectionner le système matériel correct dans la liste.

Utilisez l'option de menu Boutons de Fonction pour régler les étiquettes et les numéros de fonction à afficher. Si vous avez mis des étiquettes dans une fiche dans JMRI, ces étiquettes peuvent être copiées à partir du dialogue (accès au menu après la sélection d'une locomotive). Enlever les boutons en occultant les étiquettes.

Sur le côté informatique de JMRI, vérifiez vos préférences pour vous assurer que vous autorisez ou interdisez WiThrottle à contrôler les caractéristiques souhaitées. Si vous avez défini vos aiguillages et les itinéraires dans un panneau, assurez-vous de spécifier "les noms d'utilisateur" pour ceux que vous aimeriez voir inscrits sur votre régulateur. Vous pouvez aussi "filtrer" les aiguillages montrés en utilisant les WiThrottle-> option Filtre de Contrôles dans la fenêtre WiThrottle. Mettre On "utilisation ESTOP " et "Utilisation de port fixe #" sont également recommandés.

Saisie des fiches, itinéraires et des aiguillages défini dans JMRI va rendre votre pilote de régulateurr de machine facile à utiliser ( m help page ais pas nécessaire).
