Programmateur Simple CV

Le programmeur simple CV vous permet de facilement lire et écrire Les valeurs de CV dans les décodeurs DCC.

Au sommet se trouvent deux boutons pour "Lire CV" et "Ecrire CV".

Avant de les utilisez, vous devrez mettre le numéro CV (adresse), et si vous voulez faire une écriture, la nouvelle valeur valeur.

Sur la gauche vous pouvez choisir le mode de programmation. Les modes que votre centrale de commande ne peut pas fournir sont grisés et ne peuvent pas être sélectionnés.

Si vous sélectionnez un mode "ops", qui peut pas programmer sur la voie principale, vous devez entrer l'adresse du décodeur à programmer Selon votre système DCC, vous pourriez avoir besoin de cocher la case "Adresse Longue" s'il s'agit d'une adresse longue, pour certains systèmes DCC, JMRI peut comprendre cela à partir de l'adresse. Certains systèmes DCC ne vous permettent pas de lire sur la voie principale, dans ce cas le bouton "Lire" sera désactivé.

Á la droite de ceci est un endroit où vous pouvez sélectionner le format des valeurs d'entrées soit en valeur Décimale (base 10) ou hexadécimale (base 16).
