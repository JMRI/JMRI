Ajout/Modification de Logix

Créer un nouveau Logix

Pour créer un nouveau Logix, cliquez sur Ajouter .... La fenêtre Ajouter un logix s'affiche.

Le nom système est requis. Le nom système d'un Logix doit toujours commencer par les deux lettres IX, généralement suivies d'un numéro sélectionné par l'utilisateur. Par exemple, les noms système Logix valides incluent: IX1, IX34 et IX100. Toutes les lettres d'un nom système doivent être en majuscules. Si l'utilisateur entre des lettres minuscules, JMRI les convertira automatiquement en majuscules. De plus, pour plus de commodité, si le nom système entré ne commence pas par IX, le programme ajoute IX avant ce qui est entré. Par exemple, pour entrer un nom de système IX32, entrez simplement 32 dans le champ du nom système et le programme se chargera du reste.

Le nom utilisateur est facultatif mais recommandé. Un nom utilisateur peut être n'importe quel texte utile, à condition que le nouveau nom d'utilisateur n'ait pas été précédemment attribué à un autre Logix. Une fois Logix créé, son nom système ne peut plus être modifié. Son nom utilisateur peut être modifié dans la fenêtre Tableau Logix ou Éditer Logix. Le nom utilisateur est généralement utilisé pour identifier le Logix particulier, par exemple "Contrôle 5 du signal".

Après avoir cliqué sur Créer un Nouveau, Logix sera créé et la fenêtre Modifier Logix apparaîtra.

Modifier un Logix existant

Cliquez sur le menu Sélectionner situé à droite du Logix souhaité. Le menu résultant contient un certain nombre d'options.

Sélectionnez Modifier pour ouvrir la fenêtre Modifier Logix de l'Éditeur de Condition approprié en fonction de la sélection de l'éditeur de menu Options.

Pour plus d'informations sur l'éditeur, voir la page Editeur de liste Condition ou la page Éditeur d'Arborescence de Condition.

Parcourir

En sélectionnant l'élément Parcourir dans le menu déroulant du Tableau Logix, une fenêtre apparaît avec une liste indiquant les détails de chaque condition. C'est un moyen efficace de revoir tout le contenu d'un Logix. Cliquez sur le bouton de fermeture pour fermer la fenêtre. depuis la 4.7.3

Copier

Sélectionnez Copier pour dupliquer un Logix existant. La fenêtre Copier Logix s'affiche.

Entrez le nom système et le nom utilisateur, puis cliquez sur Copier. Une zone de saisie sera affichée pour chaque condition dans le Logix dâ€™origine.

Modifiez le nom si vous le souhaitez, puis cliquez sur OK. Si cette condition n'est pas nécessaire, cliquez sur Annuler.

Supprimer

Sélectionnez Supprimer pour supprimer le Logix sélectionné et tous ses conditions. Une boîte de dialogue de confirmation sera affichée.

Pour plus d'informations, consultez la main Logix documentation .
