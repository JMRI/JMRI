Mât Signal Répéteur

L'outil Mât Signal Répéteur vous permet de configurer un mât signal pour qu'il suive l'état d'un autre, ou pour les deux Mâts Signaux mettre à jours l'état ​​des autres.
Pour que cela fonctionne chaque MâtSignal doit être du même système de signal et Type de mât.
Quand un Mât principal est choisi dans la liste au bas de l'écran la liste d'esclaves est remplie avec les MâtsSignaux correspondants du même type. S'il n'y a pas Mâts valides alors la possibilité de sélectionner l'esclave restera désactivée.

Dans le tableau principal, il est possible de choisir la direction dans laquelle les mises à jour auront lieu, soit maître à l'esclave, l'esclave au maître ou Bidirectionnel. Vous pouvez également activer/désactiver le répéteur avec sa suppression de cet écran.
