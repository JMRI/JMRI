Le Tableau Tête de Signal

Toutes les têtes de signal que JMRI connaît peuvent être référencées en utilisant l'outil "Tableau Tête de Signal" dans le menu Outils de la plupart des programmes JMRI. Il y a huit colonnes dans le tableau:

  • Le nom système qui est affecté à la tête de signal lors de sa création, et ne peut pas être changé.
  • Si vous êtes intéressé pour avoir un nom plus parlant pour votre tête de signal , vous pouvez cliquer dans la colonne "Nom Utilisateur" et modifier les informations comme vous voulez.
  • Quand les têtes de signal changent, pour une raison quelconque, l'apparence courante se montre dans la colonne "état" du Tableau. Vous pouvez également cliquer sur le bouton d'apparence pour faire défiler les apparences disponibles: rouge, jaune, vert, rouge clignotant, clignotant jaune, vert clignotant, et éteinte. Certaines têtes de signal peuvent aussi avoir l'apparence lunaire et clignotant lunaire disponibles.
  • La colonne "Commentaire" est pour votre usage pour enregistrer tout commentaire que vous voudrez peut-être pour vous aider à garder une trace de vos têtes de signal. Il est pour information seulement.
  • Cliquez sur le bouton "Supprimer" dans une ligne spécifique pour supprimer la tête de signal. Soyez sûr d'enlever d'abord l'icône tête de signal associé à cette tête de signal, toute logique simple signal associée, et toutes les références à cette tête de signal. Utilisez cette touche avec précaution!
  • La case la colonne "Allumé" de vous permet de contrôler si la tête de signal sur le réseau est allumée(cochée) ou reste teinte (non cochée).
  • La case à cocher dans la colonne "Tenue" indique le "lieu" propriété de la tête de signal et vous permet de le changer. Cette propriété est par exemple utilisé par la logique du pupitre CTC* pour dire "Ne changez pas cette tête signal du rouge, même s'il serait prudent de le faire, parce que je veux retenir un train ici ".
  • Cliquez sur le bouton "éditer" dans une rangée pour afficher ou modifier les informations facultatives pour une tête de signal. Notes: Nom système et le type ne peuvent pas être changés. Un seul tête de signal peut être éditée à la fois.

Pour définir une nouvelle tête de signal, cliquez sur le bouton Ajouter ..". Il vous sera demandera le type (sortie triple; Double sortie; SE8c; etc), et toutes les informations de configuration nécessaire.

Utilisez l'option Conserver dans le menu Fichier pour enregistrer les informations de signal sur le disque. Ces informations sont enregistrées avec la configuration dans un fichier XML, avec l'installation du panneau de commande et des choses similaires. Pour stocker vos panneaux dans le même fichier (recommandé), sélectionnez Fichier> Stocker> Stocker Configuration > et les panneaux dans le Fichier ... ou sélectionnez Panneaux> Stocker panneaux ... dans la fenêtre principale JMRI.

Pour plus d'informations sur la création et l'affichage des panneaux, y compris la manière de montrer des signaux sur vos panneaux, s'il vous plaît voir les pages Panneaux.

*CTC commande centralisée manuelle
