Ajouter/éditer Action

La Fenêtre Ajouter/Modifier Action

Les Actions sont des tâches qui peuvent être lancées automatiquement comme un train qui traverse en mode automatique le réseau. Le volet Ajouter Action permet de créer nouvelles actions:

La version "Modifier" de cette fenêtre est la même que celle montrée ci-dessus, canton l'exception du bouton Créer Nouvelle Action qui est est remplacé par un bouton Mise canton Jour Actionet les sélections et données sont remplies par les valeurs actuelles de l'action en cours d'édition.

Une Action est définie en deux étapes:

  1. Quelle action doit être exécutée
  2. Quandl'action doit être engagée.

    3. Quand Agir

    Pour spécifier la partie Quand, sélectionnez une option dans le menu contextuel Quand, et remplir toutes les données qui pourraient être nécessaires. Les Champs d'élément de données ou de menus apparaî tront lorsque l'élément qui en a besoin est sélectionné. Chaque élément de données est fourni avec une description textuelle (bulle d'aide) qui peut être appelée en pointant le curseur sur l'élément. Chaque option Quand est décrite plus en détail dans Actions.

    Un retard optionnel, spécifié en millisecondes, est disponible pour chaque option Quand . Aucun retard est un défaut. Si un délai optionnel n'est pas entré (ou un temporisation zéro est entrée), l'action sera lancée immédiatement après le quand l'événement se soit produit. Si un délai est spécifié, le programme attendra pendant le temps spécifié après le quand l'événement se soit produit avant d'initier l'action.

    Pour spécifier Quelle action est demandée, sélectionner une option dans le menu contextuel Quelle , et remplir toutes les données qui pourraient être nécessaires. Les Champs d'élément de données apparaî tront lorsque l'élément qui en a besoin est sélectionné. Chaque élément de données est fourni avec une description textuelle (bulle d'aide) qui peut être appelée en pointant le curseur sur l'élément.
    Dans la liste déroulante, choisissez l'une des options suivantes:

    • Entrée de section: lorsque le train automatique entre dans la section en cours (voir l'image ci-dessus).
    • Á la sortie de la section - Lorsque le train automatique quitte la section en cours. Cela signifie que le dernier wagon détectable quitte la section et que celle-ci est inoccupée.
    • Entrée sur canton: lorsque le train automatique entre dans un canton spécifié dans la section. Pour les sections avec un seul canton, cela équivaut canton Entrée de section.
    • Á la sortie du canton - Lorsque le train automatique quitte un canton spécifié dans la section. Cela signifie que le dernier wagon détectable quitte le canton spécifié et que le canton est inoccupé. Pour les sections avec un seul canton, cela équivaut canton Sortie de section.
    • Á l'arrêt du train: lorsque le train automatique s'arrête. Cela peut être utilisé, par exemple, pour arrêter la sonnerie dans un décodeur de son ou pour lancer d'autres actions lorsque le train s'arrête canton une gare.
    • Au démarrage du train - Lorsque les trains automatiques démarrent après avoir été arrêtés. Cela peut être utilisé, par exemple, pour déclencher des actions lorsqu'un train démarre après l'arrêt en gare.
    • Sur le capteur ACTIF - Lorsque le capteur spécifié passe canton l'état Actif. Le capteur spécifié peut être interne ou matériel et doit figurer dans le tableau des capteurs. Cela peut être utilisé, par exemple, avec un capteur matériel connecté canton un bouton-poussoir de l'interface client pour déclencher une action. Il pourrait également être utilisé pour déclencher des actions canton l'aide de capteurs de position matériels sur le réseau. Avec des capteurs internes, il peut fournir un lien vers Logix.
    • Sur le capteur INACTIF - Lorsque le capteur spécifié passe canton l'état Inactif. C'est la même chose que ci-dessus, mais avec l'état du capteur opposé. Utilisez celui qui convient le mieux canton votre matériel ou canton Logix.

    Que faire

    Après avoir sélectionné Lorsque, continuez en choisissant l'une des options suivantes Quoi et remplissez les paramètres:

    • Mettre en Pause le Train
    • Définir la Vitesse Maximale
    • Définir la Vitesse du Train
    • Rampe de Vitesse du Train
    • Accéder au Mode Manuel
    • Allumer l'É clairage de la Locomotive
    • Démarrer La Cloche
    • Arrêtez La Cloche
    • Klaxon Sonore
    • Avertisseur Sonore de modèle
    • Définir la Fonction du Décodeur
    • Définir le Capteur ACTIF
    • Définir le Capteur sur INACTIF
    • Signal d'Attente
    • Signal de Libération

    Quand l'action est complètement définie cliquez sur le bouton Créer une Nouvelle Action (ou le bouton Mise canton Jour Action si édition en cours). Toutes les données nécessaires seront validées. Si un problème est détecté, un message apparaî tra, et la fenêtre Ajouter/éditer Actions restera pour permettre la correction du problème. Si tout est vérifié, la fenêtre Ajouter/éditer action va disparaî tre et les actions(nouvelles ou modifiées) apparaî tront dans la fenêtre Voir Actions.

    Cliquez sur le bouton Annuler pour refermer la fenêtre Ajouter/Modifier l'Action, sans créer une nouvelle action ou sans achever le montage d'une action existante . La fermeture de la fenêtre Ajouter/Modifier l'Action équivaut canton cliquer sur le bouton Annuler.
