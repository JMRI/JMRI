Ajout/Édition Aiguillages

Lorsque vous ajoutez un Aiguillage, vous sélectionnez dans la liste déroulante le système auquel l' aiguillage est rattaché.

Entrez une adresse valide pour l'Aiguillage et, si désiré entrer un nom utilisateur.

Création d'un Nombre d'Aiguillage

Il est également possible de créer un certain nombre d'Aiguillage .
La case "Ajouter une plage" sera activée.

Après l'activation de la boite "Ajouter Plage", il faudra ensuite activer le champ "nombre à ajouter". Ici entrez Le nombre d'Aiguillages que vous souhaitez ajouter, ce qui créera ensuite un nombre d'aiguillages séquentiel.

Si un nom utilisateur a été entré un numéro sera ajouté à la fin du nom pour garder chaque nom utilisateur unique.
Voir aide générale pour plus d'informations"sur les Noms".
