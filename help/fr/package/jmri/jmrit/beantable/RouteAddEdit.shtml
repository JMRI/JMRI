Ajout/modification d'Itinéraires

Après avoir cliqué sur Ajouter ... sous la table d'Itinéraire, le panneau suivant apparaît:

La première chose à faire lors de la création d'un itinéraire est de le nommer. Dans la zone intitulée "Nom Système", entrez un nom unique pour votre itinéraire. Vous pouvez éventuellement entrer un nom plus descriptif dans la case "Nom Utilisateur".

Le bouton radio en dessous du "Nom Utilisateur de l'Itinéraire" vous permet d'afficher seuls les aiguillages ou les capteurs qui font partie de cet Itinéraire. Il y a deux tableaux dessous le bouton "Montrer", le premier contient les aiguillages et le second contient les capteurs. Cliquez sur la boite "Inclure" pour ajouter dans les tableaux soit les aiguillages souhaités soit un capteur à votre itinéraire. A côté de la boite "Inclure" est l'action que l'aiguillage ou le capteur peut effectuer lorsque l'Itinéraire est déclenché. Par exemple, si vous souhaitez que l'aiguillage soit dévié lorsque l'itinéraire est déclenché sélectionnez "Définir Dévié" dans le menu déroulant.

Dessous les tableaux Aiguillage et Capteurs il y a trois procédures en option:

  • Lecture d'un fichier son
  • Exécuter un script et
    • li>
  • Spécifier un capteur à activer lorsque tous les aiguillages spécifiés pour l'itinéraire sont correctement alignés.
Les fichiers Audio et Scripts sont configurés en appuyant sur le bouton "...." approprié et en sélectionnant le fichier son à joué, ou le fichier de script à exécuter lorsque l'Itinéraire est déclenché.
Pour ajouter un Capteur qui va devenir actif lorsque tous les aiguillages sont alignés, sélectionnez le Capteur dans la liste. Le Capteur deviendra Actif lorsque les Aiguillages seront correctement alignés pour l'Itinéraire.

Un Itinéraire peut éventuellement être déclenché par l'un des trois capteurs indiqué dans le tableau. Entrez leurs noms (nom système ou nom utilisateur) et sélectionner l'action qui va déclencher l'Itinéraire. Les choix logiques sont décrits en détail Contrôle d'Itinéraires à partir de Capteurs .

Une Itinéraire peut aussi être déclenché par un Aiguillage. Entrez le nom de l'Aiguillage depuis la liste et sélectionnez l'Action de l'aiguillage qui va déclencher l'Itinéraire. Les choix Logique sont expliqués en détail dans Commande Itinéraires à partir d'un Aiguillage.

Dans certains cas, il pourrait être nécessaire de retarder (de ralentir) l'ordre que JMRI envoie pour changer les Aiguillages sur le réseau. En entrant une valeur dans la case "Retard Ajouter:", vous pouvez augmenter l'intervalle de temps (en millisecondes) entre les commandes d'activation lorsque cet itinéraire est déclenché. Par défaut JMRI insère un délai minimum de 250 millisecondes (1/4 de seconde) entre les commandes des aiguillages.

Le prochain menu déroulant vous permet de sélectionner un aiguillage qui verrouille cet itinéraire. Lorsqu'un itinéraire est verrouillé, tous les aiguillages définis pour l'itinéraire deviennent verrouillés .

Une fois que vous êtes satisfait de votre itinéraire, utilisez le bouton " Ajouter Itinéraire" pour créer l'itinéraire. Pour enregistrer votre Itinéraire, vous devez utiliser la commande "Enregistrer Panneau "qui se trouve sous le" Panneaux "du menu principal. Si vous décider que vous devez changer votre itinéraire, utiliser la fonction "Modifier" et "Mise à jour Itinéraire "pour modifier votre itinéraire.

Fonction Verrouillage Aiguillage

La fonction Verrouillage Aiguillage vous permet de désactiver sélectivement les commandes d'accessoires (d'aiguillage) depuis les cabines et/ou les boutons-poussoirs c√¢blés localement. JMRI désactive les boutons de commande en envoyant des commandes de désactivation de bouton de commande aux décodeurs Aiguillage lorsque la route est verrouillée et active les boutons de commande lorsque la route est déverrouillée. Lors du verrouillage d'une commande Aiguillage depuis cabine, le programme JMRI surveillera l'adresse de l'accessoire d'Aiguillage et, s'il détecte une commande de changement depuis une cabine, annulera immédiatement la commande en envoyant la commande opposée à l'Aiguillage. Tous les Aiguillages surveillés par le programme JMRI et inclus dans l'Itinéraire seront verrouillés.

Pour vous aider à sélectionner un ensemble d'appareils de voie à verrouiller ou à déverrouiller, un itinéraire peut être verrouillé ou déverrouillé. Dans la fenêtre éditer l'itinéraire, un verrou de contrôle de l' aiguillage en option est disponible. Vous pouvez utiliser la commande de verrouillage de l'aiguillage pour activer ou désactiver la fonction de verrouillage d'itinéraire depuis un du Panneau d'icône d'Aiguillage, un aiguillage du TCO ou un aiguillage fantôme. Un Aiguillage fantôme est un aiguillage qui peut être consulté par votre cabine, mais n'existe pas réellement sur votre réseau.

Pour plus d'informations sur la définition du verrouillage des aiguillages, voir la page Tableau Aiguillage.
