Ajout/Modification des Objets Audio

Pour créer un objet audio, cliquez sur le bouton "Ajouter ..." Tableau Audio. Cela ouvrira une petite fenêtre avec un espace pour entrer un nom système et nom utilisateur pour la nouvelle mémoire, plus un bouton "Ajouter".

Vous devez entrer un nom système unique. Le système des noms pour les objets audio ont un ensemble de modèle. Ils commencent par "IAL" pour "Auditeur audio interne", "IAB" pour "Tampon Audiointerne", ou "IAS" pour "Source audio interne".

Il ya des fenêtres séparées pour l'ajout de chacun des différents types.

Le nom utilisateur est facultatif, mais si on leur en donne un, il doit être unique.

Retour à la page d'Aide Tableau Audio.

Pour plus d'informations sur les objets audio, consultez la page d'aide audio.
