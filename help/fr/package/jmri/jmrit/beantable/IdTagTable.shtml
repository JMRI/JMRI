Le Tableau étiquette ID

Vous pouvez utiliser le Tableau étiquette ID pour regarder le contenu des objets IdTag JMRI.

Notez que vous pouvez trier le tableau par chacune des colonnes, cliquez simplement sur la tête de colonne.

Pour créer une nouvelle étiquette d'identification, cliquez sur le bouton " Ajouter ..." au bas de la fenêtre.

Pour plus d'informations sur les étiquettes d'identification, consultez les page d'aide .
