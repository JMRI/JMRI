Ajout de variables mémoire

Pour créer une variable mémoire, cliquez sur le bouton "Ajouter ..." du bouton Table de variables mémoire . Cela ouvrira un nouveau volet avec un espace permettant de saisir un Nom Système et un Nom Utilisateur pour la nouvelle Variable Mémoire, ainsi qu'un bouton "Créer" en bas.

Vous devez entrer un Nom Système unique. Les Noms Système des Variables Mémoire n'ont pas de modèle défini. Vous pouvez les appeler comme vous voulez. Il existe une faible convention consistant à les démarrer avec "MI" pour "Mémoire interne", mais ce n'est pas obligatoire.

Le Nom Utilisateur est facultatif, sauf si vous cochez la case "Énoncer automatiquement le Nom Système". Si vous en entrez un, il doit être unique dans votre configuration JMRI. Reportez-vous à la page "Noms" pour plus d'informations.

Création d'un nombre de mémoires

Il est également possible de créer plusieurs mémoires en une fois.
Si vous cochez la case "Ajouter une plage séquentielle", le champ "Nombre d'éléments" sera activé. Définissez le nombre de mémoires que vous souhaitez ajouter. Cliquez sur [Créer] pour créer un certain nombre de mémoires numérotées séquentiellement.
Si un Nom Utilisateur a été saisi, un numéro sera ajouté à la fin de ce nom (en commençant par le deuxième élément) pour que chaque Nom Utilisateur en mémoire reste unique.
