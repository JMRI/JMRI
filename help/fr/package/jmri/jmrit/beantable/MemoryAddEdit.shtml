Ajout/Modification de Mémoires

Â Â Â Â Pour créer une variable mémoire, cliquez sur le bouton "Ajouter ..." sur le Tableau mémoire. Cela ouvrira une petite fenêtre avec un espace pour y entrer un nom système et un nom 'utilisateur pour la nouvelle mémoire, plus et bouton "Ajouter".

Vous devez entrer un nom système unique. Les noms système de mémoires n'ont pas de modèle. Vous pouvez tous mettre tout ce que vous voulez. Il y a une faible convention pour les commencer par "IM" pour "mémoire interne", mais ce n'est pas nécessaire.

Le nom utilisateur est facultatif, mais si on le donne, il doit être unique.
