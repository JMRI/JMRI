Ajouter des cantons

Pour ajouter un nouveau canton, cliquez sur le bouton Ajouter... sur le bord inférieur gauche du tableau des cantons.

Dans le volet qui s'ouvre, saisissez un nom système valide pour le un nom système valide pour le bâtiment et, si vous le souhaitez, un nom utilisateur. Par exemple, un nom système valide pour un réseau LocoNet pourrait être LB456. Un autre exemple, un nom système de Canton interne valide pourrait être . IB123.

Le nom utilisateur est facultatif, à moins que vous ne cochiez "Générer automatiquement le nom système", et si vous en saisissez un, il doit être unique dans votre configuration JMRI . Voir la page d'aide " Noms" pour plus d'informations.

Cliquez sur [Créer] pour terminer cette étape. Cliquez sur [Annuler] pour fermer le volet (ne vous inquiétez pas, les éléments déjà créés resteront).

Complétez la définition du nouveau canton en fermant le Volet Ajouter un nouveau canton et en cliquant sur le bouton Modifier à droite de l'entrée du canton nouvellement créé dans le volet Tableau des cantons.
Remplissez tous les champs comme décrit sur la page d'aide Attributs Canton .

Créer un certain nombre de cantons

Il est également possible de créer un certain nombre de cantons en une seule fois.
Lorsqu'elle est prise en charge, la case à cocher " Ajouter une plage séquentielle " sera activée.

En cochant la case "Ajouter une plage séquentielle", vous activerez le champ "Nombre d'éléments". Définissez le nombre de cantons que vous souhaitez ajouter. Cliquez sur [Créer] pour créer un certain nombre de cantons numérotés de manière séquentielle.
Si un nom utilisateur a été saisi, un numéro sera ajouté à la fin de ce nom (en commençant par le numéro 2) pour que chaque nom utilisateur de canton soit unique.

Ajouter des propriétés de canton

Pour ajouter les détails d'un nouveau bâtiment, fermez le volet Ajouter un bâtiment en cliquant sur Annuler ou sur le bouton Fermer/X et cliquez sur Editer à la droite de la nouvelle entrée dans le tableau des cantons.

Voir Noms sous la rubrique Aide générale pour plus d'informations sur la dénomination des objets JMRI.
