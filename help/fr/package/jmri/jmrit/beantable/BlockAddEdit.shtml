Ajout de cantons

Pour ajouter un nouveau canton, cliquez sur le bouton Ajouter ... en bas à gauche du volet Tableau des cantons.

Dans le volet qui s'ouvre, entrez un nom système valide pour le canton et, si vous le souhaitez, entrez un nom utilisateur. Par exemple, un nom système valide pour un réseau locoNet pourrait être LB456. Un autre exemple, un nom système de canton interne valide pourrait être IB123.

Le nom utilisateur est facultatif, sauf si vous cochez "Générer automatiquement le nom système", et si vous en entrez un, il doit être unique dans votre configuration JMRI. Voir les Pages d'aide" Noms " pour plus d'informations.

Terminez la définition du nouveau canton en fermant le volet Ajouter un nouveau canton et en cliquant sur le bouton Modifier à droite de l'entrée du canton nouvellement créée dans Table de canton .
Remplissez tous les champs comme décrit sur la page d'aide Attributs de canton .

Création d'un certain nombre de cantons

Il est également possible de créer plusieurs cantons en une seule fois.
Lorsqu'elle est prise en charge, la case à cocher "Ajouter une plage séquentielle" sera activée.

Cochez la case "Ajouter une plage séquentielle" pour activer le champ "Nombre d'éléments". Définissez le nombre de cantons que vous souhaitez ajouter. Cliquez sur [Créer] pour créer un certain nombre de cantons numérotés séquentiellement.

Ajout de propriétés de bloc

Pour ajouter les détails d'un nouveau bloc, fermez le volet Ajouter un bloc en cliquant sur Annuler ou sur le bouton Fermer/X et cliquez sur Modifier à droite de la nouvelle entrée dans la table des blocs.

Voir Noms sous Aide générale pour plus d'informations sur la dénomination des objets JMRI.
