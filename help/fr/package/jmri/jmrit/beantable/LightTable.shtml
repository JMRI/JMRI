Le Tableau des Lumières

Les Lumières sont des bits de sortie numériques qui fonctionnent comme des interrupteurs. Les Lumières ont été créées pour fournir un Commande facile pour les Lumières des TCO et l'éclairage des décors. Les Lumières peuvent également être utilisées pour tout usage nécessitant un interrupteur marche/arrêt.

Les Lumières peuvent également être définies comme étant d'intensité variable (et pas seulement On/Off). Cela dépendra de l'interface et du matériel qui pilotent la Lumière.

La table des Lumières contient les colonnes suivantes :

  • Nom Système
  • Nom Utilisateur (facultatif)
  • Etat
  • Commentaire (facultatif, double-cliquez pour le modifier)
    • .
  • Activé (Commande automatisé de la Lumière, sauf Logix)
  • Intensité (disponible dans certains matériels uniquement)

Le tableau des Lumières contient une colonne 'Activée'. Pour qu'une Lumière soit déclenchée par son Capteur de commande, l'Horloge Rapide ou son Aiguillage de Commande, elle doit être "activée", c'est-à-dire que sa case à cocher dans la colonne "Activée" doit être cochée. Vous pouvez décocher cette case pour empêcher temporairement une Lumièrede se déclencher, c'est-à-dire pour l'empêcher de s'allumer lorsque son mécanisme de commande se déclenche.
Vous pouvez modifier l'ordre des colonnes en faisant glisser leurs titres, et trier les lignes en cliquant sur l'un des triangles de tri à côté du titre d'une colonne

Notez que la colonne activée/désactivée ne s'applique qu'au mécanisme de Commande Automatique d'une Lumière.


Un état " désactivé " n'empêchera pas une Lumière d'être allumée ou éteinte à l'aide du bouton dans le tableau des Lumières ou d'être commandée par un Logix.

Choisissez entre l'affichage textuel et graphique de l'état de la Lumière. dans les Préférences d'affichage.

Création de nouvelles Lumières

Pour définir une nouvelle Lumière, cliquez sur le bouton Ajouter.... dans le tableau des Lumières. Cela vous amènera à une nouvelle fenêtre pour ajouter ou modifier des Lumières.

Allumer/éteindre les Lumières/régler l'intensité

Les Lumières peuvent être allumées ou éteintes en cliquant sur le bouton/icône dans la colonne État du Tableau des Lumières.

Une Lumière peut être configurée dans le Tableau des Lumières pour être commandée par l'état d'un Aiguillage, un Capteur d'Occupation de Canton, ou un ou deux Capteurs. Par exemple, le bouton de TCO peut être spécifié comme le Capteur de commande de la Lumière, l'allumant ou l'éteignant. La commande à deux Capteurs est utile pour Commander les sons de structure ou les barrières de passage à niveau.

Une Lumière peut également être commandée à partir d'un Logix, en tant qu'action d'une de ses conditions. Si vous avez besoin d'une logique plus puissante pour commander votre Lumière, envisagez d'utiliser un Logix.

Notez que l'option Activé/Désactivé ne s'applique qu'au mécanisme de Commande automatisé d'une Lumière. Un état " désactivé " n'empêchera pas une Lumière d'être allumée ou éteinte à l'aide du bouton du Tableau Lumière ou d'être Commandée par un Logix.

Une Lumière variable peut également être ajustée en saisissant une valeur de 0,00 à 1,00 dans la colonne Intensité pour la Lumière. Une Lumière qui ne prend pas en charge les intensités variables considèrera toute valeur non nulle comme signifiant "On". Si une Lumière a des effets de transition activés et configurés, la Lumière répondra en fonction de ces options.

Le bouton d'état d'une Lumière variable sera également une indication d'état de la Lumière variable. Lorsqu'il n'est pas activé ou Off, il indiquera si le dernier changement était un déplacement vers le haut ou vers le bas. . L'intensité affichée sera l'intensité actuelle, et non l'intensité cible, lorsque les effets de transition sont activés.

Enregistrer les Lumières sur le disque

Le Tableau Lumières est conservé dans votre configuration de réseau, avec les Aiguillages, les Capteurs, les Feux de signalisation, la configuration du panneau de commande, etc. Pour stocker ces informations sur disque, utilisez Enregistrer > dans le menu Fichier en haut du Tableau Lumières (ou d'autres Tableaux). ou sélectionnez Sauvegarder Panneau... dans le menu Panneau. Notez que l'état activé/désactivé de chaque Lumière n'est pas enregistré dans le fichier de configuration. Lorsque les Lumières sont chargées à partir d'un fichier de configuration, elles sont toutes activées.
