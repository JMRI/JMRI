Ajouter/Modifier un Groupe de Signaux

Après avoir cliqué sur le bouton "Ajouter ..." situé sous le Tableau des Groupes de Signaux, le volet suivant apparaît:

Entrer dans le Groupe de Signaux

Les Groupes de Signaux sont construits à partir d'un ou plusieurs Signaux et comprennent les éléments suivants:

  • Un Mât de Signalisation parent (par Nom Système) auquel ce Groupe de Signaux est attaché.
  • Une liste des Aspects qui rendent le Groupe de Signaux actif, par exemple: le Mât de Signalisation parent doit être défini sur Continuer ou Avertissement, mais pas sur Danger.
  • Une liste des Signaux qui composent chaque indication du Groupe de Signaux.
    Les Signaux inclus dans un Groupe de Signaux sont sélectionnés dans le Tableau des Signaux, mais seuls les Signaux créées à l'aide du type de connexion Sortie Simple (illustré à droite) apparaissent dans la liste.

Dans la fenêtre Ajouter/Modifier un Groupe de Signaux, après avoir saisi un Nom Utilisateur pour le nouveau Groupe de Signaux et sélectionné le Mât de Signalisation parent dans la liste déroulante, cochez la case Inclure derrière un ou plusieurs Aspects du Mât dans le tableau directement ci-dessous.
Sélectionnez ensuite un ou plusieurs Signaaux en cochant les cases Inclure dans la deuxième liste.
Cliquez sur le bouton Appliquer en bas du volet ou poursuivez la configuration.

Définir les Conditions du Groupe de Signaux

Chaque signal que vous incluez dans le groupe de signaux a un ensemble de critères à satisfaire avant que ce signal ne soit allumé. Ces critères sont constitués d'états d'Aiguillage et/ou de Capteur et peuvent être modifiés en cliquant sur le bouton Modifier correspondant dans la dernière colonne du volet Ajouter/Modifier un Groupe de Signaux. Un nouveau volet comme celui présenté à droite s'ouvrira, vous permettant de définir un signal spécifique faisant partie de votre Indicateur.

Choisissez deux valeurs différentes dans les menus déroulants en haut du volet pour définir les états du Signal Indicateur. On lorsque les conditions sont remplies, et Off tant que ce n'est pas le cas.
Pour compléter la définition d'un Indicateur de Groupe de Signaux, cochez les cases Inclure appropriées pour les Aiguillages et les Capteurs, puis spécifiez leur état dans la dernière colonne de chaque liste.
Entre les deux listes, il y a une option logique finale OU/ET parmi laquelle choisir.

Vous devrez réfléchir profondément pour vous préparer à cette dernière étape, car c'est à cet endroit que vous contrôlerez le comportement de votre Signal. Le matériel de signalisation de votre réseau peut exclure certaines options. Créez-le par petites étapes et testez-le souvent.

Terminez la définition du Groupe de Signaux en cliquant sur Appliquer dans le volet de configuration de Signaux, puis à nouveau sur Appliquer pour fermer le volet Ajouter un Groupe de Signaux. Votre nouveau Groupe apparaîtra dans le Tableau.
