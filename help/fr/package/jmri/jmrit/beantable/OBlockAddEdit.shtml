Ajouter des OCantons

Pour ajouter un nouvel OCanton dans l'interface à onglets, cliquez sur le bouton Ajouter un OCanton... sur le bord inférieur gauche du volet Tableaux Occupation.

.

Dans le volet qui s'ouvre, saisissez un Nom Système valide pour l'OCanton et, si vous le souhaitez, entrez un Nom Utilisateur. Par exemple, un Nom Système valide pour une disposition LocoNet pourrait être OB456.

Le Nom Utilisateur est facultatif, à moins que vous ne cochiez "Générer automatiquement le Nom Système", et si vous en saisissez un, il doit être unique dans votre configuration JMRI. Voir la page d'aide " Noms" pour plus d'informations.

Cliquez sur [Créer] pour terminer cette étape. Cliquez sur [Annuler] pour fermer le volet (ne vous inquiétez pas, les éléments déjà créés resteront).

Complétez la définition du nouvel OCanton en fermant le Volet Ajouter un nouvel OCanton et en cliquant sur le bouton Modifier à droite de l'entrée du Canton nouvellement créé dans le volet Tableau OCanton.
Remplissez tous les champs comme décrit sur la page d'aide Attributs Cantons.

Créer un certain nombre d'OCantons

Il est également possible de créer un certain nombre de Cantons en une seule opération .
Lorsqu'elle est prise en charge, la case à cocher " Ajouter une plage séquentielle " sera activée.

En cochant la case "Ajouter une plage séquentielle", vous activerez le champ "Nombre d'éléments". Définissez le nombre d'OCantons que vous souhaitez ajouter. En cliquant sur [Créer], vous créerez un certain nombre de Cantonss numérotés de manière séquentielle.
Si un Nom Utilisateur a été saisi, un numéro sera ajouté à la fin de ce nom (en commençant par le nÂ° 2) pour que chaque Nom Utilisateur de Canton reste unique.

Ajouter des propriétés dd'OCantons

Pour ajouter les détails d'un nouvel OCanton, fermez le volet Ajouter OCanton en cliquant sur Annuler ou sur le bouton Fermer/X et cliquez sur Modifier à la droite de la nouvelle entrée dans le tableau OCanton.

Voir Noms sous la rubrique Aide générale pour plus d'informations sur la dénomination des objets JMRI.
