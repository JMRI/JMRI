Ajout/Modification d'étiquettes ID

Pour créer une étiquette ID, cliquez sur le bouton "Ajouter ..." dans le volet Tableau Étiquette ID.
Cela ouvrira une petite fenêtre avec un espace pour entrer un Nom Système et un Nom Utilisateur pour la nouvelle étiquette ID, plus un bouton "Ajouter" pour confirmer votre saisie.

Vous devez entrer un Nom Système unique. Les Noms Système des Étiquettes ID doivent avoir le modèle suivant.

  • Ils devraient commencer par "ID" pour"iDentité Interne",
  • suivi par le numéro actuel de l'Étiquette elle-même.

Le Nom Utilisateur est facultatif, mais si on lui en donne un, il doit être unique.

L'Étiquette ID indiquée dans le tableau est déterminée à partir du Nom Système.
