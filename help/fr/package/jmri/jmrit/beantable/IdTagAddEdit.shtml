Ajout/Modification d'étiquettes ID

Pour créer une étiquette d'identification, cliquez sur le bouton "Ajouter ..." sur la Table ID Tag. Cela ouvrira une petite fenêtre avec un espace pour entrer un nom système et un nom utilisateur pour la nouvelle étiquette d'identification, plus une bouton "Ajouter".

Vous devez entrer un nom système unique. Les Noms système des étiquettes ID doivent avoir le modèle suivant. Ils devraient commencer par "ID" pour "identité interne", suivi par le numéro de l'étiquette elle-même.

Le nom utilisateur est facultatif, mais si on leur en donne un, il doit être unique.

L'étiquette ID indiquée dans le tableau est déterminée à partir du nom système.
