JMRI: Sélectionnez le mode PR3

Une PR3 Digitrax peut agir soit en tant que programmateur autonome, ou comme une interface à un LocoNet, mais il ne peut pas faire les deux en même temps. Normalement, vous choisissez le mode PR3 dans le cadre fixant les préférences JMRI .

Dans certains cas, vous pourriez vouloir changer rapidement le mode pendant le débogage. Cet outil offre deux boutons pour vous permettent de changer le mode de la PR3 immédiatement. Remarque que cela peut entraîner un comportement inhabituel (et généralement mauvais) du programme! Il est généralement préférable de basculer le mode en changeant les préférences JMRI et en redémarrant le programme.

Pour plus d'informations sur la PR3, s'il vous plaît voir ses page de référence JMRI.
