LocoNet Moniteur d'emplacements

L'outil Moniteur slot affiche un tableau du contenu des "emplacements de la centrale de commande". Ces emplacements sont utilisés pour contrôler les locomotives individuelles et le couplage. L'affichage inclut le format des crans de vitesse du décodeur, la vitesse actuelle et les paramètres de fonction, les information de couplage et de statut. Vous pouvez choisir d'afficher soit tous les emplacements ou tout simplement ceux utilisés actuellemnet.

Les cases à cocher en haut vous permettent de montrer juste les emplacements actifs (en service), ou tous les emplacements de locomotive. Vous pouvez également choisir d'afficher ou non le "système" des slots, qui ne portent pas d'information de locomotive. Le bouton "ESTOP tous" définit la vitesse de toutes les locomotives à l'"Arrêt d'urgence" très rapidement.

Les colonnes sont:

slot
Le numéro de l'emplacement
E Stop
Cliquez sur ce bouton pour basculer la vitesse du décodeur sur l'arrêt d'urgence
Adresse
L'adresse de la locomotive de l'emplacement de contrôle
Vitesse
Pour les locomotives non couplées et les adresses qui sont en tête d'un couplage, c'est le réglage de la vitesse actuelle. C'est une valeur interne pour l'emplacement de couplage.
Décodeur de Type
Le format des commandes envoyées au décodeur, généralement soit 128 crans ou 28 crans, bien que d'autres valeurs soient possibles.
Statut
L'un des quatre états possibles:
Libre
non réservé à un usage particulier, aucun contenu
Inactif
non réservée à un usage particulier, aucun contenu
Commun
non utilisés pour le moment, mais contient encore des données valides pour le dernier décodeur utilisé
En Service
Contrôle Actuellement un décodeur
Libre
Cliquez sur ce bouton pour libérer l'emplacement, de sorte qu'il peut être réutilisé pour une autre locomotive
Couplé
Affiche l'état couplé de l'emplacement, si ce n'est pas couplé, le d'un couplage, inclus dans un couplage, etc.
ID Régulateur
L'ID du régulateur utilisant cet emplacement. C'est affiché sur deux octets hexadécimaux. La deuxième paire peut être configurées dans le régulateur, voir les instructions Digitrax.
Direction
si le décodeur est actuellement commandé pour aller en avant ou en arrière
F0-F8
Les paramètres de la fonction actuelle.
