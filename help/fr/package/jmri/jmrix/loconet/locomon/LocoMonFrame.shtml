Moniteur de Trafic LocoNet


L'outil Moniteur LocoNet affiche le trafic LocoNet dans un une forme lisible. Les 20 derniers messages sont disponibles dans une fenêtre de défilement. En option, vous pouvez inclure le moment où le message a été reçu, et/ou le paquet octets brut . Les informations du journal peuvent également être stockées dans un texte fichier.

Pour plus d'informations sur les différents contrôles et les options, s'il vous plaît voir la page Fenêtre Moniteur de Communications
