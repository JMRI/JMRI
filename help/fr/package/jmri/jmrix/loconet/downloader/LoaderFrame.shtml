Digitrax Téléchargeur de Programme

Certains produits peuvent avoir de nouveau firmware (programmes internes) Digitrax téléchargés pour eux. Les mises à jour du firmware sont répartis dans des fichiers". dmf" (Digitrax Firmware Mangled). Ils contiennent à la fois le code lui-même, et des informations de gestion variées pour s'assurer qu'il peut être téléchargé dans le matériel qui l'attend.

Notez que ce n'est pas le même que L'outil de Téléchargement, qui est utilisé avec un PR2 ou PR3 pour charger des projets solides dans un nouveau décodeur sonore Digitrax.

Pour utiliser cet outil:

  • Cliquez sur le bouton Sélectionner et sélectionnez le fichier. dmf vous souhaitez télécharger.
  • Cliquez sur Lire le fichier pour lire le fichier dans le programme et vérifier son contenu
  • Seulement si demandé par Digitrax, modifier les différentes valeurs dans Les champs de contrôle (ce n'est pas nécessaire habituellement)
  • Cliquez sur Télécharger pour charger le firmware. Cela va prendre un peu de temps, de sorte il ya la barre de progression sous les boutons.
  • Si désiré, cliquez sur Vérifier pour vérifier le contenu. Cela prend aussi du temps tout.
