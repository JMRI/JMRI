Conditions de déclenchement SDF

Cette page documente la signification des conditions de déclenchement qui peuvent être définies dans une instruction de Digitrax SDF.

TRIG_NEVER
Jamais déclenché. Ceci peut être utilisé pour désactiver une partie d'un fichier SDF.
TRIG_MODE_CHNG
TRIG_MATH
TRIG_DISTANCE
TRIG_SPD_INC
se déclenche lorsque la vitesse du moteur est augmentée.
TRIG_SPD_DEC
se déclenche lorsque la vitesse du moteur est réduite.
TRIG_CAM
déclencheurs soit sur la came actuelle saisie, ou le mécanisme de autochuff.
TRIG_NOT_TRIG
Fonction 0 On
se déclenche lorsque la fonction DCC 0 est On.
Fonction 28 On
TRIG_DIRNOW_CHNG
se déclenche lorsque la direction du moteur est changée.
TRIG_MOVING
Si le moteur est en cours d'exécution à une vitesse non nulle. Fonction
TRIG_SND_ACTV11 11
TRIG_BSC0
se déclenche quand un "changement de l'état binaire" du paquet DCC est reçu pour BSC 0.
TRIG_BSC7
Trigger
TRIG_IN_0 lorsque la connexion d'entrée devient active, habituellement utilisé pour une entrée chuff.
TRIG_IN_7
TRIG_NOTCH_CHNG
TRIG_TIME_16PPS
déclencheurs 16 fois par seconde.
TRIG_FACTORY_CVRESET
TRIG_OPSPROGWR_CV
se déclenche quand un CV est écrit en mode ops (sur la voie principale).
