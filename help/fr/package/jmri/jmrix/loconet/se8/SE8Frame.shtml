Configurer Digitrax SE8c

L'outil de programmation JMRI SE8c vous permet de configurer les options internes d'un SE8c directement á partir de votre ordinateur.

Pour ouvrir cet outil á partir de la page de démarrage JMRI sélectionnez l'onglet LocoNet puis trouver l'outil sur la liste déroulante.

Pour l'utiliser, vous mettez l'adresse d'une carte SE8c dans le champ texte et cliquez sur "Lire SE8c". L'outil relira l'état actuel des différentes options et définira les cases á cocher pour correspondre aux valeurs actuelles.

Vous pouvez alors, si vous le souhaitez, modifier les cases et cliquez sur "Ecrire dans SE8c" pour rendre vos modifications permanentes.

Limitations

En raison de la façon dont fonctionne la carte SE8c, cet outil ne peut pas changer l'adresse de base de l'unité.
