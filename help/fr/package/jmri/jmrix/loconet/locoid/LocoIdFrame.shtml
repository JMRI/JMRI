Configurer ID LocoNet

Si vous utilisez le récepteur radio(s) UR91 sur votre LocoNet, vous avez une ID LocoNet. Utiliser les manettes de radio et UR91 récepteur (s) pour vous assurer qu'ils se parlent bien les uns aux autres, au lieu de quelques autres appareils radios.

Vous avez seulement besoin de changer l'ID LocoNet quand il y a plusieurs LocoNets séparées utilisant la radio à proximité. Lors d'une exposition s'entraîner avec plusieurs réseaux à base LocoNet utilisant les manettes de radio, par exemple, ils ont besoin d'avoir différents identifiants LocoNet ou leurs radios va interférer les unes avec les autres. L'outil LocoID (sous le menu Digitrax) vous permet de lire et de modifier l'ID.

Notez que l'ID est stockée dans le récepteurs de radio UR91, de sorte que si vous n'avez pas un de ceux-ci, cet outil ne sera pas en mesure de lire ou écrire des valeurs.

Pour plus d'informations sur le fonctionnement UR91 et la radio, voir la page Digitrax correspondant
