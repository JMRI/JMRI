JMRI: contrôle d'entrée USB

La fenêtre de contrôle d'entrée USB vous permet de connecter des capteurs et des mémoires JMRI à l'entrée USB sur votre ordinateur. Par exemple, vous pourriez avoir un capteur JMRI qui devient ACTIF lorsque vous tapez une clé ou appuyer sur un bouton du joystick. Vous pouvez ensuite utiliser Itinéraire et Logix pour utiliser cette entrée USB pour contrôler votre réseau.

Sur le côté gauche de la fenêtre est un arbre montrant les Contrôleurs USB disponibles et les composants en leur sein. Selon les détails de votre ordinateur, cet arbre peut être complètement peuplés lorsque la fenêtre s'ouvre, ou vous pourriez avoir à changer les entrées (par exemple cliquer sur un bouton) pour qu'il se manifeste. Les appareils qui sont branchés après le démarrage du programme peuvent ou pas apparaitres, selon les spécificités de configuration de votre ordinateur.

Lorsque vous sélectionnez un seul composant dans l'arbre affiché à gauche, le panneau de droite vous montre les détails de ce composant et sa valeur actuelle.

En dessous se trouve une zone permettant de configurer le programme pour piloter un Capteur et/ou une Mémoire JMRI à partir de l'entrée USB. Vous pouvez entrer un nom de capteur dans le champ texte en haut, puis cochez la case "Copier capteur JMRI". Lorsque l'entrée devient On(pour une entrée numérique), ou a une valeur supérieure à zéro (pour une entrée analogique), le capteur sera fixé à ACTIF, et il sera autrement, réglé sur INACTIF. Vous pouvez regarder ce travail dans la Table capteur .

Notez que vous ne pouvez pas éditer le nom du capteur tant que la case est cochée; vous devez décocher la case de sorte que le Capteur arrête de suivre l"entrée USB avant de changer pour un nouveau Nom Capteur.

Cette fonctionnalité a été ajoutée sur une base expérimentale dans JMRI 2.1.5. Dans cette version, vos informations de configuration ne sont pas rétablie après vous ayez quitté et redémarréez le programme. Ce sera ajouté dans la prochaine Version JMRI.
