Modifier Adresse d'une Carte Grapevine

Cet outil vous permet de changer l'adresse d'une carte Grapevine.

La carte se souvient du changement, même à travers les cycles alimentation On/Off, jusqu'à ce que vous le changiez à nouveau.

Saisissez le numéro de nœud d'origine, le numéro du nouveau nœud, et appuyez sur "Changer le numéro de nœud".

Vous pouvez ouvrir la Table nœud et cliquez sur "Vérifier" pour voir si le renumérotation a réussi.
