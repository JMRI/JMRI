Configurer les Nœuds Grapevine

L'outil "Table Nœuds" vous permet de voir tous vos nœuds à la fois. Chaque ligne du tableau correspond à une adresse de noeud. Les colonnes sont les suivants:

La troisième colonne fournit un bouton pour vous pourajouter un noeud manquant ou modifier la configuration d'un noeud existant. Si vous Ajouter ou modifier un noeud, n'oubliez pas de sauvegarder vos préférences à nouveau!
Init
Si le noeud est configuré, il ya un bouton dans la 4e colonne qui vous permet d'envoyer les messages d'initialisation pour le noeud. Cet est utile si par exemple le nœud a été Off et On, il doit donc être réinitialisé. Ceci est uniquement disponible pour les noeuds configurés.

Au bas de la fenêtre est un bouton "Vérifier" et un champ état. Cliquez sur "Vérifier" pour balayer le Grapevine pour vérifier son état et localiser les noeuds. Les champ d'état sont les suivant:

Démarrer
Un balayage commence, mais aucune réponse n'a été entendu de Grapevine encore.
Vérification
Balayage en cours, et le Grapevine semble fonctionner.
OK
OK Analyse terminée.
Grapevine échoué
Analyse terminée sans rien entendre de Grapevine. Peut-être le câble n'est pas complètement connecté, ou quelques éléments ne sont pas alimentés.

Il y a aussi un bouton "Changement d'Adresse Nœud Carte", ce qui peut être utilisé pour changer l'adresse d'un nœud de Grapevine. Il ouvre une nouvelle fenêtre.
