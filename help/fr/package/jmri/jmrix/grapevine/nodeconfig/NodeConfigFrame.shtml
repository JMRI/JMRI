Configurer les nœuds Grapevine

Les nœud Grapevine doivent être configurés avant les travaux de saisie.

Le logiciel se décline en trois versions distinctes:

2002 noeud, version 6 ou ultérieure
Généralement considéré comme la meilleure
2002 nœud, version précedent la version 6
Si vous avez un de celles-ci, vous devriez probablement mettre à jour le logiciel
2000 (original) noeud
Bien que JMRI a le code pour soutenir ces derniers, ils n'ont pas été testés. Si vous rencontrez des problèmes, s'il vous plaît laissez-nous désormais améliorer le soutien pour ces cartes.
Bien qu'a l'heure actuelle il n'y ait pas de propriétés supplémentaires, nous nous attendons à éventuellement ajouter plus d'informations ici dès nous apprendront du nouveau au sujet des capacités des nœuds Grapevine.
