OpenLCB: Parcourir la Configuration

L'outil "Configurer Noeuds" OpenLCB vous permet de voir directement les noeuds de votre réseau OpenLCB et de faire quelques manipulations. Il fonctionne avec votre L'installation OpenLCB pour recueillir les informations que vous avez besoin pour exécuter l'outil, il n'y a pas de fichiers locaux de configuration sur votre ordinateur.
La première fois que vous ouvrez la fenêtre de l'outil, il montre un arbre des Noeuds OpenLCB accessibles. Pour la plupart des installations, il s'agit juste justela liste de ce qui est déterminé par l'intermédiaire d'un adaptateur CAN.
Vous pouvez cliquer pour ouvrir chaque noeud et en savoir plus sur lui.
Ouverture de la rubrique "Protocoles pris en charge", par exemple, va vous montrer les capacités de ce nÅ"ud particulier.
Si vous cliquez sur la rubrique "CDI" , pour Configuration Description Information, vous aurez un outil de configuration DecoderPro - comme pour le noeud. Notez que ceci est entièrement pris à partir du noeud OpenLCB lui-même. Il n'est pas nécessaire de trouver un fichier de configuration décodeur, sélectionner le type de carte à droite , etc
Vous pouvez le remplir et cliquez sur "Write" pour configurer le noeud. Les premiers champs sont utilisés pour lui donner un nom et une description choisi par l'utilisateur, de sorte que vous pouvez vous rappeler de ce noeud plus tard. Vous pouvez définir les événements qui correspondent aux différentes entrées et sorties, et même de donner des noms utilisateur commodes. C'est tout stocké en permanence dans le noeud.
Après un redémarrage, cette information est alors montree puis dans l'arbre de configuration pour vous aider à reconnaître vos noeuds.
