JMRI : CBUS Outil de commutation de mode simple

Le sélecteur de mode permet de contrôler les modes de programmation disponibles pour la centrale connectée. Toutes les centrales CBUS ne prennent pas en charge le sélecteur de mode.

Ouvrir le sélecteur de mode (s'il est disponible) à partir du menu SPROG Generation 5.

. .

Sélectionner le mode programmateur pour la programmation en mode service sur une voie de programmation.

Sélectionner le mode centrale pour faire circuler les trains et utiliser le mode de programmation ops (sur la voie principal) .

.

La barre d'état de la fenêtre principale de Decoder Pro indique les programmateurs actuellement disponibles.
