JMRI : Outil bootloader CBUS®

L'outil bootloader vous permet de mettre à jour le firmware dans un nœud CBUS connecté.

Pour mettre à jour le firmware, vous aurez besoin d'un fichier de programmation du firmware (.hex ) enregistré sur votre ordinateur.

Démarrez l'outil de mise à jour du firmware à partir du menu de la génération 5 de SPROG.

.

Entrez le numéro de nœud (la valeur par défaut est 65534) et cliquez sur Lire les paramètres du nœud.

.

Naviguez jusqu'à l'endroit où vous avez enregistré le fichier .hex et cliquez sur Ouvrir.

Dans ce cas, il est possible d'ouvrir le fichier. .

Sélectionnez l'option de mémoire EEPROM pour préserver le contenu de l'EEPROM (variables de nœud).

.

Cliquez sur le bouton Démarrer la programmation.

Cliquez sur le bouton Démarrer la programmation. .

Le nouveau firmware sera écrit sur le nœud. Attendez que la programmation soit terminée, ce qui peut prendre un peu de temps, indiqué par le message "Bootloading complete..." dans la console.
