Fenêtre Moniteur de Communication

JMRI offre une fenêtre de communication pour surveiller pour que vous puissiez voir ce qui se passe sur le lien de communication entre le programme et le matériel du réseau.

Une fois que vous ouvrez une fenêtre du moniteur, il affichera automatiquement à l' écran tout le trafic sur le lien aussi logtemps que "Filtres Octets" est vide

Commandes

Effacer Ècran
Efface la zone de connexion au haut de l'écran
Gel de l'Ècran
Arrête l'acquisition de nouvelles informations, de sorte que l'information saisie cesse de défiler
Montrer données brutes
Normalement, le moniteur affiche l'interprétation de ce qu'il voit du programme. Si vous voulez voir les octets bruts, cochez cette case.
Montrer horodatage
Habituellement, le programme affiche simplement les données du lien de communication. Si vous souhaitez que chaque message soit être précédé par l'heure à laquelle qu'il a été envoyé ou a été reçu, cochez cette case.
Ajoutez message
Si vous voulez insérer un message dans les informations capturées, par exemple de faire une note sur ce que vous vous apprêtez à faire, vous pouvez le taper dans le champ de texte à côté du bouton "Ajouter message " et puis cliquez sur ce bouton. Ceci est particulièrement utile lorsque vous vous connectez à un fichier (voir ci-dessous) et que vous voulez faire une note pour plus tard sur ce qui se passe.

Journalisation dans un fichier

Parfois, il est utile de capturer ce qui se passe. Par exemple, vous pouvez rencontrer un bug (!!), et que vous voulez envoyer une copie de ce qui se arrive aux gens qui tentent d'y remédier.

De petites quantités de données peuvent être copiées à partir de la fenêtre journal avec vos ordinateurs par la commande "Copier" (par exemple Ctrl-C ou cmd-C), puis collé dans un email.

Si vous voulez capturer davantage, vous pouvez avoir la fenêtre pour rédiger dans un fichier journal sur le disque de votre ordinateur. Pour ce faire:

  1. Cliquez sur "Choisir le fichier journal". Une fenêtre de sélection de fichier va s'ouvrir. Naviguez à l'endroit oò vous souhaitez stocker le fichier, entrez un nom pour le nouveau fichier journal, puis cliquez sur "Enregistrer". Notez que si vous choisissez un fichier qui existe déjà , il sera écrasé.
  2. Lorsque vous êtes prêt à capturer les données que vous souhaitez, cliquez sur "Démarrer l'enregistrement" sur la fenêtre du moniteur.
  3. Lorsque vous avez terminé, cliquez sur "Arrêter l'enregistrement" pour stocker les données et fermez le fichier.
