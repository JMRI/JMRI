LogixNG Reference

The reference provides the detailed information for using LogixNG. For people familiar with Logix, chapter 1 should provide a good introduction and make it possible to create a simple LogixNG/ConditionalNG. The remaining chapters in the Standard Features list, except for the clipboard, are similar to Logix documentation.

The Advanced Features list are new concepts that do not exist in Logix or are significant enhancements to Logix capabilities (References).

LogixNG Examples

The examples are detailed explanations of using various LogixNG features. Each example has a zip file that contains the layout configuration data xml file and any supporting files, such as CSV files for table driven logic. The minimum JMRI version is in parentheses.
