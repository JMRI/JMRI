Hardware Support: DCC++ (deprecated)

The original DCC++ project is no longer maintained.
For current documentation on using DCC-EX with JMRI, see the EX-CommandStation (DCC-EX) page.

DCC++ was an Open Source Arduino-based Do-it-Yourself DCC Command Station. It has been replaced by the EX-CommandStation by the DCC-EX team, which continues to function with JMRI and runs on the existing DCC++ hardware.

JMRI continues to support older DCC++ Base Stations via the same DCC-EX connection used for EX-CommandStations. From the JMRI Preferences, select DCC-EX as the System Manufacturer.
