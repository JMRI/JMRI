The DCS210+ is a sort of cross between a DCS210 and a DCS240. It includes the basic functionality of the DCS210, but, like the DCS240, also includes an internal version of something which approximates the PR4 interface device.

Connecting JMRI to a Digitrax DCS210+

For the purposes of connecting a DCS210+ command station to a computer and configuring JMRI, you may use the instructions provided for the DCS240 command station, found here, except configure the connection's "Command station type" as "DCS210 (Evolution Command Station)".

At some future time, JMRI may include a specific connection configuration option for the DCS210+ command station. At that time, it may be appropriate to re-configure your JMRI connection to use the "DCS210+"-specific command station type setting.

Notes

  • Note that some users find that the DCS210+ USB interface feature can occasionally "lock-up". When this happens, it may be necessary to perform the following steps to restore proper JMRI operation:
    1. Quit JMRI and "shut down" your computer.
    2. Disconnect the USB cable from the DCS210+.
    3. Disconnect power from the DCS210+.
    4. Wait 30 seconds.
    5. Re-apply power to the DCS210+.
    6. Re-connect the USB cable in the DCS210+ jack.
    7. "Start-up" your computer.
    8. Start JMRI.

    JMRI developers are aware that the DCS210+ USB interface can sometimes "lock-up", but have not yet found either the cause of the problem or its solution. One workaround is to use a computer-to-LocoNet interface other than the DCS210+'s integrated USB connector.
