JMRI: TrainCrew connection

The JMRI web server can host the TrainCrew application locally. This gives you a convenient way to run your own copy for use with your handheld devices.

To install it, select "Install TrainCrew Application" from the JMRI Debug menu. Note that you'll need an active internet connection when you do this. But you won't need one to run TrainCrew after that.

Windows users will probably need to run JMRI with Administrative permissions ( eg. right click on PanelPro desktop link, Run as Administrator) to install.
it is not recommended to run JMRI like this all of the time, just to install TrainCrew.
