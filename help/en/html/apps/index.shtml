JMRI: Applications

JMRI provides several applications that are optimized for specific purposes. You can of course also write your own.

DecoderPro

A better tool for programming decoders. DecoderPro simplifies the job of configuring complicated DCC decoders by providing screens on which you can select the various options and values you want.

PanelPro

Operating CRT based CTC control panels that reflects the real-time state of your railroad and let you control it - almost like a Prototype dispatcher!

LocoNet Tools

A set of Digitrax LocoNet-specific tools:
  • LocoNet monitor
  • Slot monitor
  • Packet source
  • PM4 programmer
  • BDL16 programmer
  • SE8C programmer
  • DS64 programmer
These are also available in DecoderPro and PanelPro when those are configured for a LocoNet connection, but LocoTools provides them in a easier-to-use way.
