JMRI: DispatcherPro Main Window

Dispatcher is a primary tool within PanelPro that provides functions and organizes information relating to dispatching trains on a model railroad layout, allowing for automated running.

Dispatcher is dependent on PanelPro blocks, turnouts, sensors and signaling. It normally uses a Layout Editor panel. The Getting Started with PanelPro tutorial describes the process to create these prerequisites.
