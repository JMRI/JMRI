JMRI: Operations web functions

JMRI OperationsPro™ provides many features to support operating sessions. You can learn more about JMRI OperationsPro here.

Providing access to these Operations features from mobile devices and web browsers is still under development.
With JMRI running on your network and the Web Server active, the current functionality can be accessed by visiting the http://localhost:12080/operations URL.

Under the Operations tab, that web page lists all available trains, with a link to the Manifest page for each Train.
From each Manifest page, you can view the Locations to be visited, and the current status of the Train.
You can click on each location and the status line to display additional details about the pickups, set outs, and cars on the train.
