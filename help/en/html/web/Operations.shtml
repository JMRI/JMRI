JMRI: Operations web functions

JMRI Operations is a feature-rich area of JMRI. You can learn more about JMRI Operations [here] .

Providing access to these Operations features from mobile devices and web browsers is still in its early stages.
The current functionality can be accessed by visiting the /web/operationsTrains.html URL.
This page lists all available trains, with a link to the Manifest page for each Train.
From each Manifest page, you can view the Locations to be visited, and the current status of the Train.
You can click on each location and the status line to display additional details about the pickups, setouts, and cars on the train.
