JMRI: Roster Servlet

The JMRI Roster Servlet provides web access to information about JMRI roster entries. This servlet responds to the following URL patterns:
  • /roster/
  • /roster/list
  • /roster/list?filter=filter[&filter=filter]
  • /roster/<ID>
  • /roster/entry/<ID>
  • /roster/<ID>/image
  • /roster/<ID>/icon
