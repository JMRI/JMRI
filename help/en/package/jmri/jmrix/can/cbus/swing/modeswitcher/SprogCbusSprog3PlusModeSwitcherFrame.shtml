JMRI : CBUS SPROG 3 Plus Mode Switcher Tool

The mode switcher controls the programming modes that are available for the connected command station. Not all CBUS command stations support the mode switcher.

Open the Mode Switcher (if available) from the SPROG Generation 5 menu.

Select the desired mode by clicking one of the radio buttons.

The Decoder Pro main window status bar shows the currently available programmers.
