JMRI: CBUS Event Table

The CBUS Event Table displays event information received on your CBUS connection.

Columns

The Event Table displays the information in 5 columns (the column order can be rearranged by dragging a header, so it could differ on your system). :

  • CAN ID
  • Node
  • Name
  • Event
  • Type (On or Off)
  • Comment

Printing and Export

From the File menu, choose Export to open file save dialog and save the contents of the table in a CVS (comma separated values) text file.

Choose the Print (Preview) menu to create a hard copy of the Event Table.

JMRI Help

You can view this help page within JMRI by selecting Help > Window Help in the top bar of the MERG Event Table window.

The CBUS Console can help you figure out what events are happening on your layout.
