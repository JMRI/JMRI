JMRI: CBUS Node Configuration Tool

CV : Configuration Variable

Node : Each CBUS module in FLIM operation has a node number.

The CBUS Node Configuration tool programs Variables in a module using the modules' Node number.

Node Number

Using the number spinner or keyboard, enter the numerical Node number of the board and click [Set].

Set Node Variable

Using the number spinner or keyboard, enter the numerical Variable number. Next, you may click the [Read] button to ask the Node you just set to provide the value stored in the given Variable.
Change or enter e new value for (the contents of) the variable an click [Write].
