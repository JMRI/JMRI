JMRI: Select DCS52 USB Mode

A Digitrax DCS52 USB interface can act either as a stand-alone programmer, or as an interface to a LocoNet, but it can only do one at a time. Normally, you choose the USB mode as part of setting the JMRI connection preferences.

In some cases, you might want to quickly change the mode during debugging. This tool provides two buttons to allow you to change the DCS52 USB mode immediately. Note that this may result in unusual (and usually bad) behavior from the program! It's generally better to switch mode by changing the JMRI preferences and restarting the program.

For more information on the DCS52, please see its JMRI reference page.
