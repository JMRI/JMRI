Discover LocoNet SV1 Devices

since 5.11.5The "Manage LocoIO (LNSV1) Devices" tool lets you list all connected LocoNet devices that respond to the LocoNet SV version 1 protocol.
You find the tool in the LocoNet menu as long as a LocoNet connection is available.

As the tool is still under development, we advise to use it in combination with the "Show Raw Data" option in the LocoNet Monitor to check for any additional responses rejected by the SV1 filters.
