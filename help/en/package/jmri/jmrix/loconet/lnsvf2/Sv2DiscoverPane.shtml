Discover LocoNet SV2 Devices

The "Send Discover SV2 Devices" tool lets you list all connected LocoNet devices that respond to the SV version 2 protocol.
You find the tool in the LocoNet menu as long as a LocoNet connection is available.

As the tool is still under development, we advice to use it in combination with the "Show Raw Data" option in the LocoNet Monitor to check for any additional responses rejected by the SV2 filters.

Simply click "Discover" to start the discovery sequence.

After a maximum of 3 seconds all SV2 responses will be listed in the table.
