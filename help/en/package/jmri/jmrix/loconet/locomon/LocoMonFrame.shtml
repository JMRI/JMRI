Monitor LocoNet Traffic


The LocoNet Monitor tool displays LocoNet traffic in a human-readable form. The most recent messages are available in a scrolling window. You can filter unwanted messages from the display. Optionally, you can include the time the message was received, and/or the raw packet bytes. The log information can also be stored in a text file.

For more information on the various controls and options, please see the Communications Monitor Window page.
