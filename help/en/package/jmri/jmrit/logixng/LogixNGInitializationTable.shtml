JMRI: LogixNG Initialization Table

LogixNG is normally started after loading the xml data file. The Initialization table provides a list of LogixNGs that should be run before the rest of them are enabled. This is the same concept used by the LRoute/Logix RTXINITIALIZER.

The initialization table is at PanelPro Tools ⇒ LogixNG ⇒ LogixNG Initialization table.

Select a LogixNG from the combo box and press Add Initialization LogixNG.

The Delete button will remove the entry. The Move up and Move down buttons are used to change execution sequence for the LogixNGs.
