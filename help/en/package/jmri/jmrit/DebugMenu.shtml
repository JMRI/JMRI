JMRI: Debug Menu

  • Memory Usage Monitor
  • Import Decoder File
  • Import Decoder URL
  • Recreate Decoder Index
  • Rebuild Roster
  • Validate XML file
  • Check Decoder Names
  • Check Programmer Names
  • Load Logixs Disabled
  • Log Message
  • Display Message Log
  • Virtual Sound Decoder
  • PRICOM
  • USB Input Control
  • USB Device Viewer
  • RailDriver Throttle
  • Start JMRI Web Server
  • Start WiThrottle
