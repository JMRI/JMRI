Adding/Editing Signals

This is the package/jmri/jmrit/beantable/SignalAddEdit help page

Types of signals:

Triple Turnout
This type controls a signal via three electrical connections. One goes to a green lamp/LED, one to a yellow one, and the third to a red one.

For each connection, enter the appropriate JMRI turnout, either as a number (1,2,3) or as a JMRI name (LT1, CT3, XT3). They need not be consecutive, or in any particular order.

Double Turnout
This type controls a signal via two electrical connections. One goes to a green lamp/LED, and the other to a red one. Yellow is created by turning them both on at the same time.

For each connection, enter the appropriate JMRI turnout, either as a number (1,2) or as a JMRI name (LT1, CT2). They need not be consecutive, or in any particular order.

Virtual
This type doesn't connect to the layout at all. Use it when you want to e.g. display a signal on a PanelPro panel, but don't have a real one on the layout.
SE8c 4 Aspect
This type controls signals attached to a Digitrax SE8c signal driver card.

In the text field, enter the appropriate address as a number. For example, the 1st signal head on the 1st SE8c, as configured from the Digitrax factory, is addressed by putting a "257" here. See e.g. page 5 of the Digitrax SE8c manual (http://www.digitrax.com/ftp/se8c.pdf).

LDT LS-DEC
This type controls signals attached to a Littfinski DatenTechnik (LDT) LS-DEC signal decoder.

These controllers use seven DCC turnout (accessory decoder) addresses to specify the seven possible appearances of the signal. Enter those, either as numbers (1,2,3,...) or as JMRI turnout names (LT1, LT2, LT3, ...) in the appropriate text fields.

DCC Signal Decoder
This type controls signals attached to a decoder that uses the DCC signal packets defined by the NMRA DCC Working Group.
Grapevine
This type controls signals attached to a ProTrak Grapevine node.

In the text field, enter a system name for a Grapevine signal head, e.g. GH4 for the 4th connector on the 1st node or GH1003 for the 3rd signal connector on the second node.
