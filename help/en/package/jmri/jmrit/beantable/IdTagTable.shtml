The ID Tag Table

You can use the ID Tag Table to look at the contents of JMRI IdTag objects.

Note that you can sort the table by each of the columns; just click on a column heading.

ID Tag Table Controls

To create a new ID Tag, click on the "Add..." button at the bottom left of the pane.

Next to it, two check boxes let you activate:

  • Save when and where last seen
  • Use Fast Clock Time

For more information on ID Tags, see the ID Tag help page.
