Signal Mast Repeater

The Signal Mast repeater tool allows you to configure one signal mast to follow the state of another, or for both SignalMasts to update each others state.
For this to work each SignalMast must be of the same Signal System and Mast Type.
When a Master mast is selected from the list at the bottom of the screen the Slave list will be populated with the corresponding SignalMasts of the same type. If there are no valid Masts then the option to select the slave will remain disabled.

In the main table it is possible to select which direction the updates will take place, either Master to Slave, Slave To Master or Bothway. You can also disable/enable the repeater along with deleting it from this screen.
