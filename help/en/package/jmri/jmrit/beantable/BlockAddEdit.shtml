Adding/Editing Blocks

To add a new Block, click the Add.. button at the bottom left edge of the Block Table pane.

In the pane that opens, enter a valid Hardware Address for the Block and if desired enter a User Name. For example, a valid System Name for a LocoNet layout could be LB456. Another example, a valid Internal Block system name could be IB123. The optional User Name can be anything you wish to describe the Block as long as it is unique in your JMRI setup. See the "Names" Help page for more information.

Complete the definition of the new Block by closing the Add/Edit Block pane and clicking the Edit button at the right of the newly created Block entry in the Block Table. Fill out all fields as described on the Block Attributes Help page.

Creating A Number of Blocks

It is also possible to create a number of Blocks in one go.
Where supported, the "Add Range" check box will be enabled.

When checking the "Number to Add" box, this will enable the "Number To Add" field. Enter the number of Blocks that you wish to add. This will create a number of sequential Blocks.
If a User Name has been entered, a number will be added to the end of the name (starting by #2) to keep each Block User Name unique.

Adding Block Properties

To add the details of a new Block, close the Add Block pane by clicking the Close/X button and click Edit to the right of the new entry in the Block Table.

See Names under General Help for more information on JMRI object naming.
