Adding/Editing Turnouts

When adding a Turnout, from the drop down list select the System (hardware connection) to which the Turnout is attached.

Enter a valid Hardware Address for that Turnout and if desired enter a User Name.

One of the newer options for the System, MERG, let's the user optionally use 2 CBUS event values for each Turnout added (either discovered automatically or entered manually as -x;+y).

Creating A Number of Turnouts

It is also possible to create a number of Turnouts in one go.
Where supported, the "Add Range" check box will be enabled.

When checking the "Add Range" box, this will enable the "Number To Add" field. Here enter in the number of Turnouts that you wish to add. This will create that number of sequential Turnouts.

If a User Name has been entered, a number will be added to the end of the name to keep each Turnout User Name unique.
See Names under general help for more information.
