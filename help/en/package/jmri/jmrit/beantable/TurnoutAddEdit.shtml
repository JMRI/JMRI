Adding/Editing Turnouts

When adding a Turnout, you select from the drop down list the system to which the turnout is attached to.

Enter in a valid address for that turnout and if desired enter in a user name.

Creating A Number of Turnouts

It is also possible to create a number of turnouts in go.
Where supported the "Add Range" check box will be enabled.

When checking the "Add Range" box, this will then enable the "Number To Add" field. Here enter in the number of turnouts that you wish to add, this will then create that number of sequential turnouts.

If a username has been entered a number will be added to the end of the name to keep each username unique.
"Names" under general help for more information.
