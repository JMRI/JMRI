Adding/Editing Audio objects

To create an audio object, click either the "Add Source..." or the "Add Buffer..." button at the bottom of the Audio Table pane. This will open a new pane with space to enter a System Name and User Name for the new Audio object.

You must enter a unique System Name. System Names for Audio objects have a set pattern: they start with "IAL" for "Internal Audio Listener", "IAB" for "Internal Audio Buffer", or "IAS" for "Internal Audio Source".
