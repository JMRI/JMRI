Adding/Editing Audio objects

To create an audio object, click the "Add..." button on the Audio Table. This will open a small new window with space to enter a system name and user name for the new memory, plus and "Add" button.

You must enter a unique system name. System names for Audio objects have a set pattern. They start with "IAL" for "Internal Audio Listener", "IAB" for "Internal Audio Buffer", or "IAS" for "Internal Audio Source".

There are separate windows for adding each of the different types.

The user name is optional, but if given has to be unique.

For more information on audio objects, see the audio help page.
