Adding/Editing ID Tags

To create an ID Tag, click the "Add..." button at the bottom of the ID Tag Table pane.
This will open a new pane with space to enter a System Name and User Name for the new ID Tag, plus an "Add" button to confirm your entry.

You must enter a unique System Name. System Names for ID Tags should have the following pattern:

  • they should start with "ID" for "Internal iDentity",
  • followed by the actual Tag number itself.

The User Name is optional, but if given has to be unique.

The Tag ID shown in the table is determined from the System Name.
