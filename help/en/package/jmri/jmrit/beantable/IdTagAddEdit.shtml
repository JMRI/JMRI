Adding/Editing Id Tags

To create an Id Tag, click the "Add..." button on the Id Tag Table. This will open a small new window with space to enter a system name and user name for the new Id Tag, plus an "Add" button.

You must enter a unique system name. System names for Id Tags should have the following pattern. They should start with "ID" for "Internal iDentity", followed by the actual tag number itself.

The user name is optional, but if given has to be unique.

The tag ID shown in the table is determined from the system name.
