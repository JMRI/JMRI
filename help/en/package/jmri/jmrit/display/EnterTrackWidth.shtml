Set Track Width

The Set Track Width dialog allows changing widths of both types of track used in Layout Editor track diagrams. Track may be designated as mainline track or side track when creating or editting Track Segments. The track type of turnout parts and level crossing parts are inferred from the track type of connected Track Segments. All track initially defaults to side track. Specifying mainline track type for specific segments of track is an option that you may or may not choose to take advantage of.

Each type of track may be drawn using a different track width. Changing the width of a track type changes the widths of all track of that type in the track diagram.
