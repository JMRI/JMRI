The Label and Icon Property Editor

A contextual menu in the Panel Editor allows you modify the properties of elements on your Panel:

Edit Location

Right click an Icon or Label and select Edit Location... > Set x & y... to set the icon location on your Panel using two text boxes where you can change the x and y coordinates for this icon/label. By pressing the "Set" button, the icon will move to that location. If you wish to return the icon to its original location before the editor was opened, press the "Cancel" button to keep the icon at its original position and close the editor window.
Use the spinner (up/down arrows to the right of both fields) to change the value one pixel.

Edit Level

Right click an Icon or Label and select Edit Level... > Set Level... to set the desired display level using a text box. Levels range from 1 to 10, with 10 being the top layer.
Use the Level value to place one icon above another to get the view you want on your Panel.
Use the spinner (up/down arrow to the right of the text field) to change the value one level.

Rotating an Icon

Right click an Icon or Label and select Rotate (Orthogonal) to flip it 90 degrees counterclockwise (Icons only) or select Rotate (degrees)... and enter the angle in degrees to rotate the item clockwise.
To restore an Icon or Label to its original angle, enter the value "0".

Scaling an Icon

Right click an icon and select Scale (percentage)... to increase or decrease the size of the icon relative to its original size. To restore the icon to its original size, enter the value 100 (%).

Text Label Editor

Label Text

Right click an icon and select Edit Text... to edit the desired Label Text in the text box.
Close this pane by clicking on Set to use the value you entered, or click Cancel to keep the old value.

Fixed Size

Right click an icon and select Edit Fixed Size... to edit the width and height for a fixed space to contain the Labels (Note: a small size could obscure the Label). Enter "0" to release the Fixed Size property.

Label Margin

Right click an icon and select Edit Margin... > Set Margin Size to edit the size in pixels for the Label Margin (the space between the text and a Border).

Label Border

Right click an icon and select Edit Border... > Set Border Size to edit the size in pixels for the Label Border width.
Select Edit Border... > Border Color to pick a Label Border Color from the list.

More options

The contextual menu provides more options to set the Font, Font Size, Background Color, Justification and Orientation of Labels.

Label Properties

Selecting Properties from the contextual menu opens the Text Label pane. Using four tabs, you can control the Font, Contents, Border and Size & Position respectively.
Close this pane by clicking on "Okay" to update the Panel with your changes, "Cancel" to abort or click "Apply" to update the Panel using your edits but keep the Properties pane open.
