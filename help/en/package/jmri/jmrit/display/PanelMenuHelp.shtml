JMRI: Panels Menu

since 4.23.1

Note: The PanelPro main menu is being changed. See Loading and Storing Your Work for details. The menu references below are the for the new menu structure.

The JMRI "Panels" menu contains the following items:

  • New Panel...: Select to create a new panel in either Panel Editor, Control Panel Editor, Layout Editor or Switchboard Editor.
  • Show Panel: Shows the list of all Panels currently loaded in the program, and allows you to bring one to the front by selecting it.
  • Delete Panel...: Select a panel to be deleted.

Continue to the help pages for Panel Editor, Control Panel Editor, Layout Editor or Switchboard Editor.
