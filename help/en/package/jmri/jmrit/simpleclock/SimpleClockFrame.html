Setup Fast Clock

The JMRI internal fast clock system consists of two parts: A "Timebase" provides the basic time information, while one or more displays show you what's going on.

This window (reached from "Setup Fast Clock" item in the "Clocks" submenu under the "Tools" menu) is where you set up the internal timebase.

To configure the time base, you can:

  • Select the timebase source to control (typically, this is the internal computer clock)
  • Set the rate at which the clock runs
  • Set the time
  • Start and stop the clock
Once you have created a timebase by configuring it in this window, you can open clocks that will display on your screen. Currently, JMRI has three types available:
  • Analog clock - the traditional circular clock with hands. You can resize it by dragging the window edges. There's also a pause/run button.
  • Nixie clock - a digital clock that looks like the old-style Nixie tubes
  • Panel clock - you can also embed a clock in a panel (instead of a separate window) from the panel editor.
