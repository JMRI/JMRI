JMRI: DecoderPro Main Window

The DecoderPro main window gives you access to all the capabilities of JMRI. This is the original DecoderPro that used the same menus as PanelPro. This version is not included in recent JMRI installs. For more information on DecoderPro itself, please see the DecoderPro online manual.

The buttons on this window are short cuts to the decoder programers. Plus a help link to a write up about DecoderPro Symbolic Programmer and a Quit button that ends the JMRI program.
