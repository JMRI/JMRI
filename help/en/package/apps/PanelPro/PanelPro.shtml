JMRI: PanelPro Main Window

The PanelPro main window gives you access to all the capabilities of JMRI.

Help on specific menus:

  • Edit - (Preferences)
  • Tools
  • Roster
  • Panels
  • (Menus specific to your layout connection will appear here)
  • Debug
  • Window - Shows the currently available windows
    • Minimize - minimize the current window, hides it.
  • Help
    • Window Help - Opens a help page specific to the current window. For the main window, that's this page.
    • General Help - Opens the main JMRI Help page
    • License - Displays the JMRI license.
    • Locations - Show where the JMRI program and preferences directories are located
    • Context - Displays information about your setup of JMRI that might be useful when somebody is trying to help with a problem.
    • Upload Debugging Info - Opens a window for uploading debugging info.
