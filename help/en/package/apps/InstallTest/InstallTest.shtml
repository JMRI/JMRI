JMRI: InstallTest Main Window

The InstallTest main window is intended to show if you have Java and JMRI correctly installed. You should not use it for normal activity.

If you've gotten to this page by running InstallTest and selecting help, your JMRI installation is basically OK. You can move on to configuring and running DecoderPro, PanelPro, etc.
