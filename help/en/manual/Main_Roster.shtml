Create Entry...

Opens New Roster Setup Pane.

Edit Entry...

Opens Edit Roster Entry Setup Pane.

Copy Entry...

Opens the Select roster entry dialog box. After you select the roster,the Input prompt will be displayed. Useful when you have two locomotives with the same decoder program but different addresses.

Import Entry...

The Open dialog is displayed, select the roster file (roster.xml) that you wish to import. The Input prompt will be displayed. The roster file will be imported, update your roster index in the Debug menu.

Export Entry...

Opens the Select roster entry dialog box. Next the Save dialog will be displayed, select the location where you would like to save the roster file.

Delete Entry...

Opens Delete roster entry prompt box. Used to remove an entry from the DecoderPro roster.

Print Summary...

Opens your operating systems printer dialog and when print is selected, prints a complete summary of your roster entries. (typically 5 entries per page, so be prepared to print several pages).

Print Preview Summary...

Open your operating systems printer and when the print is initiated the preview window is shown.

Roster Grouping...

Roster Grouping allows you to allocate Roster Entries into different groups. If for example you run multiple layouts with different locomotives, you can allocate those locomotives to the relative group, thus when you select a roster group to work with, the drop down lists in the throttle or edit roster entries will only contain locomotives that are in that specific list.

When working with a Roster Group, some of the functions above change slightly.

Create Entry - The new entry will automatically be placed into the current group.
Delete Entry - this will now only remove the roster entry from the roster group. Entries can only be deleted, when working in "Global" mode
Print Summary and Print Preview Summary - will only list locomotives that are allocated in the specific roster group

Create Roster Group

Allows you to create a new Roster Group

Set Active Roster Group

Select the roster group to work with, this will restrict the roster entries displayed in the throttle address panel, edit roster entry, print preview etc.
to work with all roster entries select the group "Global".

Delete Roster Group

Allows you to remove a Roster Group. This will not delete the roster entries that have been allocated to the group.

Roster Table Association

This table lists all the Roster entries, from the drop down box at the top select the group that you wish to view/add/remove entries to. By default all entries belong to the "Global" group and can not be removed or added.

Associate Roster Entry to Group

This allows you to associate roster entries to roster groups. The drop down list of Roster Entries will only display entries that have not been allocated to that group.

Disassociation of Roster Entry to Group

This allows you to remove an association of a roster entry to a roster group. The drop down list of Roster Entries, will only display entries that have been allocated to the select group.
