JMRI®: DecoderPro® Manual

This Manual deals with DP3 up to version 3.6.
Online Help for the current version of JMRI® DecoderPro® is here.

DecoderPro3® is a Java-based cross-platform application for model railroaders.

DecoderPro® can run on any computer system that will run Java 1.6.0 or later, whether it is Macintosh, Windows, or Linux based. It does require that Java be installed on the computer.
You can get Java as a free (but large) download at: java.com/download/

Notes:
  1. All screen shots in this manual are from a Windows machine using the "Metal" User Interface option. While your screens will probably differ in some layout details, they will be essentially the same. All programming examples shown are with Digitrax PR3 (MS100 mode) connected to Digitrax Super Chief Radio Command Station. Most of the decoders in locomotive samples will be Digitrax, except for sound examples.
  2. This entire manual has been updated to correspond to JMRI® 3.6 DecoderPro only, and the SHTML verified using HTML4 and Cascading Style Sheets. If the formatting looks odd, you probably need to update your browser.
  3. DecoderPro® v3.x was at one time known as DecoderPro3® v3.x, however since JMRI version 4 we leave out the 3 in the name.
