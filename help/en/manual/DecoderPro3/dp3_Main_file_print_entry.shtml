The Select Items to Print is opened from the DecoderPro Main Window menubar, File --> Print/Preview Roster-->Entry.

This dialog allows you to select what features of an individual decoder in Roster you would like to Preview or Print.

Cancel button -- Cancels the selections

Okay button -- displays the print dialog for your printer or printer preview window.
