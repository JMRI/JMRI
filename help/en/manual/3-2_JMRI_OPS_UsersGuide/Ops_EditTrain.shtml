allows you to remove locomotives and cars that have been assigned to a train. Useful when you want to try various build options and don't want to physically move locomotive and cars on the railroad.

Press to save your train. The program will create a file (OperationsTrainRoster.xml) in your JMRI directory in a folder called "operations". A backup file with the last settings for all of your trains is also created each time the is pressed.

Note that the edited  train is now shown in theTrains window.
