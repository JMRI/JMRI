Tools

  • Car Roster
    • Import from file
      Opens the Open dialog box, to import a roster of Cars if you already have a list of cars in a database. The import function requires an ACSII text file, with one line for each car. The Car import requires four attributes and the order is:
      car number, road name, type, and length.
      An example of a correct format:

      336 GCR Flat 40
      655579 UP Boxcar 40

      Optionally you may include additional attributes in the following order: weight, color, owner name, date built, and location.
      Check the Cars pane Help menu file for further information.
