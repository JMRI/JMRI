Tools

  • Locomotive Roster
    • Import from JMRI roster
    • Import from file
    • Export to file

      Saves the Roster to your Operations\Export folder and displays the Export complete dialog. The file is exported automatically as a .csv file.

    • Delete all locomotives

      Opens the Delete all locomotives? dialog. OK will delete all Locomotives in the Operations Roster; Cancel will abort the deletion.

    • Reset move count all locomotives

      Allows resetting all locomotive move counts.

    • Print -- Opens your computers Print window and will print your locomotive roster.

    • Preview

      Opens your computer's Print Preview window and shows a preview of your Locomotive Roster.

  • Synchronize with NCE Consists (only active with an NCE connection; greyed out for LocoNet)
