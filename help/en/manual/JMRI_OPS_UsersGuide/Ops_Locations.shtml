The Operations Tool, Locations menu opens the Locations window as shown. Locations are places on your railroad that trains visit to pickup or drop off cars and engines. Locations can be cities, towns, or just places where trains need to work cars.  

  1. Title Bar:  Window title Locations.
  2. MenuBar:  Windows menus
  3. Locations List
  4. Bottom of Window:
At the bottom of the window you may select Sort order of the locations

Sort by Name Id             Sets the sort order for the locations

Press to add a location
