OperationsPro: Modify Locations by Car Type window

Back

This tool opens from the (Edit) Locations window's Tool menu.
Use this tool to quickly modify the type of Cars your Locations and Tracks will service. It is very useful when adding a new Car Type to your Roster.

First select the Car Type you wish to modify from the drop-down list. When the Type is selected, notice that all Cars of that Type at the selected Locations will be checked.

Select the Locations and Tracks you want to service - or deselect the ones you don't want to service - and press to modify your Locations and Tracks.

You can also Copy how all cars of an existing Car Type are to be serviced by your Locations and apply those rules to another Car Type. To make a Car Type behave like another, first select the Car Type you would like to copy, then select the Copy.

The Car Type will now appear to the right of the Copy checkbox.

Now select the Car Types you want to apply these settings to and press to have both Car Types serviced by the same Locations and Tracks.
