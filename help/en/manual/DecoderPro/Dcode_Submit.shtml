Submitting to JMRI

How do I go about getting these approved and into the next release?

If you're happy with them, you can contribute them to the next release by creating a "Patch" for the project.

Save your DecoderDefinition.xml in a .zip file to process.

To do this:

  1. Log into GitHub (create a new GitHub account if needed).
  3. Create a new issue.
  4. Provide a descriptive but short title, a comment, and attach your zip file.
  5. Click Submit new issue.
