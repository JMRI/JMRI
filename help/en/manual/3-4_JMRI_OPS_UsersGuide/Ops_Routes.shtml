A route is a list of locations that a train will visit. Later you can decide if the train stops at a location for work. Routes can be point to point or out and back. The current Routes are shown in the Routes window with their status, i.e. Orphan means it has not been assigned to a train.

Panel showing possible routes that are in the Routes table.

Menu Bar

Body of  Routes window
List all the Routes defined for your Railroad.
Id column -- Id assigned by program to Route.
Name column -- Name of Route
Comment column -- Any comments that you wish to include in route data.
Status column -- Indicates the state of the route.
Okay route is assigned to a Train
Orphan route has not been assigned to a Train
Edit button column -- allows you to edit selected route attributes.
Bottom of Routes window

Sort by Name Id establishes the sort order of the routes.

opens the Add Routes window to allow you to add routes to your Operations data base.
