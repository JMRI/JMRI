The JMRI 2.5.7 test version is now available for download. 

This is the specific version for Linux users. 

It contains many new features, including:
* Panel Editor users may now change the default icons to be icons of their own choosing.
* Fix to a problem saving custom icons in the Panel Editor that was introduced in 2.5.7.
* Fix to a problem editing Signal Head definitions from their table.

For more information, please see the release note at 
http://jmri.sf.net/releasenotes/jmri2.5.7.shtml
