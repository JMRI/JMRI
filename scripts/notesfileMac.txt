The JMRI 2.7.1 test version is now available for download. 

This is the specific version for Mac OS X users. 

It contains many new features, including:
* Several new and updated decoder definitions
* Significant improvements in Panel Editor icon support
* Turnout invert capability for XPressNet
* Fix to invert for LocoNet sensors

For more information, please see the release note at 
http://jmri.sf.net/releasenotes/jmri2.7.1.shtml
